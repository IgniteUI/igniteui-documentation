# 高度なレイアウト要素
高度なレイアウト要素は、完全なレポートを作成するために必ずしも必要としない要素です。これらの要素は、レポートのレイアウトに関連した特定のタスクを実行する支援を行います。

高度なレイアウト 要素の詳細は以下のリンクをクリックしてください。

-   [チェーン](DocumentEngine-Chain.html)
-   [ギャップ](DocumentEngine-Gap.html)
-   [回転器](DocumentEngine-Rotator.html)
-   [規則](DocumentEngine-Rule.html)
-   [サイト](DocumentEngine-Site.html)
-   [ストレッチャ](DocumentEngine-Stretcher.html)
