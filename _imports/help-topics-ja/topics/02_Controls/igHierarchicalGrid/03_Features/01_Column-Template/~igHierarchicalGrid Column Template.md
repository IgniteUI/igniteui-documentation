# 列テンプレート (igHierarchicalGrid)

## 概要

`igHierarchicalGrid`™ の列テンプレート機能を使用すると、列テンプレートを割り当てることができ、指定されたテンプレートに従った方法で列データが表示されます。列テンプレートは、親レイアウトと子レイアウトの両方に適用できます。列テンプレート機能は Infragistics® Template Engine を使用するため、列テンプレート自体がエンジンでサポートされる機能に従う必要があります。列テンプレートの利点は、個別のテンプレートを任意の列に割り当てることができることです。テンプレート化する際にひとつの列を対象とするか、複数あるいはすべてを対象とするかは、列テンプレートを親レイアウトに適用するか子レイアウトに適用するかと同様に、開発者の裁量に任されています。
