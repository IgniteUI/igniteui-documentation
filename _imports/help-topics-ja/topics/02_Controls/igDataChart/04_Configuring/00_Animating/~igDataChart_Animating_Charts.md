# チャートのアニメーション化 (チャートの Motion Framework)


##トピックの概要
### 概要

Infragistics® Motion Framework™ は、%%ProductName%%™ チャート コントロールのデータ変更をアニメーション化するためのフレームワークです。

### トピック

このセクションには、Motion Framework によるチャートのアニメーション化に関するトピックが記載されています。


-	[チャートの Infragistics Motion Framework の概要 (igDataChart)](igDataChart-Motion-Framework.html): このトピックでは、チャートの Infragistics® Motion Framework™ の概要とチャート アニメーションを構成できる項目について説明します。

-	[HTML および JavaScript におけるチャートのアニメーション化 (igDataChart)](igDataChart-Animating-HTML.html): このトピックでは、コード例を使用して、HTML および JavaScript で簡単なアニメーション化された柱状チャートを追加する方法を示します。

-	[ASP.NET MVC でのチャートのアニメーション化 (igDataChart)](Animating-Charts-in-ASP.NET-MVC.html): このトピックでは、コード例を使用して、AJAX POST 要求を使用してデータを動的に追加しているMVC で、簡単なアニメーション化された柱状チャートを追加する方法を示します。
