# レスポンス Web デザイン (RWD) モード (igGrid)

## このグループのトピックについて

### 概要

このグループのトピックでは、`igGrid`™ コントロールの レスポンス Web デザイン (RWD) モードのサポートを説明します。

列を非表示にする、またはテンプレートを使用してグリッドの UI をカスタマイズすることにより、レスポンシブ Web サイトで `igGrid` を構成できます。

RWD モードは、`igGrid` の`レスポンシブ`機能で実装されます。

- [レスポンス Web デザイン (RWD) モード構成を有効にする (igGrid)](igGrid-Enabling-Responsive-Web-Design-Mode.html): このトピックは、コード例を用いて、`igGrid` コントロールでレスポンス Web デザイン (RWD) モードを有効にする方法について説明します。
