# 列テンプレート (igGrid)

## 概要

列テンプレート機能を使用すると、列テンプレートを割り当てることができ、指定されたテンプレートに従った方法で列データが表示されます。列テンプレート機能は Infragistics® Templating Engine を使用するため、列テンプレート自体がエンジンでサポートされる機能に従う必要があります。列テンプレートの利点は、個別のテンプレートを任意の列に割り当てることができることです。開発者は、1 つの列、複数の列、またはすべての列にテンプレートを自由に適用できます。
