# igUpload での作業 (igUpload)

## このグループのトピックについて
### 概要

このグループのトピックは、`igUpload`™ コントロールに関する構成とその他のタスクを説明します。

### トピック

- [igUpload の構成](igUpload-Configuring-igUpload.html): このトピックは、`igUpload` コントロールの構成方法をコード例を用いて説明します。

- [HTTP ハンドラーとモジュールの使用](igUpload-Using-HTTP-Handler-and-Modules.html): このトピックは、HTTP モジュールおよび HTTP ハンドラーを構成して、`igUpload` コントロールでアップロードされるデータを処理します。

- [クライアント側イベントを使用](igUpload-Using-Client-Side-Events.html): このトピックは、`igUpload` コントロールにより起動されたクライアント側イベントの処理方法を説明します。

- [サーバー側イベントを使用](igUpload-Using-Server-Side-Events.html): このトピックは、`igUpload` コントロールにより起動されたサーバー側イベントの処理方法を説明します。

- [ファイルをストリームとして保存 (igUpload)](igUpload-Saving-Files-as-Stream.html): このトピックは、アップロード ファイルをファイルまたはメモリストリームとして処理し、保存する方法を説明します。詳細な手順は、各プロセスでメモリストリームとしてファイルを保存トピックをご参照ください。
