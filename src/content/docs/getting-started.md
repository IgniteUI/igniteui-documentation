# Getting Started

Welcome to the **IgniteUI Docs Template**! This guide will walk you through setting up your documentation site.

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/IgniteUI/docs-template.git my-docs
cd my-docs
npm install
```

## Running the Dev Server

```bash
npm run dev
```

Your site will be available at `http://localhost:4321`.

## Building for Production

```bash
npm run build
```

The output will be in the `dist/` directory, ready to deploy.

## Adding Pages

Add new `.md` or `.mdx` files to `src/content/docs/` with the required frontmatter.
