## Използване

<PlatformBlock for="WebComponents">

Първо трябва да инсталирате {ProductName}, като изпълните следната команда:

```cmd
npm install {PackageWebComponents}
```

След това трябва да импортирате <ApiLink pkg="core" type="Chip" />, необходимия му CSS, и да регистрирате неговия модул, както следва:


За пълно въведение в {ProductName} прочетете темата [**Първи стъпки**](../general-getting-started.md).

### Слотове

С предоставените слотове на компонента можете да добавяте потребителско съдържание към различни части на <ApiLink pkg="core" type="Chip" />. Компонентът предоставя икони по подразбиране за избор и премахване, но можете да ги персонализирате чрез слотовете `select` и `remove`. Можете също да добавите допълнително съдържание преди или след основното съдържание чрез слотовете `start` и `end`.

Препоръчваме да използвате елемент `<span>` при добавяне на прост текст, символи или емоджита, и компонент [`<igc-icon>`](../layouts/icon.md) при добавяне на икони към слотовете `remove`, `select`, `start` и `end`.

<PlatformBlock for="WebComponents">

```html
<igc-chip selectable removable>
    <igc-icon slot="select" name="verified-account"></igc-icon>
    <igc-icon slot="start" name="brush"></igc-icon>
    Chip
    <igc-icon slot="end" name="blood"></igc-icon>
    <igc-icon slot="remove" name="pacifier"></igc-icon>
</igc-chip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbChip Selectable="true" Removable="true">
    <IgbIcon slot="select" IconName="verified-account" Collection="material"></IgbIcon>
    <IgbIcon slot="start" IconName="brush" Collection="material"></IgbIcon>
    Chip
    <IgbIcon slot="end" IconName="blood" Collection="material"></IgbIcon>
    <IgbIcon slot="remove" IconName="pacifier" Collection="material"></IgbIcon>
</IgbChip>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrChip selectable={true} removable={true}>
    <IgrIcon slot="select" name="verified-account"></IgrIcon>
    <IgrIcon slot="start" name="brush"></IgrIcon>
    <span>Chip</span>
    <IgrIcon slot="end" name="blood"></IgrIcon>
    <IgrIcon slot="remove" name="pacifier"></IgrIcon>
</IgrChip>
```

</PlatformBlock>

## Стилизиране

Компонентът <ApiLink pkg="core" type="Chip" /> предоставя CSS частите `base`, `prefix`, `suffix`, които могат да се използват за промяна на всички негови свойства за стил.

```css
igc-chip::part(base) {
  background: var(--ig-primary-500);
  color: var(--ig-primary-500-contrast);
}

igc-chip::part(suffix) {
  color: var(--ig-gray-400);
}
```
