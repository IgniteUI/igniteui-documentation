# {Platform} 単一選択 ComboBox

{Platform} `ComboBox` は、単一選択モードと、メインの入力プロンプトを介した項目リストのクイック フィルタリングをサポートしています。ユーザーは、数文字タイプすることで、オプションのリストに探している項目を表示できます。Enter キーを押すと、最初にハイライト表示された一致が選択されます。

## {Platform} 単一選択の例

単一選択とクイック フィルタリングを有効にするには、<ApiLink type="Combo" /> で <ApiLink type="Combo" member="singleSelect" label="SingleSelect" /> プロパティを設定します。ユーザー エクスペリエンスとキーボード ナビゲーションはほとんど変わりませんが、オプション リストの上にある特別なフィルター ボックスに検索クエリを入力する代わりに、メインの入力ボックスが使用されます。

<PlatformBlock for="WebComponents">

```html
<igc-combo single-select></igc-combo>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbCombo SingleSelect></IgbCombo>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrCombo singleSelect></IgrCombo>
```

</PlatformBlock>

## 選択 API

<ApiLink type="Combo" member="singleSelect" label="SingleSelect" /> プロパティが適用された ComboBox の選択 API はほとんど同じままですが、このプロパティが設定されていない ComboBox と比べて重要な違いがいくつかあります。

主な違いは、一度に 1 つの項目しか選択できないことです。たとえば、コンボ コンポーネントに <ApiLink type="Combo" member="valueKey" label="ValueKey" /> を指定した場合、複数の項目を <ApiLink type="Combo" member="select" label="Select" />/<ApiLink type="Combo" member="deselect" label="Deselect" /> メソッドに渡しても効果はありません。これは、以前に選択した項目が、新しい選択を行うと自動的に選択解除されることも意味します。

単一の選択コンボでプログラムによって項目を選択 / 選択解除する方法は次のとおりです。

### 項目の選択:


<PlatformBlock for="WebComponents">


```ts
// select the item matching the 'BG01' value of the value key field.
combo.select('BG01');
```

</PlatformBlock>


<PlatformBlock for="Blazor">

```razor
<IgbCombo SingleSelect @ref="Combo"></IgbCombo>

@code {
    private IgbCombo Combo;

    this.Combo.Select(new object[] { "UK01" });
}
```

</PlatformBlock>


<PlatformBlock for="React">


```tsx
// select the item matching the 'BG01' value of the value key field.
comboRef.current.select('BG01');
```

</PlatformBlock>


新たに選択せずに項目の選択を解除するには、<ApiLink type="Combo" member="deselect" label="Deselect" /> メソッドを呼び出します。

#### 項目の選択解除:


<PlatformBlock for="WebComponents">


```ts
// deselect the item matching the 'BG01' value of the value key field.
combo.deselect('BG01');
```

</PlatformBlock>


<PlatformBlock for="Blazor">

```razor
<IgbCombo SingleSelect @ref="Combo"></IgbCombo>

@code {
    private IgbCombo Combo;

    this.Combo.Deselect(new object[] { "UK01" });
}
```

</PlatformBlock>


<PlatformBlock for="React">


```tsx
// deselect the item matching the 'BG01' value of the value key field.
comboRef.current.deselect('BG01');
```

</PlatformBlock>


## 無効な機能

当然のことながら、一部の構成オプションは単一選択 ComboBox では効果がありません。

### プレースホルダー

<ApiLink type="Combo" member="placeholderSearch" label="PlaceholderSearch" /> プロパティに値を割り当てても結果は得られません。これは、通常、オプションのリストの上に配置されるフィルター入力が単一の選択 ComboBox に存在しないためです。

### オプション リストのオートフォーカス

単一選択 ComboBox に <ApiLink type="Combo" member="autofocusList" label="AutofocusList" /> オプションを設定しても効果はありません。

## キーボード ナビゲーション

キーボード ナビゲーションは、非単一選択 ComboBox と同じように動作しますが、メイン入力がフィルタリング プロンプトの役割を果たし、フィルタリング / 検索入力に適用されるすべてのキーボード アクションがメイン入力プロンプトに移動される点が異なります。

## その他の機能

他のすべての機能は、非単一選択 ComboBox コンポーネントと同じように動作します。
