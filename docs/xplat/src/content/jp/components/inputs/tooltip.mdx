# {Platform} Tooltip (ツールチップ)

{ProductName} Tooltip コンポーネントは、特定の要素のツールチップを表示する方法を提供します。ツールチップは、通常、要素がキーボード フォーカスを受け取ったとき、またはマウスを要素の上に置いたときに、要素に関連する情報を表示するポップアップです。

## 使用方法

### ツールチップ ターゲット

目的の要素にツールチップを添付するには、<ApiLink pkg="core" type="Tooltip" /> の <ApiLink pkg="core" type="Tooltip" member="anchor" label="Anchor" /> プロパティを使用し、それをターゲット要素の ID に設定します。

<PlatformBlock for="WebComponents">

```html

<igc-tooltip anchor="target-button">
  Congrats you've hovered the button!
</igc-tooltip>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrButton id="target-button">Hover me</IgrButton>
<IgrTooltip anchor="target-button">
  Congrats you have hovered the button!
</IgrTooltip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbTooltip Anchor="target-button">
  Congrats you have hovered the button!
</IgbTooltip>
```

</PlatformBlock>

<DocsAside type="info" title="情報">
Tooltip API メソッド (<ApiLink pkg="core" type="Tooltip" member="show" label="Show" />、<ApiLink pkg="core" type="Tooltip" member="hide" label="Hide" />、<ApiLink pkg="core" type="Tooltip" member="toggle" label="Toggle" />) では、<ApiLink pkg="core" type="Tooltip" member="showDelay" label="ShowDelay" /> プロパティと <ApiLink pkg="core" type="Tooltip" member="hideDelay" label="HideDelay" /> プロパティが考慮されないことに注意することが重要です。呼び出されるとすぐに動作します。

### 配置

<ApiLink pkg="core" type="Tooltip" /> は、ターゲット要素を基準にして簡単に配置することもできます。必要なのは、<ApiLink pkg="core" type="Tooltip" member="placement" label="Placement" /> プロパティと <PlatformBlock for="Angular,React,WebComponents"><ApiLink pkg="core" type="PopoverPlacement" /></PlatformBlock><PlatformBlock for="Blazor"><a href="https://staging.infragistics.com/api/blazor/IgniteUI.Blazor.Lite/0.0.1/enums/PopoverPlacement"><code>PopoverPlacement</code></a></PlatformBlock> のいずれかの配置オプションとともに使用するだけです。

<ApiLink pkg="core" type="Tooltip" member="placement" label="Placement" /> プロパティが設定されていない場合、デフォルト値は `Bottom` となり、<ApiLink pkg="core" type="Tooltip" /> はターゲット要素の下に配置されます。

さらに、<ApiLink pkg="core" type="Tooltip" member="sticky" label="Sticky" /> プロパティを使用して <ApiLink pkg="core" type="Tooltip" /> を固定することができます。これにより、閉じるボタンが追加され、ユーザーが閉じるボタンをクリックするか、`Esc` キーを押して手動で閉じるまで、<ApiLink pkg="core" type="Tooltip" /> が表示されたままになります。この動作はデフォルトのホバー動作をオーバーライドし、ユーザーがターゲット要素上でホバーを停止したときに <ApiLink pkg="core" type="Tooltip" /> が消えないようにします。

<ApiLink pkg="core" type="Tooltip" /> には、<ApiLink pkg="core" type="Tooltip" member="withArrow" label="WithArrow" /> プロパティを介して構成できるオプションの矢印インジケータも含まれています。矢印はツールチップをその anchor 要素に視覚的に接続し、その配置はツールチップの <ApiLink pkg="core" type="Tooltip" member="placement" label="Placement" /> に基づいて自動的に調整されます。

<PlatformBlock for="WebComponents">

```html
<igc-button id="target-button">Hover me</igc-button>
<igc-tooltip anchor="target-button" placement="top-start" sticky with-arrow>
  Congrats you've hovered the button!
</igc-tooltip>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrButton id="target-button">Hover me</IgrButton>
<IgrTooltip anchor="target-button" placement="top-start" sticky withArrow={true}>
  Congrats you have hovered the button!
</IgrTooltip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbButton id="target-button">Hover me</IgbButton>
<IgbTooltip Anchor="target-button" Placement="PopoverPlacement.TopStart" Sticky="true" DisableArrow="true">
  Congrats you have hovered the button!
</IgbTooltip>
```

</PlatformBlock>

### トリガー

デフォルトでは、<ApiLink pkg="core" type="Tooltip" /> はターゲット要素の上にマウスを移動したときにのみトリガーされます。ただし、<ApiLink pkg="core" type="Tooltip" member="showTriggers" label="ShowTriggers" /> プロパティと <ApiLink pkg="core" type="Tooltip" member="hideTriggers" label="HideTriggers" /> プロパティを使用してこの動作を変更し、<ApiLink pkg="core" type="Tooltip" /> が表示されるタイミングと消えるタイミングを制御できます。これらのプロパティは、`click`、`focus`、`keypress` などのイベント名を値として受け入れ、さまざまなシナリオで <ApiLink pkg="core" type="Tooltip" /> をトリガーできるようにします。

### 追加のプロパティ

これまでに説明したプロパティの他に、<ApiLink pkg="core" type="Tooltip" /> コンポーネントには、動作、配置、外観をさらに構成できるさまざまな追加プロパティが用意されています。

|名前|タイプ|説明|
|--|--|--|
| <ApiLink pkg="core" type="Tooltip" member="open" label="Open" /> | boolean | ツールチップの表示/非表示を決定します。 |
| <ApiLink pkg="core" type="Tooltip" member="withArrow" label="WithArrow" /> | boolean | ツールチップの矢印インジケーターをレンダリングするかどうかを決定します。 |
| <ApiLink pkg="core" type="Tooltip" member="offset" label="Offset" /> | number | ツールチップとその <ApiLink pkg="core" type="Tooltip" member="anchor" label="Anchor" /> 間のピクセル距離を設定します。 |

### メソッド

構成可能なプロパティに加えて、<ApiLink pkg="core" type="Tooltip" /> では次の 3 つのメソッドも使用できます。

|名前|説明|
|--|--|
| <ApiLink pkg="core" type="Tooltip" member="show" label="Show" /> | ツールチップが表示されていない場合は表示します。ターゲットが指定されている場合は、ターゲットを一時的な <ApiLink pkg="core" type="Tooltip" member="anchor" label="Anchor" /> として設定します。 |
| <ApiLink pkg="core" type="Tooltip" member="hide" label="Hide" /> | ツールチップがまだ非表示になっていない場合は非表示にします。 |
| <ApiLink pkg="core" type="Tooltip" member="toggle" label="Toggle" /> | ツールチップの表示状態と非表示状態を切り替えます。 |

## アクセシビリティと ARIA サポート

<ApiLink pkg="core" type="Tooltip" /> はアクセシビリティを考慮して構築されており、次の ARIA 属性が含まれています。

- `role` - ツールチップがデフォルトの動作になっている場合、`role="tooltip"` が適用されます。<ApiLink pkg="core" type="Tooltip" member="sticky" label="Sticky" /> プロパティが有効な場合、ロールは `status` に変更されます。
- `inert` - 可視性に基づいて動的に切り替えられます。ツールチップが非表示になると、ツールチップは非アクティブになります。
- `aria-atomic` - true に設定すると、ツールチップのコンテンツ全体が変更時に通知されるようになります。
- `aria-live` - polite に設定して、ユーザーがアイドル状態のときにのみ更新を通知することをスクリーン リーダーに示します。

## スタイル設定

<ApiLink pkg="core" type="Tooltip" /> コンポーネントは、スタイル設定に使用できる 2 つの CSS パーツを公開します。

|名前|説明|
|--|--|
| `base` | ツールチップ コンポーネントの基本ラッパー。 |
| `top, right, bottom, left ...` | ツールチップ矢印を含む領域。パーツ名はツールチップ配置プロパティの値と一致します。 |

```css
igc-tooltip::part(base) {
  background-color: var(--ig-primary-500);
  color: var(--ig-primary-500-contrast);
}

igc-tooltip::part(bottom) {
  border-bottom-color: var(--ig-primary-500);
}
```
