{ProductName} 評価コンポーネントを使用すると、ユーザーはフィードバックを表示して提供できます。

## 構成

### Single (単一)

評価の <ApiLink type="Rating" member="single" label="Single" /> ビジュアル モードをオンにします。フィードバック絵文字の顔など、固有の値を伝える記号を使用する場合に便利です。

### Value (値)

<ApiLink type="Rating" member="value" label="Value" /> 属性は、コンポーネントの現在の値を設定します。

### Label (ラベル)

<ApiLink type="Rating" member="label" label="Label" /> 属性を使用すると、評価コンポーネントのラベル値を設定できます。

### 値の形式
[aria-valuetext (英語)](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-valuetext) を設定する書式文字列。そのすべてのインスタンスは、コントロールの現在の値に置き換えられます。スクリーン リーダーにとって重要であり、ローカライズに役立ちます。

### Max Value (最大値)

<ApiLink type="Rating" member="max" label="Max" /> 属性は、評価コンポーネントの最大許容値を設定します。

### Step (ステップ)

<ApiLink type="Rating" member="step" label="Step" /> 属性は、2 つのシンボル間の許容されるステップの割合を設定します。評価記号を半分に分割する場合に便利です。

### Hover Preview (ホバー プレビュー)


<PlatformBlock for="WebComponents">


`hover-preview` 属性により、ホバー時にユーザー選択の可能な結果がコンポーネントに表示されます。選択した値が何であるかについて即座にフィードバックしたい場合に便利です。

</PlatformBlock>



<PlatformBlock for="Blazor">


`HoverPreview` 属性により、ホバー時にユーザー選択の可能な結果がコンポーネントに表示されます。選択した値が何であるかについて即座にフィードバックしたい場合に便利です。

</PlatformBlock>



<PlatformBlock for="React">


`hoverPreview` 属性により、ホバー時にユーザー選択の可能な結果がコンポーネントに表示されます。選択した値が何であるかについて即座にフィードバックしたい場合に便利です。

</PlatformBlock>


### Read-Only (読み取り専用)

<ApiLink type="Rating" member="readOnly" label="ReadOnly" /> 属性を使用すると、ユーザーは <ApiLink type="Rating" /> を読み取り専用モードで設定できます。この属性は、コンポーネントを情報提供のみを目的として使用する場合に役立ちます。

### Disabled (無効)

<ApiLink type="Rating" member="disabled" label="Disabled" /> 属性はコンポーネントを無効にし、マウスやキーボードを使用して値を選択することを不可能にします。

## メソッド

### Step Up (ステップアップ)

<ApiLink type="Rating" member="stepUp" label="StepUp" /> メソッドは、コンポーネントの値を `n` ステップずつ増やします。`step` 係数によって決定されます。

### Step Down (ステップダウン)

<ApiLink type="Rating" member="stepDown" label="StepDown" /> メソッドは、コンポーネントの値を `n` ステップ分減らします。`step` 係数によって決定されます。

## スタイル設定

<ApiLink type="Rating" /> コンポーネントは、その内部要素のほとんどすべての CSS パーツを公開します。次の表に、公開されているすべての CSS パーツを示します。

|名前|説明|
|--|--|
| `base` | すべての評価要素を保持するメイン ラッパー。 |
| `label` | ラベルのパーツ。 |
| `value-label` | 値ラベルのパーツ。 |
| `symbols` | すべての評価記号のラッパー。 |
| `symbol` | カプセル化されたデフォルト シンボルのパーツ。 |
| `full` | カプセル化された完全なシンボルのパーツ。 |
| `empty` | カプセル化された空のシンボルのパーツ。 |

```css
igc-rating::part(full) {
  color: var(--ig-primary-500)
}

igc-rating::part(empty) {
  color: var(--ig-secondary-200);
}
```
