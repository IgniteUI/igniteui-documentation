# {Platform} オープン ストリート マップ画像の表示

{Platform} <ApiLink pkg="maps" type="OpenStreetMapImagery" /> は、世界中の OpenStreetMap© のコントリビューターが共同で作成した無料の地理的画像マッピングサービスです。これは、構成オプションなしで、道路地図スタイル限定で世界の地理的画像を提供します。この地理的画像サービスは、<a href="http://www.openstreetmap.org" target="_blank">www.OpenStreetMap.org</a> に直接アクセスして利用できます。
デフォルトでは、{ProductName} マップ コンポーネントには、Open Street Maps の地理的画像が既に表示されています。したがって、Open Street Maps から地理的画像を表示するように、コントロールを構成する必要はありません。

## コード スニペット
このコード例では、マップ コンポーネントの <ApiLink pkg="maps" type="GeographicMap" member="BackgroundContent" label="BackgroundContent" /> を OpenStreetMap© コントリビューターの地理画像を提供する <ApiLink pkg="maps" type="OpenStreetMapImagery" /> オブジェクトに明示的に設定します。

<PlatformBlock for="Angular">

```html
<igx-geographic-map #map
    width="100%"
    height="100%"
    zoomable="true" >
</igx-geographic-map>
```

</PlatformBlock>

<PlatformBlock for="Angular">

```ts
import { IgxGeographicMapComponent } from 'igniteui-angular-maps';
import { IgxOpenStreetMapImagery } from 'igniteui-angular-maps';
// ...
public map: IgxGeographicMapComponent;

const tileSource = new IgxOpenStreetMapImagery();
this.map.backgroundContent = tileSource;
```

</PlatformBlock>

<PlatformBlock for="React">

```ts
import { IgrGeographicMap } from 'igniteui-react-maps';
import { IgrOpenStreetMapImagery } from 'igniteui-react-maps';
// ...
const tileSource = new IgrOpenStreetMapImagery();

const geoMap = new IgrGeographicMap({ name: "geoMap" });
geoMap.backgroundContent = tileSource;
```

</PlatformBlock>

<PlatformBlock for="WebComponents">

```html
<igc-geographic-map id="geoMap" width="100%" height="100%">

</igc-geographic-map>
```

</PlatformBlock>

<PlatformBlock for="WebComponents">

```ts
import { IgcGeographicMapComponent } from 'igniteui-webcomponents-maps';
import { IgcOpenStreetMapImagery } from 'igniteui-webcomponents-maps';
// ...

let geoMap = document.getElementById("geoMap") as IgcGeographicMapComponent

const mapImagery = new OpenStreetMapImagery();
this.geoMap.backgroundContent = mapImagery;
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbGeographicMap Height="100%" Width="100%" Zoomable="true"
    BackgroundContent="OSMImagery" />

@code {

    public IgbOpenStreetMapImagery OSMImagery;

    protected override void OnInitialized()
    {
        this.OSMImagery = new IgbOpenStreetMapImagery();
    }
}
```

</PlatformBlock>
