# {Platform} {ComponentTitle} 列選択の概要

{Platform} {ComponentTitle} の {ProductName} 列選択機能は、シングルクリックで列全体を選択してハイライト表示する、簡略化された Excel のような方法を提供します。これは `ColumnSelection` 入力を通じて有効にできます。豊富な API のおかげで、この機能により、選択状態の操作、選択された部分からのデータ抽出、データ分析操作、視覚化が簡単に行えます。

## 基本的な使用方法

列選択機能は、<ApiLink type="GridSelectionMode" /> 値を受け取る `ColumnSelection` 入力によって有効にすることができます。

## インタラクション

デフォルトの選択モードは `None` です。`Single` または `Multiple` に設定されると、すべての列は <ApiLink type="Column" member="selectable" label="Selectable" /> になります。列を選択するには、列をクリックして <ApiLink type="Column" member="selected" label="Selected" /> としてマークします。列が選択不可な場合、ホバー時に選択スタイルはヘッダーに適用されません。

<DocsAside type="info" title="情報">
[複数列ヘッダー](multi-column-headers.mdx)機能は <ApiLink type="Column" member="selectable" label="Selectable" /> 入力に反映されません。その子の 1 つ以上で選択動作が有効な場合、<ApiLink type="ColumnGroupComponent" /> は <ApiLink type="Column" member="selectable" label="Selectable" /> です。さらに、すべての <ApiLink type="Column" member="selectable" label="Selectable" /> 子孫が <ApiLink type="Column" member="selected" label="Selected" /> である場合、コンポーネントは <ApiLink type="Column" member="selected" label="Selected" /> としてマークされます。
</DocsAside>

<ComponentBlock for="Grid">
*Country Information 列グループでは、City 列 および Postal code 列のみを選択できます。

</ComponentBlock>
<ComponentBlock for="TreeGrid">
*Personal Details 列グループでは、列 ID と Title のみを選択できます。

</ComponentBlock>
<ComponentBlock for="HierarchicalGrid">
*Location 列グループでは、City 列のみを選択できます。

</ComponentBlock>

## キーボードの組み合わせ

<DocsAside type="info" title="情報">
キーボードの組み合わせは、グリッドの `ColumnSelection` 入力が `multiple` に設定されている場合にのみ使用できます。
</DocsAside>

**列の選択**機能のキーボード ナビゲーションには 2 つのオプションがあります。
- 複数列選択 - <kbd>CTRL</kbd> キーを押しながら、すべての**選択可能な**ヘッダー セルを <kbd>クリック</kbd>。
- 範囲列の選択 - <kbd>SHIFT</kbd> キーを押しながら + <kbd>クリック</kbd>、間にあるすべての**選択可能な**列を選択します。

## API 操作

**API** は、**非表示**列に関していくつかの追加機能を提供し、対応する **setter** を設定することにより、すべての**非表示**列を <ApiLink type="Column" member="selected" label="Selected" /> としてマークできます。

<DocsAside type="info" title="情報">
上記は <ApiLink type="ColumnGroupComponent" /> にも適用されますが、<ApiLink type="Column" member="selected" label="Selected" /> プロパティを変更する場合、その子孫の状態を変更します。
</DocsAside>

API 操作の詳細については、[API リファレンス](#api-リファレンス) セクションを参照してください。

## スタイル設定

スタイル設定オプションに移動する前に、core ジュールとすべてのコンポーネント mixin をインポートする必要があります。

```scss
@use "igniteui-angular/theming" as *;

// IMPORTANT: Prior to Ignite UI for Angular version 13 use:
// @import '~igniteui-angular/lib/core/styles/themes/index';
```

<DocsAside type="info" title="情報">
</DocsAside>
<ComponentBlock for="Grid">
>Please note that [row selection](row-selection.mdx) and [column selection](column-selection.mdx) can't be manipulated independently. They depend on the same `variables`.
</ComponentBlock>

>[行選択](row-selection.mdx)と[列選択](column-selection.mdx)は個別に操作できないことに注意してください。同じ`変数`に依存します。

**選択**と**ホバー**のスタイル設定を変更します。 <br/>

<ComponentBlock for="TreeGrid">
```scss
$custom-grid-theme: grid-theme(
    $row-selected-background: #011627,
    $row-selected-text-color: #ECAA53,
    $row-selected-hover-background: #011627,
    $header-selected-text-color: #ECAA53,
    $header-selected-background: #011627,
    $expand-icon-color: #ECAA53,
    $expand-icon-hover-color: #B64B80
);
```

</ComponentBlock>

<ComponentBlock for="Grid, HierarchicalGrid">
```scss
$custom-grid-theme: grid-theme(
    $row-selected-background: #011627,
    $row-selected-text-color: #ECAA53,
    $row-selected-hover-background: #011627,
    $header-selected-text-color: #ECAA53,
    $header-selected-background: #011627
);
```

</ComponentBlock>

最も簡単な方法は、カスタム **テーマ**を定義する方法です。

### CSS 変数の使用
最後にカスタム `{ComponentSelector}` テーマを含めます。

```scss
@include css-vars($custom-grid-theme)
```

### 基本テーマのオーバーライド
Internet Explorer 11 のコンポーネントをスタイル設定するには、CSS 変数をサポートしていないため、別のアプローチが必要です。

<DocsAside type="info" title="情報">
コンポーネントが [Emulated](../themes/styles.mdx#表示のカプセル化) ViewEncapsulation を使用している場合、`::ng-deep` を使用してこのカプセル化を`解除する`必要があります。カスタム テーマが他のコンポーネントに影響しないようにするには、`::ng-deep` の前に `:host` セレクターを含めるようにしてください。
</DocsAside>

```scss
:host {
    ::ng-deep {
        @include grid($custom-column-selection-theme);
    }
}
```
