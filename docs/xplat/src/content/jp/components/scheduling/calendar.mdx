# {Platform} Calendar (カレンダー) の概要

{ProductName} Calendar コンポーネントは、軽量で構成が簡単です。日付と曜日を表示するために使用されます。また、エンド ユーザーに月次または年次ビューを提供するための最良の方法でもあります。{ProductName} カレンダー コントロールを使用すると、ユーザーがナビゲートできる最小および最大の日付範囲を制限できます。

Ignite UI for {ProductName} Calendar は、日付情報を表示するための簡単で直感的な方法を提供します。単一または複数の日付選択モード、日付範囲の強表示と選択、キーボード ナビゲーション、週番号の有効化、サイズと間隔のオプションなど、さまざまな機能がパックされています。

### 日付の無効化

場合によっては、エンド ユーザーが選択できない Calendar の日付を無効にしたいことがあります。この機能は、<ApiLink pkg="inputs" type="Calendar" member="disabledDates" label="DisabledDates" /> プロパティを使用して実現されます。<ApiLink pkg="inputs" type="Calendar" member="disabledDates" label="DisabledDates" /> プロパティは、<PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> オブジェクトの配列です。各記述子には `Type` があり、オプションで `Date` オブジェクトの配列である <PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> があります。

`Type` プロパティで使用できるオプションは次のとおりです:

- `After` - <PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> の最初の日付以降の日付を無効にします。
- `Before` - <PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> の最初の日付より前の日付を無効にします。
- `Between` - <PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> の最初の日付と 2 番目の日付の間の日付を無効にします。
- `Specific` - <PlatformBlock for="Angular,WebComponents,Blazor"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} exclude="WebComponents" /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="inputs" type="DateRangeDescriptor" member="dateRange" label="DateRange" suffix={false} kind="interface" prefixed={false} /></PlatformBlock> 配列で指定された日付を無効にします。
- `Weekdays` - すべての平日を無効にします。
- `Weekends` - すべての週末を無効にします。

現在の月の 3 日から 8 日の日付を無効にするサンプルを作成します。

```ts
const range = [
    new Date(today.getFullYear(), today.getMonth(), 3),
    new Date(today.getFullYear(), today.getMonth(), 8)
];

this.calendar.disabledDates = [{ type: DateRangeType.Between, dateRange: range }];
```

## キーボード ナビゲーション

- [月の選択] ボタン
- [年の選択] ボタン
- [前へ] ボタン
- [次へ] ボタン
- アクティブな日付要素

<ApiLink pkg="inputs" type="Calendar" /> コンポーネントの**日/月/年**がフォーカスされている場合は、次を使用します:

- <kbd>PAGE UP</kbd> キーを押すと、前の月/年のページに移動します。
- <kbd>PAGE DOWN</kbd> キーを押すと、次の月/年のページに移動します。
- <kbd>HOME</kbd> キーを使用して、現在の月の最初の日/最初の月を表示/最初の年を表示します。
- <kbd>END</kbd> キーを押すと、当月/先月/昨年の最終日にフォーカスされます。
- <kbd>矢印</kbd> キーを使用して、日/月/年をナビゲートします。最初の項目の前と最後の項目の後に移動すると、ビューが次/前の月/年のページに切り替わります。

`days` ビュー内の**日**がフォーカスされている場合は、次を使用します:

- <kbd>SHIFT</kbd> + <kbd>PAGE UP</kbd> キーで前年に移動します。
- <kbd>SHIFT</kbd> + <kbd>PAGE DOWN</kbd> キーを押して翌年に移動します。
- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを押して、現在フォーカスされている日を選択します。

`months` ビュー内の**月**がフォーカスされている場合は、次を使用します:

- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを押すと、<ApiLink pkg="inputs" type="Calendar" member="activeDate" label="ActiveDate" /> が現在フォーカスされている月に変更され、`days` ビューに切り替わります。

`years` ビュー内の**年**がフォーカスされている場合は、次を使用します:

- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを使用して、<ApiLink pkg="inputs" type="Calendar" member="activeDate" label="ActiveDate" /> を現在フォーカスされている年に変更し、`months` ビューに切り替えます。

サブヘッダー内の**前**または**次**のボタンにフォーカスがある場合は、次を使用します:

- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを押すと、前/翌月/年のページに切り替わります。

サブヘッダー内の**月**ボタンにフォーカスがある場合は、次を使用します:

- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを押して、`months` ビューに切り替えます。

サブヘッダー内の**年**ボタンにフォーカスがある場合は、次を使用します:

- <kbd>SPACE</kbd> または <kbd>ENTER</kbd> キーを押して、`years` ビューに切り替えます。

- <kbd>SPACE</kbd> or <kbd>ENTER</kbd> key to switch to `years` view.

| `navigation-button` | 前/次のナビゲーション ボタン。 |

| `label` | 週ヘッダー ラベル要素。 |
