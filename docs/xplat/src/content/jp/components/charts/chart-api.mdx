# {Platform} チャート API

{ProductName} チャートは、`CategoryChart`、`FinancialChart`、`DataChart`、`DataPieChart`、`DoughnutChart`、`PieChart`、および `Sparkline` UI 要素でデータをプロットするためのシンプルで使いやすい API を提供します。
