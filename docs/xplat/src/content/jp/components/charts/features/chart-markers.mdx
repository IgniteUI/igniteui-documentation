# {Platform} チャート マーカー

{ProductName} マーカーは、カテゴリ チャートのプロット領域にデータ ポイントの値を表示する視覚要素です。値が主グリッド線と副グリッド線の間にある場合も指定したデータ ポイントの値をただちに識別できるようユーザーをサポートします。

## {Platform} チャート マーカーの例

次の例では、[折れチャート](../types/line-chart.md)は、2009 年から 2019 年までのヨーロッパ、中国、および米国の国々の再生可能エネルギーの発電量を比較しています。マーカーが <ApiLink pkg="charts" type="CategoryChart" member="markerType" label="MarkerType" /> プロパティを <ApiLink pkg="charts" kind="enum" type="MarkerType" member="Circle" label="Circle" prefixed={false} /> 列挙値に設定して有効になっています。

マーカーのカラーは、以下のサンプルの <ApiLink pkg="charts" type="CategoryChart" member="markerBrushes" label="MarkerBrushes" />プロパティと <ApiLink pkg="charts" type="CategoryChart" member="markerOutlines" label="MarkerOutlines" /> プロパティを設定することによっても管理されます。このサンプルでは、ドロップダウンを使用してマーカーと <ApiLink pkg="charts" type="CategoryChart" member="chartType" label="CategoryChart.ChartType" /> を構成できます。

## {Platform} チャート マーカー サイズ

<ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> プロパティをマーカーをサポートするシリーズに設定することで、データ ポイント マーカーのデバイス非依存ピクセル寸法を正確に制御できます。これにより、マーカー テンプレートやスタイルに関係なく、マーカーが画面上に表示される大きさを正確に制御できます。

デフォルトでは、マーカーのサイズはシリーズのマーカー テンプレートによって決まります。<ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> に特定の数値を設定すると、そのシリーズのすべてのマーカーがその正確なデバイス非依存ピクセルの幅と高さでレンダリングされます。<ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> を `NaN` に戻すと、デフォルトのテンプレート駆動のサイズ設定が復元されます。

<ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> プロパティは、<ApiLink pkg="charts" type="MarkerSeries" /> から派生するすべてのシリーズ タイプ (<ApiLink pkg="charts" type="LineSeries" />、<ApiLink pkg="charts" type="SplineSeries" />、<ApiLink pkg="charts" type="AreaSeries" />、<ApiLink pkg="charts" type="ColumnSeries" />、<ApiLink pkg="charts" type="ScatterSeries" />、<ApiLink pkg="charts" type="ScatterLineSeries" />、<ApiLink pkg="charts" type="ScatterSplineSeries" />、極座標/放射状シリーズ タイプなど) で使用できます。

次のコード例は、<ApiLink pkg="charts" type="DataChart" /> コントロールの <ApiLink pkg="charts" type="ScatterLineSeries" /> で <ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> を 30 デバイス非依存ピクセルに設定する方法を示しています。

<PlatformBlock for="Blazor">
```razor
<IgbDataChart>
    <IgbScatterLineSeries
        MarkerSize="30"
        MarkerType="Circle"
        XMemberPath="X"
        YMemberPath="Y"
        XAxisName="xAxis"
        YAxisName="yAxis" />
</IgbDataChart>
```
</PlatformBlock>

<PlatformBlock for="React">
```tsx

    <IgrScatterLineSeries
        markerSize={30}
        markerType="Circle"
        xMemberPath="X"
        yMemberPath="Y"
        xAxisName="xAxis"

</IgrDataChart>
```
</PlatformBlock>

<PlatformBlock for="WebComponents">
```html

</igc-data-chart>
```
</PlatformBlock>

マーカーをデフォルトのテンプレート駆動サイズにリセットするには、<ApiLink pkg="charts" type="MarkerSeries" member="markerSize" label="MarkerSize" /> を `NaN` に設定します (またはマークアップで属性を削除します)。

<PlatformBlock for="Blazor">
```razor
<IgbLineSeries MarkerSize="NaN" MarkerType="Circle" ... />
```
</PlatformBlock>

<PlatformBlock for="React">
```tsx
<IgrLineSeries markerSize={NaN} markerType="Circle" ... />
```
</PlatformBlock>

## {Platform} チャート チェックマーク マーカー タイプ

{ProductName} チャートは、<ApiLink pkg="charts" type="MarkerSeries" member="markerType" label="MarkerType" /> 列挙型に `Checkmark` オプションを含んでいます。このマーカーは、チャートのデータ ポイントに円の中に V 字型のチェックマーク アイコンを描画します。

`Checkmark` マーカー タイプを個々のシリーズに適用するには、シリーズの <ApiLink pkg="charts" type="MarkerSeries" member="markerType" label="MarkerType" /> プロパティを `MarkerType.Checkmark` に設定します。チャート内のすべてのシリーズに同時にチェックマーク形状を使用するには、チャートの <ApiLink pkg="charts" type="CategoryChart" member="markerAutomaticBehavior" label="MarkerAutomaticBehavior" /> プロパティを `MarkerAutomaticBehavior.Checkmark` に設定します。

`SeriesViewer.CheckmarkMarkerTemplate` プロパティは、チェックマーク マーカー タイプを持つシリーズに使用されるマーカー テンプレートを定義し、チャート全体の外観をカスタマイズするために使用できます。
