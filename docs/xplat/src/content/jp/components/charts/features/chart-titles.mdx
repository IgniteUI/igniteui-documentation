# {Platform} チャート タイトルとサブタイトル

チャート コントロールのタイトルとサブタイトル機能を使用すると、{Platform} チャートの上部セクションに情報を追加できます。

## 例

<Sample src="/charts/category-chart/line-chart-with-titles" height={600} alt="{Platform} チャート同期の例" />



## API リファレンス

チャート コントロールにタイトルまたはサブタイトルを追加すると、タイトルとサブタイトルの情報に応じて、チャートの内容が自動的にサイズ変更されます。

| プロパティ名         | プロパティ タイプ   |     説明 |
| ----------------------|------------------|------------ |
| <ApiLink type="ChartTitle" />          | 文字列 |  タイトルのテキスト コンテンツ |
| <ApiLink type="CategoryChart" member="titleTextColor" label="TitleTextColor" />      | 文字列 |  タイトルのテキスト色 |
| <ApiLink type="CategoryChart" member="titleAlignment" label="TitleAlignment" />      | HorizontalAlignment |  タイトルの水平方向の配置 |
| <ApiLink type="CategoryChart" member="titleTextStyle" label="TitleTextStyle" />      | 文字列 | タイトルのフォント スタイル。例えば、Italic Bold 8pt Times New Roman |
| <ApiLink type="CategoryChart" member="titleTopMargin" label="TitleTopMargin" />      | 数 | タイトルの上マージン。 |
| <ApiLink type="CategoryChart" member="titleLeftMargin" label="TitleLeftMargin" />     | 数 | タイトルの左マージン。 |
| <ApiLink type="CategoryChart" member="titleRightMargin" label="TitleRightMargin" />    | 数 | タイトルの右マージン。 |
| <ApiLink type="CategoryChart" member="titleBottomMargin" label="TitleBottomMargin" />   | 数 | タイトルの下マージン。 |
| <ApiLink type="CategoryChart" member="subtitle" label="Subtitle" />            | 文字列 |  タイトルのテキスト コンテンツ |
| <ApiLink type="CategoryChart" member="subtitleTextColor" label="SubtitleTextColor" />   | 文字列 |  タイトルのテキスト色 |
| <ApiLink type="CategoryChart" member="subtitleAlignment" label="SubtitleAlignment" />   | HorizontalAlignment |  タイトルの水平方向の配置 |
| <ApiLink type="CategoryChart" member="subtitleTextStyle" label="SubtitleTextStyle" />   | 文字列 | タイトルのフォント スタイル。例えば、Italic Bold 8pt Times New Roman |
| <ApiLink type="CategoryChart" member="subtitleTopMargin" label="SubtitleTopMargin" />   | 数 | タイトルの上マージン。 |
| <ApiLink type="CategoryChart" member="subtitleLeftMargin" label="SubtitleLeftMargin" />  | 数 | タイトルの左マージン。 |
| <ApiLink type="CategoryChart" member="subtitleRightMargin" label="SubtitleRightMargin" /> | 数 | タイトルの右マージン。 |
| <ApiLink type="CategoryChart" member="subtitleBottomMargin" label="SubtitleBottomMargin" />| 数 | タイトルの下マージン。 |
