# {Platform} チャートのオーバーレイ

{Platform} `DataChart` を使用すると、<ApiLink pkg="charts" type="ValueOverlay" /> を使用して定義した単一の数値で水平線または垂直線を配置できます。特定のシリーズの平均値や中央値などのデータを視覚化するのに役立ちます。

## {Platform} 値オーバーレイのプロパティ

データバインディングに `DataSource` を使用する他のシリーズ タイプとは異なり、値オーバーレイは <ApiLink pkg="charts" type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" /> プロパティを使用して単一の数値をバインドします。さらに、値オーバーレイでは、使用する単一の <ApiLink pkg="charts" type="ValueOverlay" member="axis" label="Axis" /> を定義する必要があります。X 軸を使用する場合、値のオーバーレイは垂直線になり、Y 軸を使用する場合は、水平線になります。

数値の X 軸または Y 軸を使用する場合、<ApiLink pkg="charts" type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" /> プロパティは、値のオーバーレイを描画する軸上の実際の数値を反映する必要があります。カテゴリ X または Y 軸を使用する場合、<ApiLink pkg="charts" type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" /> は、値オーバーレイを表示するカテゴリのインデックスを反映する必要があります。

数値オーバーレイを角度角軸で使用すると、チャートの中心からの線として表示され、半径半径軸を使用すると、円として表示されます。

<ApiLink pkg="charts" type="ValueOverlay" /> appearance properties are inherited from `Series` and so <ApiLink pkg="charts" type="ValueOverlay" member="brush" label="Brush" /> and <ApiLink pkg="charts" type="ValueOverlay" member="thickness" label="Thickness" /> for example are available and work the same way they do with other types of series.

<ApiLink pkg="charts" type="ValueOverlay" /> 外観プロパティは、`Series` から継承されているため、例えば `Brush` と `Thickness` を使用でき、他のタイプのシリーズと同じように機能します。

## {Platform} 値レイヤー

{Platform} チャート コンポーネントは、値の線を使用して、最小値、最大値、平均値などのデータのさまざまな焦点を示す機能も公開します。

<ApiLink pkg="charts" type="CategoryChart" /> および <ApiLink pkg="charts" type="FinancialChart" /> コンポーネントに <ApiLink pkg="charts" type="ValueLayer" /> を適用するには、チャート上で <ApiLink pkg="charts" type="CategoryChart" member="valueLines" label="ValueLines" /> プロパティを設定します。このプロパティは、<ApiLink pkg="charts" type="ValueLayerValueMode" kind="enum" suffix={false} prefixed={false} /> 列挙体のコレクションを受け取ります。複数の <ApiLink pkg="charts" type="ValueLayerValueMode" kind="enum" suffix={false} prefixed={false} /> 列挙をチャートの <ApiLink pkg="charts" type="CategoryChart" member="valueLines" label="ValueLines" /> コレクションに追加することで、同じチャート内で複数の値レイヤーを組み合わせたりできます。

`DataChart` では、これは、チャートの `Series` コレクションに <ApiLink pkg="charts" type="ValueLayer" /> を追加し、次に <ApiLink pkg="charts" type="ValueLayer" member="valueMode" label="ValueMode" /> プロパティを <ApiLink pkg="charts" type="ValueLayerValueMode" kind="enum" suffix={false} prefixed={false} /> 列挙の 1 つに設定することによって行われます。これらの各列挙とその意味を以下に示します。

- `Auto`: <ApiLink pkg="charts" type="ValueLayerValueMode" kind="enum" suffix={false} prefixed={false} /> 列挙体のデフォルト値モード。
- `Average`: 複数の値の線を適用して、チャートにプロットされた各系列の平均値を呼び出します。
- `GlobalAverage`: 単一の値線を適用して、チャート内のすべての系列値の平均を呼び出します。
- `GlobalMaximum`: 単一の値線を適用して、チャート内のすべての系列値の絶対最大値を呼び出します。
- `GlobalMinimum`: 単一の値線を適用して、チャート内のすべての系列値の絶対最小値を呼び出します。
- `Maximum`: チャートにプロットされた各系列の最大値を示すために、複数の値線を適用する可能性があります。
- `Minimum`: チャートにプロットされた各系列の最小値を示すために、複数の値線を適用する可能性があります。

<ApiLink pkg="charts" type="ValueLayer" /> 要素を使用するときに特定のシリーズが考慮されないようにする場合は、レイヤーに <ApiLink pkg="charts" type="ValueLayer" member="targetSeries" label="TargetSeries" /> プロパティを設定できます。これにより、レイヤーは定義したシリーズを強制的にターゲットにするようになります。単一の `DataChart` 内に必要な数の <ApiLink pkg="charts" type="ValueLayer" /> 要素を含めることができます。

次のサンプルは、<ApiLink pkg="charts" type="CategoryChart" /> 内のさまざまな <ApiLink pkg="charts" type="CategoryChart" member="valueLines" label="ValueLines" /> の使用法を示しています。

## チャート オーバーレイ テキスト <Badge variant="preview" />

{Platform} <ApiLink pkg="charts" type="ValueOverlay" />、<ApiLink pkg="charts" type="ValueLayer" />、およびすべてのデータ注釈レイヤーは、DataChart コンポーネントのプロット領域内にカスタム オーバーレイ テキストを描画できます。このオーバーレイ テキストを使用すると、レイヤーとの関係において、x 軸上の重要なイベント (例: 企業の四半期決算) または y 軸上の重要な値に注釈を付けることができます。

たとえば、`DataAnnotationSliceLayer`、<ApiLink pkg="charts" type="ValueOverlay" />、<ApiLink pkg="charts" type="ValueLayer" /> を使用してオーバーレイ テキストを表示できます。

<Sample src="/charts/data-chart/data-annotation-multiple-with-overlay-text" height={600} alt="{Platform} 複数のオーバーレイ テキスト" />


### オーバーレイ テキストのスタイル設定

このコード例は、`DataAnnotationSliceLayer`、<ApiLink pkg="charts" type="ValueOverlay" />、および <ApiLink pkg="charts" type="ValueLayer" /> 上のオーバーレイ テキストのスタイルを設定およびカスタマイズする方法を示しています。

<div class="divider--half"></div>

<PlatformBlock for="Blazor">

```razor
public Series StylingOverlayText()
{
    var annoLayer = new IgbDataAnnotationSliceLayer();

    // styling optional overlay text
    annoLayer.OverlayTextColor = Brushes.White;
    annoLayer.OverlayTextBackground = Brushes.Green;
    annoLayer.OverlayTextBorderColor = Brushes.Black;
    annoLayer.OverlayTextBorderThickness = 1;
    annoLayer.OverlayTextBorderRadius = 4;
    annoLayer.OverlayTextHorizontalMargin = 5;
    annoLayer.OverlayTextHorizontalPadding = 2;
    annoLayer.OverlayTextVerticalMargin = 5;
    annoLayer.OverlayTextVerticalPadding = 2;

    return annoLayer;
}
```

</PlatformBlock>
