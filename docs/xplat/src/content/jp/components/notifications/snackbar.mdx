## コード例

### 表示時間

<ApiLink type="Snackbar" member="displayTime" label="DisplayTime" /> プロパティを使用して、Snackbar コンポーネントが表示される期間を構成します。デフォルトでは、4000 ミリ秒に設定されています。

<Sample src="/notifications/snackbar/display-time" height={230} alt="{Platform} Snackbar 表示時間の例" />




### Action Text (アクション テキスト)

デフォルトでは、Snackbar コンポーネントは、<ApiLink type="Snackbar" member="displayTime" label="DisplayTime" /> で指定された期間が経過すると自動的に非表示になります。<ApiLink type="Snackbar" member="keepOpen" label="KeepOpen" /> プロパティを使用して、この動作を変更できます。この場合、Snackbar は非表示になりません。Snackbar の <ApiLink type="Snackbar" member="actionText" label="ActionText" /> を使用すると、コンポーネント内にアクション ボタンを表示できます。

<Sample src="/notifications/snackbar/action-text" height={230} alt="{Platform} Sanckbar アクション テキストの例" />




## スタイル設定

<ApiLink type="Snackbar" /> コンポーネントはいくつかの CSS パーツを公開し、スタイルを完全に制御できるようにします。

|名前|説明|
|--|--|
| `base` | スナックバー コンポーネントの基本ラッパー。 |
| `message` | スナックバーのメッセージ。 |
| `action` | デフォルトのスナックバー アクション ボタン。 |
| `action-container` | アクションを保持する領域。 |

```css
igc-snackbar::part(base) {
  background: var(--ig-primary-500);
  border-color: var(--ig-primary-800);
  color: white;
}
```
