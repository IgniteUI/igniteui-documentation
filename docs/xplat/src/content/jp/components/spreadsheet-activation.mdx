# {Platform} スプレッドシートのアクティブ化

{Platform} Spreadsheet コンポーネントは、コントロールで現在アクティブなセル、ペイン、およびワークシートを決定できるプロパティを公開します。これは、ユーザーがコントロール内で移動または編集している場所を判断するのに役立ちます。

## アクティベーションの概要

{Platform} <ApiLink pkg="spreadsheet" type="Spreadsheet" exclude="Angular,Blazor,WebComponents" /> コントロールのアクティブ化は、スプレッドシートの現在の <ApiLink pkg="excel" type="Workbook" prefixed={false} /> のセル、ペイン、およびワークシート間で分割されます。3 つの アクティブなプロパティは以下のとおりです。

- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activeCell" label="ActiveCell" exclude="Blazor" />: スプレッドシートのアクティブ セルを設定します。設定するには、<ApiLink pkg="spreadsheet" type="SpreadsheetCell" suffix={false} prefixed={false} exclude="Blazor" /> の新しいインスタンスを作成し、そのセルに関する列と行、またはセルの文字列アドレスなどの情報を渡す必要があります。
- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activePane" label="ActivePane" exclude="Blazor" />: スプレッドシート コントロールの現在アクティブなワークシートのアクティブ ペインを返します。
- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activeWorksheet" label="ActiveWorksheet" exclude="Blazor" />: スプレッドシート コントロールの <ApiLink pkg="excel" type="Workbook" prefixed={false} /> 内のアクティブ ワークシートを返すか、設定します。これは、スプレッドシートに添付されている <ApiLink pkg="excel" type="Workbook" prefixed={false} /> 内の既存のワークシートに設定することで設定できます。

## コード スニペット

次のコード スニペットは、<ApiLink pkg="spreadsheet" type="Spreadsheet" exclude="Angular,Blazor,WebComponents" /> コントロールのセルとワークシートのアクティブ化の設定を示しています。

```ts
this.spreadsheet.activeWorksheet = this.spreadsheet.workbook.worksheets(1);

this.spreadsheet.activeCell = new SpreadsheetCell("C5");
```
