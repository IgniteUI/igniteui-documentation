# {Platform} テーブルの使用

Infragistics {Platform} Excel Engine の <ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> 機能は、行列のデータを書式設定できます。ワークシート テーブルのデータは <ApiLink pkg="excel" prefixed={false} type="Worksheet" /> の他の行や列のデータから個々に管理できます。

## テーブルをワークシートに追加
Infragistics {Platform} Excel Engine のワークシート テーブルは <ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> オブジェクトによって表され、ワー​​クシートの <ApiLink pkg="excel" type="Worksheet" member="tables" label="Tables" prefixed={false} suffix={false} /> コレクションに追加されます。テーブルを追加するには、このコレクションの `Add` メソッドを呼び出す必要があります。このメソッドでは、テーブルを追加する領域、テーブルにヘッダーを含めるかどうか、およびオプションで <ApiLink pkg="excel" type="WorksheetTableStyle" prefixed={false} suffix={false} /> オブジェクトとしてテーブルのスタイルを指定できます。

以下のコード サンプルは、ヘッダーを含むテーブルを <ApiLink pkg="excel" prefixed={false} type="Worksheet" /> の A1 to G10 (A1 to G1 が列ヘッダー) 領域に追加する方法を示します。

```ts
var workbook = new Workbook(WorkbookFormat.Excel2007);
var worksheet = this.workbook.worksheets().add("Sheet1");

worksheet.tables().add("A1:G10", true);
```


テーブルを追加後 <ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> で <ApiLink pkg="excel" type="WorksheetTable" member="insertColumns" label="InsertColumns" prefixed={false} suffix={false} />、<ApiLink pkg="excel" type="WorksheetTable" member="insertDataRows" label="InsertDataRows" prefixed={false} suffix={false} />、<ApiLink pkg="excel" type="WorksheetTable" member="deleteColumns" label="DeleteColumns" prefixed={false} suffix={false} />、または <ApiLink pkg="excel" type="WorksheetTable" member="deleteDataRows" label="DeleteDataRows" prefixed={false} suffix={false} /> メソッドを呼び出して行列を追加または削除して変更できます。テーブルの <ApiLink pkg="excel" type="WorksheetTable" member="resize" label="Resize" prefixed={false} suffix={false} /> メソッドを使用して新しいテーブル範囲を設定できます。

以下のコード スニペットは、3 つのメソッドの使用方法を示します。

```ts
var workbook = new Workbook(WorkbookFormat.Excel2007);
var worksheet = workbook.worksheets().add("Sheet1");
var table = worksheet.tables().add("A1:G10", true);

//Will add 5 columns at index 1.
table.insertColumns(1, 5);

//Will add 5 rows at index 0.
table.insertDataRows(0, 5);

//Will delete 5 columns starting at index 1.
table.deleteColumns(1, 5);

//Will delete 5 rows starting at index 0.
table.deleteDataRows(0, 5);

//Will resize the table to be in the region of A1:G15.
table.resize("A1:G15");
```

## テーブルのフィルタリング
<ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> の列にフィルターを適用します。フィルターが列で適用されると、テーブルに適用したすべてのフィルター条件と一致する行を決定するために再評価されます。

テーブルのデータを後で変更または行の `Hidden` プロパティを変更した場合、フィルター条件は自動的に再評価されません。テーブルのフィルター条件は、テーブルの列フィルターが追加、削除、変更されたときか、<ApiLink pkg="excel" type="WorksheetTable" member="reapplyFilters" label="ReapplyFilters" prefixed={false} suffix={false} /> メソッドがテーブルに対して呼び出されたときに限り再適用されます。

以下は、<ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> の列で使用できるフィルター タイプです。

- <ApiLink pkg="excel" type="AverageFilter" prefixed={false} suffix={false} /> - このコードは、列のすべてのセルの平均値の上か下かに基づいてセルをフィルターする方法を示します。
- <ApiLink pkg="excel" type="CustomFilter" prefixed={false} suffix={false} /> - 1 つ以上のカスタム条件に基づいてセルをフィルターできます。
- <ApiLink pkg="excel" type="DatePeriodFilter" prefixed={false} suffix={false} /> - 年の特定の月または四半期の日付を含むセルのみが表示されます。
- <ApiLink pkg="excel" type="FillFilter" prefixed={false} suffix={false} /> - 特定の塗りつぶしを含むセルのみが表示されます。
- <ApiLink pkg="excel" type="FixedValuesFilter" prefixed={false} suffix={false} /> - 特定の表示値のみに一致するまたは日付/時間の特定のグループ内に分類されるセルが表示されます。
- <ApiLink pkg="excel" type="FontColorFilter" prefixed={false} suffix={false} /> - 特定のフォントの色を含むセルのみが表示されます。
- <ApiLink pkg="excel" type="RelativeDateRangeFilter" prefixed={false} suffix={false} /> - フィルターが適用されたときに、以下の日または前の四半期のように日付の相対的な時間の範囲内で発生するかどうかに基づいて、日付値ををフィルターできます。
- <ApiLink pkg="excel" type="TopOrBottomFilter" prefixed={false} suffix={false} /> - このフィルターはトップまたはボトム N 値をフィルターします。このフィルターはトップまたはボトム N %値をフィルターします。
- <ApiLink pkg="excel" type="YearToDateFilter" prefixed={false} suffix={false} /> - 年の始まりとフィルターが適用される日付の間に発生する場合、日付値を含むYearToDateFilter-をフィルターできます。

以下のコード スニペットは、<ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> の最初の列に平均を超えるフィルターを適用する方法を示します。

```ts
var workbook = new Workbook(WorkbookFormat.Excel2007);
var worksheet = workbook.worksheets().add("Sheet1");
var table = worksheet.tables().add("A1:G10", true);

table.columns(0).applyAverageFilter(AverageFilterType.AboveAverage);
```

## テーブルのソート
テーブル列でソート条件を設定するとソートが実行されます。ソート条件が列で設定されると、テーブルのセルの順番を決定するためにテーブルのすべてのソート条件が再評価されます。ソートの基準を満たすためにセルを移動させる必要があるとき、テーブルのセルの行全体が 1 つの単位として移動されます。

テーブルのデータが後で変更される場合、ソート条件は自動的に再評価されません。テーブルのソート条件は、ソート条件が追加、削除、変更される時に、または <ApiLink pkg="excel" type="WorksheetTable" member="reapplySortConditions" label="ReapplySortConditions" prefixed={false} suffix={false} /> メソッドがテーブルで呼び出されるときに限り再適用されます。ソート条件が再評価されると、表示されたセルのみがソートられます。非表示行のすべてのセルは適切に維持されます。

テーブル列からソート条件へアクセスする以外に <ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> の <ApiLink pkg="excel" type="WorksheetTable" member="sortSettings" label="SortSettings" prefixed={false} suffix={false} /> プロパティの <ApiLink pkg="excel" type="SortSettings" member="sortConditions" label="SortConditions" prefixed={false} suffix={false} exclude="Angular,React,WebComponents" /> コレクションからも公開されます。これは、列/ソート条件のペアの順番に並べられたコレクションです。このコレクション内の順序はソートの優先順位です。

列に設定可能なソート条件タイプは以下のとおりです。

- <ApiLink pkg="excel" type="OrderedSortCondition" prefixed={false} suffix={false} /> - セル値に基づいてセルを昇順または降順にソートします。
- <ApiLink pkg="excel" type="CustomListSortCondition" prefixed={false} suffix={false} /> - テキストまたは表示値に基づいて定義された順序でセルをソートします。このソート方法は、日付がカレンダーに表示されるためアルファベット順よりも便利です。
- <ApiLink pkg="excel" type="FillSortCondition" prefixed={false} suffix={false} /> - 塗りつぶしが特定のパターン/グラデーションであるかどうかに基づいてセルをソートします。
- <ApiLink pkg="excel" type="FontColorSortCondition" prefixed={false} suffix={false} /> - フォントが特定の色であるかどうかによってセルをソートします。

また <ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> の <ApiLink pkg="excel" type="WorksheetTable" member="sortSettings" label="SortSettings" prefixed={false} suffix={false} /> の <ApiLink pkg="excel" type="SortSettings" member="caseSensitive" label="CaseSensitive" prefixed={false} suffix={false} exclude="Angular,React,WebComponents" /> プロパティは、文字列が大文字と小文字を区別してソートできるかどうかを開発者が設定できます。

以下のコード スニペットは、<ApiLink pkg="excel" prefixed={false} type="WorksheetTable" /> に <ApiLink pkg="excel" type="OrderedSortCondition" prefixed={false} suffix={false} /> を適用する方法です。

```ts
var workbook = new Workbook(WorkbookFormat.Excel2007);
var worksheet = this.workbook.worksheets().add("Sheet1");
var table = worksheet.tables().add("A1:G10", true);

table.columns(0).sortCondition = new OrderedSortCondition(SortDirection.Ascending);

//Alternative:
table.sortSettings.sortConditions().addItem(table.columns(0), new OrderedSortCondition(SortDirection.Ascending));
```
