# {Platform} ラジアル ゲージの概要

{Platform} Radial Gauge コンポーネントは、針、目盛り、範囲、ラベルなどの視覚要素をサポートし、定義済みの図形やスケールを表示できます。<ApiLink pkg="gauges" type="RadialGauge" /> には、アニメーション化されたトランジションのサポートも組み込まれています。アニメーションは、<ApiLink pkg="gauges" type="RadialGauge" member="transitionDuration" label="TransitionDuration" /> プロパティの設定で簡単にカスタマイズできます。

## バッキング

ゲージには、スケールの後ろ側に描かれた背景図形があり、図形はゲージの背景として動作します。

バッキング要素はラジアル ゲージ コントロールの背景と境界線を表します。常に最初に描画される要素で針、ラベルやメモリなどの残りの要素はその上のオーバーレイです。

バッキングは、円形またはフィットにできます。円形の場合は 360 度の円形のゲージが作成されますが、一方フィット図形の場合は <ApiLink pkg="gauges" type="RadialGauge" member="scaleStartAngle" label="ScaleStartAngle" /> および <ApiLink pkg="gauges" type="RadialGauge" member="scaleEndAngle" label="ScaleEndAngle" /> プロパティで円弧部分が塗りつぶされます。これには、<ApiLink pkg="gauges" type="RadialGauge" member="backingShape" label="BackingShape" /> プロパティを設定します。

<PlatformBlock for="Angular">

```html
<igx-radial-gauge
    backingShape="Fitted"
    backingBrush="#fcfcfc"
    backingOutline="DodgerBlue"
    backingOversweep=5
    backingCornerRadius=10
    backingStrokeThickness=5
    backingOuterExtent=0.8
    backingInnerExtent=0.15
    scaleStartAngle=135 scaleEndAngle=45
    height="300px" width="300px"
    minimumValue=0 value=50
    maximumValue=80 interval=10>
</igx-radial-gauge>
```

</PlatformBlock>

## スケール

スケールは視覚要素で、<ApiLink pkg="gauges" type="RadialGauge" member="minimumValue" label="MinimumValue" /> と <ApiLink pkg="gauges" type="RadialGauge" member="maximumValue" label="MaximumValue" /> 値を設定してゲージの値範囲全体をハイライト表示できます。バッキングとともにゲージの全体的な図形を定義します。<ApiLink pkg="gauges" type="RadialGauge" member="scaleStartAngle" label="ScaleStartAngle" /> と <ApiLink pkg="gauges" type="RadialGauge" member="scaleEndAngle" label="ScaleEndAngle" /> プロパティは、スケールの円弧の境界線を定義します。<ApiLink pkg="gauges" type="RadialGauge" member="scaleSweepDirection" label="ScaleSweepDirection" /> プロパティが、スケールが時計回りまたは反時計回りのどちらの方向に動くかを指定します。<ApiLink pkg="gauges" type="RadialGauge" member="scaleBrush" label="ScaleBrush" />、<ApiLink pkg="gauges" type="RadialGauge" member="scaleStartExtent" label="ScaleStartExtent" />、<ApiLink pkg="gauges" type="RadialGauge" member="scaleEndExtent" label="ScaleEndExtent" /> プロパティを設定してスケールの外観をカスタマイズできます。

## ラベルとタイトル

ゲージ ラベルは <ApiLink pkg="gauges" type="RadialGauge" member="minimumValue" label="MinimumValue" /> と <ApiLink pkg="gauges" type="RadialGauge" member="maximumValue" label="MaximumValue" /> の値の間で指定された間隔で数値を表示する視覚要素です。0 はゲージ中央、1 はゲージ バッキングの外側範囲を表す <ApiLink pkg="gauges" type="RadialGauge" member="labelExtent" label="LabelExtent" /> プロパティで小数を使用してラベルの配置を設定できます。<ApiLink pkg="gauges" type="RadialGauge" member="fontBrush" label="FontBrush" /> や <ApiLink pkg="gauges" type="RadialGauge" member="font" label="Font" /> など、さまざまなスタイル プロパティを設定してラベルをカスタマイズできます。

## タイトルとサブタイトル

<ApiLink pkg="gauges" type="RadialGauge" member="titleText" label="TitleText" /> プロパティと <ApiLink pkg="gauges" type="RadialGauge" member="subtitleText" label="SubtitleText" /> プロパティが使用可能であり、どちらも針のカスタム テキストを表示するために使用できます。あるいは、<ApiLink pkg="gauges" type="RadialGauge" member="titleDisplaysValue" label="TitleDisplaysValue" /> と <ApiLink pkg="gauges" type="RadialGauge" member="subtitleDisplaysValue" label="SubtitleDisplaysValue" /> を true に設定すると、針の値が表示され、<ApiLink pkg="gauges" type="RadialGauge" member="titleText" label="TitleText" /> と <ApiLink pkg="gauges" type="RadialGauge" member="subtitleText" label="SubtitleText" /> がオーバーライドされます。したがって、タイトルにカスタム テキストを使用しながらサブタイトルで値を表示したり、その逆を行ったりすることができます。

## オプティカル スケーリング

ラジアル ゲージのラベルとタイトルにより、スケーリングを変更できます。これを有効にするには、まず <ApiLink pkg="gauges" type="RadialGauge" member="opticalScalingEnabled" label="OpticalScalingEnabled" /> を true に設定します。次に、ラベルが 100% のオプティカル スケーリングを持つサイズを管理する <ApiLink pkg="gauges" type="RadialGauge" member="opticalScalingSize" label="OpticalScalingSize" /> を設定できます。ゲージのサイズが大きくなると、ラベルのフォントも大きくなります。たとえば、このプロパティが 500 に設定され、ゲージのピクセル単位のサイズが 2 倍の 1000 になると、ラベルのフォント サイズは 200% 大きくなります。

## 目盛

目盛は、ラジアル ゲージの中央から放射状に表示される細い線です。目盛には、主目盛および副目盛の 2 種類があり、主目盛りは <ApiLink pkg="gauges" type="RadialGauge" member="minimumValue" label="MinimumValue" /> と <ApiLink pkg="gauges" type="RadialGauge" member="maximumValue" label="MaximumValue" /> の間の <ApiLink pkg="gauges" type="RadialGauge" member="interval" label="Interval" /> に表示されます。また <ApiLink pkg="gauges" type="RadialGauge" member="minorTickCount" label="MinorTickCount" /> プロパティは、隣接する 2 つの主目盛間の副目盛の数を指定します。目盛りの長さは、<ApiLink pkg="gauges" type="RadialGauge" member="tickStartExtent" label="TickStartExtent" />、<ApiLink pkg="gauges" type="RadialGauge" member="tickEndExtent" label="TickEndExtent" />、<ApiLink pkg="gauges" type="RadialGauge" member="minorTickStartExtent" label="MinorTickStartExtent" />、<ApiLink pkg="gauges" type="RadialGauge" member="minorTickEndExtent" label="MinorTickEndExtent" /> に少数値 (0 から 1 の間) を設定して制御できます。

## 範囲

範囲に <ApiLink pkg="gauges" type="RadialGauge" member="minimumValue" label="MinimumValue" /> や <ApiLink pkg="gauges" type="RadialGauge" member="maximumValue" label="MaximumValue" /> プロパティで指定した連続値の境界をハイライト表示します。開始値と終了値を指定してゲージに複数の範囲を追加でき、各範囲には、<ApiLink pkg="gauges" type="RadialGaugeRange" member="brush" label="Brush" /> や <ApiLink pkg="gauges" type="RadialGaugeRange" member="outline" label="Outline" /> などのカスタマイズ プロパティがあります。または、<ApiLink pkg="gauges" type="RadialGauge" member="rangeBrushes" label="RangeBrushes" /> や <ApiLink pkg="gauges" type="RadialGauge" member="rangeOutlines" label="RangeOutlines" /> プロパティを範囲の色リストに設定することもできます。

## 針

ゲージ針は、ゲージの設定値を示す視覚要素です。針は、あらかじめ定義されたいくつかの図形の中から選択でき、ピボット図形をゲージの中心に配置できます。またピボット図形は、事前に定義された図形の 1 つを使用します。オーバーレイとアンダーレイを含むピボット図形には、図形に適用する別のピボット ブラシがあります。

サポートされている針の形とキャップは、<ApiLink pkg="gauges" type="RadialGauge" member="needleShape" label="NeedleShape" /> と <ApiLink pkg="gauges" type="RadialGauge" member="needlePivotShape" label="NeedlePivotShape" /> プロパティで設定します。

ゲージのインタラクティブ モードを有効 (<ApiLink pkg="gauges" type="RadialGauge" member="isNeedleDraggingEnabled" label="IsNeedleDraggingEnabled" /> プロパティを使用) にするとユーザーは <ApiLink pkg="gauges" type="RadialGauge" member="minimumValue" label="MinimumValue" /> と <ApiLink pkg="gauges" type="RadialGauge" member="maximumValue" label="MaximumValue" /> の値間で針をドラッグして値を変更できるようになります。

<PlatformBlock for="Angular">

```html
<igx-radial-gauge
    value=50
    isNeedleDraggingEnabled=true
    isNeedleDraggingConstrained=true
    needleShape="NeedleWithBulb"
    needleBrush="DodgerBlue"
    needleOutline="DodgerBlue"
    needleEndExtent=0.475
    needleStrokeThickness=1
    needlePivotShape="CircleOverlay"
    needlePivotBrush="#9f9fa0"
    needlePivotOutline="#9f9fa0"
    needlePivotWidthRatio=0.2
    needlePivotStrokeThickness=1
    height="300px" width="300px"

    maximumValue=80 interval=10>
```
