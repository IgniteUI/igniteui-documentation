<PlatformBlock for="WebComponents">


<PlatformBlock for="React">

```tsx

```

</PlatformBlock>

<PlatformBlock for="WebComponents">

```html

```

</PlatformBlock>

<PlatformBlock for="Blazor">

</PlatformBlock>

### Slots

Con los slots expuestos del componente, puede agregar contenido personalizado a distintas partes de <ApiLink pkg="core" type="Chip" />. El componente proporciona iconos predeterminados de selección y eliminación, pero puede personalizarlos mediante los slots `select` y `remove`. También puede agregar contenido adicional antes o después del contenido principal mediante los slots `start` y `end`.

Recomendamos usar un elemento `<span>` al agregar texto simple, símbolos o emojis, y un componente [`<igc-icon>`](../layouts/icon.md) al agregar iconos a los slots `remove`, `select`, `start` y `end`.

<PlatformBlock for="WebComponents">

```html
<igc-chip selectable removable>
    <igc-icon slot="select" name="verified-account"></igc-icon>
    <igc-icon slot="start" name="brush"></igc-icon>
    Chip
    <igc-icon slot="end" name="blood"></igc-icon>
    <igc-icon slot="remove" name="pacifier"></igc-icon>
</igc-chip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbChip Selectable="true" Removable="true">
    <IgbIcon slot="select" IconName="verified-account" Collection="material"></IgbIcon>
    <IgbIcon slot="start" IconName="brush" Collection="material"></IgbIcon>
    Chip
    <IgbIcon slot="end" IconName="blood" Collection="material"></IgbIcon>
    <IgbIcon slot="remove" IconName="pacifier" Collection="material"></IgbIcon>
</IgbChip>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrChip selectable={true} removable={true}>
    <IgrIcon slot="select" name="verified-account"></IgrIcon>
    <IgrIcon slot="start" name="brush"></IgrIcon>
    <span>Chip</span>
    <IgrIcon slot="end" name="blood"></IgrIcon>
    <IgrIcon slot="remove" name="pacifier"></IgrIcon>
</IgrChip>
```

</PlatformBlock>

## Tamaño

Permitimos que el usuario elija el tamaño de <ApiLink pkg="core" type="Chip" /> utilizando la variable CSS `--ig-size`:

<PlatformBlock for="React">

```tsx
<IgrChip className="size-small" selectable={true} removable={true}>
    <span>Chip</span>
</IgrChip>
<IgrChip className="size-medium" selectable={true} removable={true}>
    <span>Chip</span>
</IgrChip>
<IgrChip className="size-large" selectable={true} removable={true}>
    <span>Chip</span>
</IgrChip>
```

</PlatformBlock>

```css
.size-small {
  --ig-size: var(--ig-size-small);
}

.size-medium {
  --ig-size: var(--ig-size-medium);
}

.size-large {
  --ig-size: var(--ig-size-large);
}
```

## Aplicación de estilos

El componente <ApiLink pkg="core" type="Chip" /> expone los elementos del control CSS `base`, `prefix`, `suffix`, que se pueden usar para cambiar todas sus propiedades de estilo.

```css
igc-chip::part(base) {
  background: var(--ig-primary-500);
  color: var(--ig-primary-500-contrast);
}

igc-chip::part(suffix) {
  color: var(--ig-gray-400);
}
```
