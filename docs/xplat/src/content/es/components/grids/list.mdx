# {Platform} Descripción general de la lista

El elemento de lista {ProductName} es extremadamente útil al presentar un grupo de elementos. Puede crear una lista sencilla de elementos de texto, o una más compleja, que contenga una matriz de diferentes elementos de diseño. El componente <ApiLink pkg="core" type="List" /> muestra filas de elementos y también admite uno o más encabezados. Cada elemento de lista es completamente personalizable mediante plantillas y admitirá cualquier HTML válido u otros componentes.

## Ejemplo de lista {Platform}

El siguiente ejemplo representa una lista rellenada con contactos con propiedades de nombre y número de teléfono. El componente <ApiLink pkg="core" type="List" /> que se muestra a continuación utiliza los elementos <ApiLink pkg="core" type="Avatar" /> y <ApiLink pkg="core" type="Button" /> para enriquecer la experiencia del usuario y exponer las capacidades de establecer una imagen de avatar y botones para acciones de texto y llamada.

La ranura `start` está pensada para usarse al agregar algún tipo de medio antes de todo el demás contenido de nuestros elementos de lista. El elemento de destino, en nuestro caso el componente <ApiLink pkg="core" type="Avatar" />, también recibirá una posición y un espaciado predeterminados.

La ranura `end` está pensada para usarse en elementos de lista que tengan algún tipo de acción o metadatos, representados, por ejemplo, por un interruptor, un botón, una casilla de verificación, etc. Usaremos componentes <ApiLink pkg="core" type="Button" />.

Permitamos también que el usuario cambie el tamaño de la lista mediante la variable CSS `--ig-size`. Agregaremos algunos botones de opción para mostrar todos los valores de tamaño. De este modo, siempre que se seleccione uno, cambiaremos el tamaño de la lista.

  <igc-radio name="size" value="medium" label-position="after">Medium</igc-radio>
  <igc-radio name="size" value="large" label-position="after" checked>Large</igc-radio>

      case "Small":

### Elementos recomendados para ranuras

Al insertar contenido en las ranuras `title` y `subtitle`, recomendamos usar elementos `<span>` en lugar de elementos de encabezado (`<h1>`–`<h6>`). Los elementos de encabezado incluyen estilos integrados (como tamaño de fuente, altura de línea y márgenes) que pueden interferir con la tipografía y el diseño previstos del componente. Usar un `<span>` proporciona un contenedor neutro que hereda limpiamente los estilos del componente.

Para lograr los mejores resultados, recomendamos usar el componente [`<igc-avatar>`](../layouts/avatar.md) para la ranura `start`, que está pensada para contenido multimedia. Para la ranura `end`, que está pensada para usarse con acciones, recomendamos usar componentes como [`<igc-button>`](../inputs/button.md), [`<igc-switch>`](../inputs/switch.md), [`<igc-checkbox>`](../inputs/checkbox.md), etc., según el tipo de contenido que desee mostrar. Sin embargo, para esas dos ranuras, también puede usar elementos HTML simples, como `<img>` para la ranura `start` y `<button>` u otro elemento interactivo para la ranura `end`, siempre que sean apropiados para el contenido que desea mostrar.

## Estilo

<ApiLink pkg="core" type="List" /> expone varias partes CSS, lo que le da control total sobre su estilo:

| Nombre          | Descripción                              |
| --------------- | ---------------------------------------- |
| `start`  | El contenedor inicial.                   |
| `end` | El contenedor final.                     |
| `content` | El contenedor de encabezado y contenido personalizado. |
| `header` | El contenedor de título y subtítulo.     |
| `title` | El contenedor de título.                 |
| `subtitle` | El contenedor de subtítulo.              |

```css
igc-list-header {
  font-size: 20px;
  font-weight: 700;
  color: var(--ig-primary-700);
}

igc-list-item::part(title) {
  font-size: 18px;
  color: var(--ig-primary-600);
}

igc-list-item::part(subtitle) {
  color: var(--ig-primary-300);
}
```
