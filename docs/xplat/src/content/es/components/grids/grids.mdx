# La cuadrícula de datos {Platform} más rápida

<PlatformBlock for="React">

Una cuadrícula de datos o tabla de datos {Platform} es un componente utilizado para mostrar datos tabulares en una serie de filas y columnas. Las cuadrículas de datos, también conocidas como tablas, son muy conocidas en el mundo de escritorio con software popular como Microsoft Excel. Aunque las cuadrículas han estado disponibles en plataformas de escritorio durante mucho tiempo, solo recientemente se han convertido en parte de las UI de aplicaciones web, como la UI {Platform}. Las cuadrículas y tablas modernas pueden ser complejas y pueden incluir una variedad de características y funcionalidades, incluido el enlace de datos, la edición, el filtrado similar a Excel, la ordenación personalizada, la agrupación, la reordenación de filas, la inmovilización de filas y columnas, la agregación de filas y la exportación a formatos Excel, CSV y pdf.

</PlatformBlock>

<PlatformBlock for="WebComponents">

Una cuadrícula de datos {Platform}, también conocida como tabla de datos {Platform}, es un componente de visualización de datos que le permite enlazar y mostrar sus datos como una serie de filas y columnas. Son relativamente nuevas en el mundo de las interfaces de usuario de aplicaciones web, aunque han estado disponibles en plataformas de escritorio durante bastante tiempo en muchas formas diferentes; por ejemplo, Microsoft Excel. Estas cuadrículas y tablas generalmente poseen la capacidad de ser muy complejas, incluida funcionalidad como selección de celdas y filas, filtrado, edición, ocultación de filas y columnas, resúmenes de columnas, búsqueda y la capacidad de mostrar sus datos en columnas específicas por tipo.

</PlatformBlock>

<PlatformBlock for="Blazor">

Una tabla de datos {Platform}, más comúnmente conocida como cuadrícula de datos {Platform}, es un componente que le permite visualizar sus datos enlazados como una serie de filas y columnas. Estas cuadrículas de datos han estado accesibles durante bastante tiempo en plataformas relacionadas con el escritorio, como WPF y Windows Forms, pero solo más recientemente han pasado a estar disponibles para el desarrollo de aplicaciones web. Las aplicaciones web modernas utilizan tablas y cuadrículas de datos con muchas características, con la capacidad de realizar enlace de datos, selección de celdas y filas, edición de celdas y filas y validación de errores, resúmenes de columnas, agrupación interactiva al estilo de Outlook, filtrado al estilo de Microsoft Excel y filas y columnas virtualizadas para que potencialmente pueda cargar millones de registros.

</PlatformBlock>

## ¿Por qué usar una cuadrícula de datos {Platform}?

Las cuadrículas de datos {Platform} son esenciales en casos de uso en los que se deben almacenar y ordenar rápidamente grandes volúmenes de datos. Esto puede incluir sectores como servicios financieros, banca y seguros, que a menudo utilizan datos de gran volumen y alta velocidad. El éxito de estas empresas depende de las características, la funcionalidad y el rendimiento de estas cuadrículas de datos. Cuando las decisiones bursátiles deben tomarse en microsegundos, por ejemplo, es imperativo que la cuadrícula de datos funcione sin retrasos ni parpadeos.

Sus usuarios experimentarán un desplazamiento similar al de Excel, con velocidad empresarial —sin retrasos, parpadeo de pantalla ni demora visual—, lo que le brinda la mejor experiencia de usuario (UX) sin comprometer el rendimiento.
