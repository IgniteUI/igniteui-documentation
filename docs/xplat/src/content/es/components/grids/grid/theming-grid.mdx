# Tema de la cuadrícula {Platform}

Con el CSS de temas de {Platform}, la cuadrícula {ProductName} se puede personalizar fácilmente para que coincida con la identidad de su marca.
Además de los temas y paletas predefinidos, puede personalizar aún más la apariencia de su cuadrícula de datos mediante el uso de un conjunto alternativo de propiedades personalizadas de CSS.

<Sample src="/grids/grid/overview-dark" height={500} alt="{Platform} Grid Dark Theme Example" />

<PlatformBlock for="Blazor">

```razor
<head>
    <link href="_content/IgniteUI.Blazor/themes/grid/dark/bootstrap.css" rel="stylesheet" />
</head>
```

</PlatformBlock>

## Propiedades de temas disponibles

| Nombre de propiedad                      | Tipo                   | Descripción                                                                                  |
| ---------------------------------------- | ---------------------- | -------------------------------------------------------------------------------------------- |
| --grid-elevation                         | número entre 0-24      | El nivel de elevación que se utilizará para la cuadrícula                                    |
| --drag-elevation                         | número entre 0-24      | El nivel de elevación que se utilizará para elementos móviles (p. ej., encabezado de columna) |
| --grouparea-color                        | color                  | El color del texto del área de agrupación de la cuadrícula                                   |

| --drop-area-on-drop-background           | color                  | El color de fondo del área de colocación al colocar                                          |
| --header-background                      | color                  | El color de fondo del encabezado de la cuadrícula                                            |
| --header-text-color                      | color                  | El color del texto del encabezado de la cuadrícula                                           |

| --sorted-header-icon-color               | color                  | El color del icono del encabezado de tabla ordenado                                          |
| --header-border-width                    | border-width           | El ancho de borde utilizado para los bordes del encabezado.                                  |
| --header-border-style                    | border-style           | El estilo de borde utilizado para los bordes del encabezado. Puede ser 'dotted', 'dashed' o 'solid' |

| --grid-border-color                      | color                  | El color del borde que rodea la cuadrícula.                                                  |
| --content-background                     | color                  | El color de fondo del cuerpo de la cuadrícula.                                               |
| --content-text-color                     | color                  | El color del texto del cuerpo de la cuadrícula.                                              |
| --row-odd-background                     | color                  | El color de fondo de las filas impares.                                                      |
| --row-even-background                    | color                  | El color de fondo de las filas pares.                                                        |
| --row-odd-text-color                     | color                  | El color del texto de las filas impares.                                                     |
| --row-even-text-color                    | color                  | El color del texto de las filas pares.                                                       |

| --cell-selected-within-background        | color                  | El fondo de la celda seleccionada dentro de una fila/columna seleccionada.                   |

| --row-selected-hover-background          | color                  | El fondo al pasar el cursor sobre la fila seleccionada.                                      |

| --edit-mode-color                        | color                  | El color del texto en modo de edición                                                        |

| --cell-edited-value-color                | color                  | El color del valor editado de la celda.                                                      |
| --cell-disabled-color                    | color                  | El color del texto de una celda deshabilitada.                                               |
| --resize-line-color                      | color                  | El color de la línea de cambio de tamaño del encabezado de la cuadrícula.                    |
| --drop-indicator-color                   | color                  | El color del indicador de colocación.                                                        |
| --group-label-column-name-text           | color                  | El color del texto del nombre de columna de la fila de grupo de la cuadrícula.               |
| --group-label-icon                       | color                  | El color del icono de la fila de grupo de la cuadrícula.                                     |
| --group-label-text                       | color                  | El color del texto de la fila de grupo de la cuadrícula.                                     |
| --expand-all-icon-color                  | color                  | El color del icono de expandir todas las filas de grupo del encabezado de la cuadrícula.     |

| --expand-icon-hover-color                | color                  | El color del icono al pasar el cursor sobre expandir fila de la cuadrícula.                  |
| --active-expand-icon-color               | color                  | El color de fondo del área de colocación al colocar.                                         |
| --active-expand-icon-hover-color         | color                  | El color de fondo del área de colocación al colocar.                                         |
| --group-count-background                 | color                  | El color de fondo de la insignia de recuento de la fila de grupo de la cuadrícula.           |
| --group-count-text-color                 | color                  | El color del texto de la insignia de recuento de la fila de grupo de la cuadrícula.          |
| --group-row-background                   | color                  | El color de fondo de la fila de grupo de la cuadrícula.                                      |
| --group-row-selected-background          | color                  | El color de fondo del área de colocación al colocar.                                         |
| --filtering-header-background            | color                  | El color de fondo del encabezado de columna filtrada.                                        |
| --filtering-header-text-color            | color                  | El color del texto del encabezado de columna filtrada.                                       |
| --filtering-row-background               | color                  | El color de fondo de la fila de filtrado.                                                    |
| --filtering-row-text-color               | color                  | El color del texto de la fila de filtrado.                                                   |
| --tree-filtered-text-color               | color                  | El color de fondo de la fila de agrupación al enfocar.                                       |
| --body-summaries-background              | color                  | El color de fondo de los grupos de resumen ubicados en el cuerpo.                            |

| --root-summaries-text-color              | color                  | El color del texto de los grupos de resumen ubicados en el pie de página.                    |
| --row-highlight                          | color                  | El color de resaltado de la fila de la cuadrícula.                                           |
