# {Platform} {ComponentTitle} Información general sobre la selección

Con la característica de selección {ProductName} en {Platform} {ComponentTitle}, puede interactuar con los datos y manipularlos fácilmente mediante interacciones sencillas con el ratón. Hay tres modos de selección disponibles:

- Selección de filas
- Selección de celdas
- Selección de columnas

Con la propiedad <ApiLink type="{ComponentName}" member="rowSelection" />, puede especificar:

- Ninguna
- Única
- Selección múltiple

## Opciones de selección {Platform} {ComponentTitle}

<ComponentBlock for="Grid, HierarchicalGrid">

El componente {ProductName} <ApiLink type="{ComponentName}" /> proporciona tres modos de selección diferentes: [selección de filas](row-selection.md), [selección de celdas](cell-selection.md) y [selección de columnas](column-selection.md). De forma predeterminada, solo el modo de **selección multicelda** está habilitado en <ApiLink type="{ComponentName}" />. Para cambiar/habilitar el modo de selección, puede usar las propiedades <ApiLink type="{ComponentName}" member="rowSelection" />, <ApiLink type="{ComponentName}" member="cellSelection" /> o <ApiLink type="Column" member="selectable" />.

</ComponentBlock>

### Selección de filas {Platform} {ComponentTitle}

La propiedad <ApiLink type="{ComponentName}" member="rowSelection" /> le permite especificar las siguientes opciones:

- `None`: la selección de filas estaría deshabilitada para <ApiLink type="{ComponentName}" />.
- `Single`: estaría disponible la selección de una sola fila dentro de <ApiLink type="{ComponentName}" />.
- `Multiple`: la selección de varias filas estaría disponible mediante los selectores de fila, con una combinación de teclas como <kbd>CTRL</kbd> + <kbd>clic</kbd>, o al pulsar <kbd>tecla Espacio</kbd> una vez que una celda tenga el foco.

<ComponentBlock for="TreeGrid, HierarchicalGrid">

- `MultipleCascade`: este es un modo para selección en cascada, que da como resultado la selección de todos los elementos secundarios en el árbol debajo del registro que el usuario selecciona mediante interacción del usuario. En este modo, el estado de selección de un elemento primario depende completamente del estado de selección de sus elementos secundarios.

</ComponentBlock>

> Vaya al [tema de selección de filas](row-selection.md) para obtener más información.

### Selección de celdas {Platform} {ComponentTitle}

La propiedad <ApiLink type="{ComponentName}" member="cellSelection" /> le permite especificar las siguientes opciones:

- `None`: la selección de celdas estaría deshabilitada para <ApiLink type="{ComponentName}" />.
- `Single`: estaría disponible la selección de una sola celda dentro de <ApiLink type="{ComponentName}" />.
- `Multiple`: actualmente, este es el estado predeterminado de la selección en <ApiLink type="{ComponentName}" />. La selección multicelda está disponible al arrastrar el ratón sobre las celdas después de mantener pulsado el botón izquierdo del ratón.

<ComponentBlock for="Grid, TreeGrid, HierarchicalGrid">

> Vaya al [tema de selección de celdas](cell-selection.md) para obtener más información.

</ComponentBlock>

### Selección de columnas {Platform} {ComponentTitle}

La propiedad <ApiLink type="Column" member="selectable" /> le permite especificar las siguientes opciones para cada <ApiLink type="Column" />. La selección de columna correspondiente se habilitará o deshabilitará si esta propiedad se establece en true o false, respectivamente.

Esto conduce a las tres variaciones siguientes:

- Selección única: <kbd>clic del ratón</kbd> sobre la celda de columna.
- Selección de varias columnas: mantenga presionado <kbd>CTRL</kbd> + <kbd>clic del ratón</kbd> sobre las celdas de columna.
- Selección de rango de columnas: mantenga presionado <kbd>MAYÚS</kbd> + <kbd>clic del ratón</kbd> para seleccionar todo lo que hay entre medias.

## Menú contextual {Platform} {ComponentTitle}

Con el evento <ApiLink type="{ComponentName}" member="contextMenu" />, puede agregar un menú contextual personalizado para facilitar su trabajo con <ApiLink type="{ComponentName}" />. Al hacer **clic con el botón derecho** en el cuerpo de la cuadrícula, el evento emite la celda en la que se desencadena. El **menú contextual** funcionará con la celda emitida.

Si hay una **selección multicelda**, añadiremos lógica que comprobará si la celda seleccionada está en el área de la selección multicelda. Si lo está, también emitiremos los valores de las celdas seleccionadas.

Básicamente, la función principal tendrá este aspecto:

<PlatformBlock for="Angular">

```typescript
public rightClick(eventArgs: any) {
     // Prevent the default behavior of the right click
    eventArgs.event.preventDefault();
    this.multiCellArgs = {};
    // If we have multi-cell selection, check if selected cell is within the ranges
    if (this.multiCellSelection) {
        const node = eventArgs.cell.selectionNode;
        const isCellWithinRange = this.grid1.getSelectedRanges().some(range => {
            if (node.column >= range.columnStart &&
                node.column <= range.columnEnd &&
                node.row >= range.rowStart &&
                node.row <= range.rowEnd) {
                return true;
            }
            return false;
        })
        // If the cell is within a multi-cell selection range, bind all the selected cells data
        if (isCellWithinRange) {
            this.multiCellArgs = { data: this.multiCellSelection.data };
        }
    }
    // Set the position of the context menu
    this.contextmenuX = eventArgs.event.clientX;
    this.contextmenuY = eventArgs.event.clientY;
    this.clickedCell = eventArgs.cell;
    // Enable the context menu
    this.contextmenu = true;
}
```

El menú contextual tendrá las siguientes funciones:

- Copiar el *valor* de la celda seleccionada.
- Copiar el *dataRow* de la celda seleccionada.
- Si la celda seleccionada está dentro de un **rango de selección multicelda**, copiar todos los *datos seleccionados*.

//contextmenu.component.ts

public copySelectedCellData(event) {
    const selectedData = { [this.cell.column.field]: this.cell.value };
    this.copyData(JSON.stringify({ [this.cell.column.field]: this.cell.value }));
    this.onCellValueCopy.emit({ data: selectedData });
}

public copyRowData(event) {
    const selectedData = this.cell.row.data ;
    this.copyData(JSON.stringify(this.cell.row.data));
    this.onCellValueCopy.emit({ data: selectedData });
}

public copySelectedCells(event) {
    const selectedData = this.selectedCells.data;
    this.copyData(JSON.stringify(selectedData));
    this.onCellValueCopy.emit({ data: selectedData });
}

    public copySelectedRowData() {
        const selectedData = this.grid.getRowData(this.clickedCell.id.rowID);
        this.copyData(selectedData);
        const selectedDataArea = document.getElementById('selectedArea');
        selectedDataArea.innerText = JSON.stringify(selectedData);

    public copySelectedCellData() {
        const selectedData = this.clickedCell.value;
        this.copyData(selectedData);
        const selectedDataArea = document.getElementById('selectedArea');
        selectedDataArea.innerText = JSON.stringify(selectedData);
        this.toggleContextMenu();
    }


    public copySelectedData() {
        const selectedData = this.grid.getSelectedData();
        this.copyData(selectedData);
        const selectedDataArea = document.getElementById('selectedArea');
        selectedDataArea.innerText = JSON.stringify(selectedData);

        this.toggleContextMenu();
    }

    private copyData(data: any[]) {
        const tempElement = document.createElement('input');
        document.body.appendChild(tempElement);
        tempElement.setAttribute('id', 'temp_id');
        (document.getElementById('temp_id') as HTMLInputElement).value = JSON.stringify(data);
        tempElement.select();
        document.execCommand('copy');
        document.body.removeChild(tempElement);
    }

## Problemas conocidos y limitaciones

<PlatformBlock for="Angular">

- El uso de <ApiLink type="{ComponentName}" /> con la selección habilitada en IE11 requiere la importación explícita del polyfill de array en polyfill.ts de la aplicación {Platform}. IE11 ya no es compatible a partir de la versión 13.0.0.

```typescript
import "core-js/es7/array";
```

</PlatformBlock>

Cuando la cuadrícula no tiene establecido <ApiLink type="{ComponentName}" member="primaryKey" /> y los escenarios de datos remotos están habilitados (cuando la paginación, la ordenación, el filtrado o el desplazamiento desencadenan solicitudes a un servidor remoto para recuperar los datos que se mostrarán en la cuadrícula), una fila perderá el siguiente estado después de que se complete una solicitud de datos:

- Selección de filas
- Expansión/contracción de filas
- Edición de filas
- Fijación de filas
