# Exportación de {Platform} {ComponentTitle}

<PlatformBlock for="Blazor, React, WebComponents">

La {ProductName} {ComponentTitle} proporciona funcionalidad de exportación de datos mediante el componente Exportador de la barra de herramientas de la cuadrícula. Puede exportar los datos mostrados a formatos Excel, CSV o PDF. Las exportaciones a Excel usan el formato de tabla de MS Excel, que admite características como filtrado y ordenación. Para habilitar la exportación, coloque <ApiLink type="GridToolbarExporter" /> dentro de la barra de herramientas de la cuadrícula. De forma predeterminada, todos los formatos de exportación están habilitados.

</PlatformBlock>

<PlatformBlock for="Angular">

Los servicios de exportación de {ProductName} en {Platform} {ComponentTitle} pueden exportar datos a formatos Excel y PDF. La funcionalidad de exportación está encapsulada en las clases <ApiLink type="ExcelExporterService" /> y <ApiLink type="PdfExporterService" />. Las exportaciones a Excel usan el formato de tabla de MS Excel, que admite características como filtrado y ordenación, mientras que las exportaciones a PDF capturan el estado actual de la cuadrícula. Para iniciar una exportación, invoque el método `Export` de <ApiLink type="ExcelExporterService" /> o <ApiLink type="PdfExporterService" /> y pase el componente <ApiLink type="{ComponentName}" /> como primer argumento.

</PlatformBlock>

## Exportación de datos de {ComponentTitle}

Para comenzar a usar los exportadores de IgniteUI para Excel y PDF, primero importe <ApiLink type="ExcelExporterService" /> y/o <ApiLink type="PdfExporterService" /> en el archivo app.module.ts y agregue los servicios al arreglo `providers`:

```ts
// app.module.ts
import { ExcelExporterService, PdfExporterService } from 'igniteui-{Platform}';

@NgModule({
  providers: [ ExcelExporterService, PdfExporterService ]
})

export class AppModule {}
```

<DocsAside type="info">

En v12.2.1 y versiones posteriores, los servicios de exportador se proporcionan en la raíz, lo que significa que ya no necesita declararlos en los proveedores de AppModule.

</DocsAside>

Para iniciar una exportación, puede usar el controlador de un botón en la plantilla de su componente.

```html
<{ComponentTitle} #{ComponentTitle} [data]="localData" [autoGenerate]="true"></{ComponentTitle}>
<button (click)="exportButtonHandler()">Export {ComponentTitle} to Excel</button>
<button (click)="exportPdfButtonHandler()">Export {ComponentTitle} to PDF</button>
```

Puede acceder a los servicios de exportador definiendo argumentos de constructor de tipo <ApiLink type="ExcelExporterService" /> o <ApiLink type="PdfExporterService" />, y el framework {Platform} proporcionará instancias de los servicios. Para exportar datos a Excel o PDF, invoque el método `Export` y pase el componente {ComponentTitle} como primer argumento.

El siguiente código ejecuta los procesos de exportación en el archivo del componente:

```ts
// component.ts
import { ExcelExporterService, ExcelExporterOptions, PdfExporterService, PdfExporterOptions } from 'igniteui-{Platform}';
import { {ComponentName} } from 'igniteui-{Platform}';

@ViewChild('{ComponentName}') public {ComponentName}: {ComponentName};

constructor(private excelExportService: ExcelExporterService, private pdfExportService: PdfExporterService) {
}

public exportButtonHandler() {
  this.excelExportService.export(this.{ComponentName}, new ExcelExporterOptions('ExportedDataFile'));
}

public exportPdfButtonHandler() {
  this.pdfExportService.export(this.{ComponentName}, new PdfExporterOptions('ExportedDataFile'));
}
```

Si todo ha ido bien, debería ver el componente {ComponentTitle} y los dos botones de exportación. Cuando presiona el botón correspondiente, se desencadena el proceso de exportación y el navegador descarga un archivo llamado "ExportedDataFile.xlsx" o "ExportedDataFile.pdf" que contiene los datos del componente <ApiLink type="{ComponentName}" /> en formato Excel o PDF.

</PlatformBlock>

## Exportar todos los datos

Cuando usa operaciones remotas como **paginación**, la cuadrícula podría no tener acceso al conjunto de datos completo de una sola vez. En estos casos, recomendamos usar el [servicio de exportación de Excel](../exporter-excel.md) y pasar toda la colección de datos, si está disponible. Ejemplo:

```ts
public exportButtonHandler() {
  this.excelExportService.exportData(this.localData, new ExcelExporterOptions('ExportedDataFile'));
}
```

Al ofrecer descargas en PDF para datos remotos, considere obtener primero el conjunto de datos completo y luego llamar a `export` para que el documento refleje lo que los usuarios esperan ver.

</PlatformBlock>

## Exportar cuadrícula con encabezados de columna inmovilizados

De forma predeterminada, el servicio Excel Exporter exporta la cuadrícula con encabezados de columna desplazables (no inmovilizados). En muchos escenarios, es posible que desee inmovilizar todos los encabezados en la parte superior del archivo de Excel exportado para que siempre permanezcan a la vista mientras el usuario se desplaza por los registros. Para lograrlo, establezca `ExporterOption` <PlatformBlock for="Angular"><ApiLink type="IGridToolbarExportEventArgs" member="freezeHeaders" /></PlatformBlock><PlatformBlock for="React"><ApiLink type="ExporterOptionsBase" member="freezeHeaders" /></PlatformBlock><PlatformBlock for="WebComponents"><ApiLink type="ExporterOptionsBase" member="freezeHeaders" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="GridToolbarExportEventArgs" member="freezeHeaders" /></PlatformBlock> en `true`.

<DocsAside type="info">

Las exportaciones a PDF incluyen automáticamente la fila de encabezado de columna en la parte superior del documento, de modo que los lectores conserven el mismo contexto cuando abran o impriman el archivo.

</DocsAside>

```ts
public exportButtonHandler() {
```

## Personalización del contenido exportado

En los ejemplos anteriores, el servicio Excel Exporter exporta todos los datos disponibles. Hay situaciones en las que es posible que desee omitir la exportación de una fila o incluso de una columna completa. Para lograrlo, suscríbase a los eventos <ApiLink type="{ComponentName}" member="columnExporting" /> y/o <ApiLink type="{ComponentName}" member="rowExporting" />, que se desencadenan para cada columna y cada fila respectivamente, y cancele el evento estableciendo la propiedad `cancel` del objeto de argumento del evento en `true`.

El siguiente ejemplo excluirá una columna de la exportación si su encabezado es "Age" y si su índice es 1:

```ts
// component.ts

this.excelExportService.columnExporting.subscribe((args: IColumnExportingEventArgs) => {
  if (args.header == 'Age' && args.columnIndex == 1) {
      args.cancel = true;
  }
});
this.excelExportService.export(this.{ComponentTitle}, new ExcelExporterOptions('ExportedDataFile'));
```

Cuando exporta datos desde el componente <ApiLink type="{ComponentName}" />, el proceso de exportación tiene en cuenta características como el filtrado de filas y la ocultación de columnas, y exporta solo los datos visibles en <ApiLink type="{ComponentName}" />. Puede configurar el servicio de exportador para incluir filas filtradas o columnas ocultas estableciendo propiedades en el objeto <ApiLink type="ExcelExporterOptions" /> o <ApiLink type="PdfExporterOptions" />.
