# Acciones de fila en {Platform} {ComponentTitle}

La característica Acciones de fila de {ProductName} en {Platform} {ComponentTitle} permite a los desarrolladores usar un <ApiLink type="ActionStrip" /> y utilizar CRUD para componentes de fila/celda y el anclaje de filas. Hay varios controles de UI predefinidos para estas operaciones que se aplican a una fila específica en <ApiLink type="{ComponentName}" />: edición y anclaje.

## Uso

<PlatformBlock for="Angular">

El primer paso es importar **IgxActionStripModule** en nuestro archivo **app.module.ts**:

```typescript
// app.module.ts
import { IgxActionStripModule } from 'igniteui-angular';

@NgModule({
    imports: [..., IgxActionStripModule],
})
```

</PlatformBlock>

Los componentes de UI de acciones predefinidas son:

- <ApiLink type="GridEditingActions" /> - incluye funcionalidad y UI diseñadas específicamente para la edición de <ApiLink type="{ComponentName}" />. Permite alternar rápidamente el modo de edición para celdas o filas, según la opción <ApiLink type="{ComponentName}" member="rowEditable" />, así como eliminar filas de <ApiLink type="{ComponentName}" />.

- <ApiLink type="GridPinningActions" /> - incluye funcionalidad y UI diseñadas específicamente para el anclaje de filas de <ApiLink type="{ComponentName}" />. Permite anclar filas rápidamente y navegar entre las filas ancladas y sus equivalentes deshabilitadas.

Se agregan dentro de <ApiLink type="{ComponentName}" /> y esto es todo lo necesario para tener un <ApiLink type="ActionStrip" /> que proporcione interacciones predeterminadas.

<PlatformBlock for="Angular">

```html
<{ComponentSelector} [data]="data" [rowEditable]="true" [primaryKey]="'ID'">
    <igx-column *ngFor="let c of columns" [field]="c.field">
    </igx-column>

    <igx-action-strip #actionStrip>
        <igx-grid-pinning-actions></igx-grid-pinning-actions>
        <igx-grid-editing-actions></igx-grid-editing-actions>
    </igx-action-strip>
</{ComponentSelector}>

```

## Implementación personalizada

Estos componentes exponen plantillas que ofrecen flexibilidad para la personalización. Por ejemplo, si desea usar <ApiLink type="ActionStrip" /> para un escenario de Gmail con acciones de fila como **eliminar**, **editar**, etc. Puede simplemente crear un componente de botón con icono, agregarle un evento de clic e insertarlo en <ApiLink type="ActionStrip" />.

<PlatformBlock for="Angular">

```html
<{ComponentSelector}>
    <igx-action-strip #actionstrip>
        <igx-grid-pinning-actions></igx-grid-pinning-actions>
        <button title="Edit" igxButton="icon" igxRipple (click)='startEdit(actionstrip.context)'>
            <igx-icon>edit</igx-icon>
        </button>
        <button title="Delete" igxButton="icon" igxRipple *ngIf='!isDeleted(actionstrip.context)' (click)='actionstrip.context.delete()'>
            <igx-icon>delete</igx-icon>
        </button>
    </igx-action-strip>
</{ComponentSelector}>
```

</PlatformBlock>

<ComponentBlock for="Grid, TreeGrid, HierarchicalGrid">

<PlatformBlock for="Blazor">

```razor
<div class="grid__wrapper">
    <{ComponentSelector} Data=northwindEmployees>

            </IgbGridEditingActions>
        </IgbActionStrip>
```
