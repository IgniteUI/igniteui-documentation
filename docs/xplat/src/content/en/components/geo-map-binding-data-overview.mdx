# {Platform} Data Binding

The {ProductName} map component is designed to display geo-spatial data from shape files and/or geographic locations from data models on geographic imagery maps. The <ApiLink type="GeographicMap" member="dataSource" label="DataSource" /> property of geographic series is used for the purpose of binding to data models.
