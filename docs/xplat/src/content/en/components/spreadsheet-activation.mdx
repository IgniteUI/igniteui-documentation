# {Platform} Spreadsheet Activation

The {Platform} Spreadsheet component exposes properties that allow you to determine the currently active cell, pane, and worksheet in the control. This is helpful as it can help you to determine where the user may be navigating or editing in the control.

## Activation Overview

The activation of the {Platform} <ApiLink pkg="spreadsheet" type="Spreadsheet" /> control is split up between the cells, panes, and worksheets of the current <ApiLink pkg="spreadsheet" type="Workbook" /> of the spreadsheet. The three "active" properties are described below:

- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activeCell" label="ActiveCell" />: Returns or sets the active cell in the spreadsheet. To set it, you must create a new instance of <ApiLink pkg="spreadsheet" type="SpreadsheetCell" /> and pass in information about that cell, such as the column and row or the string address of the cell.
- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activePane" label="ActivePane" />: Returns the active pane in the currently active worksheet of the spreadsheet control.
- <ApiLink pkg="spreadsheet" type="Spreadsheet" member="activeWorksheet" label="ActiveWorksheet" />: Returns or sets the active worksheet in the <ApiLink pkg="spreadsheet" type="Workbook" /> of the spreadsheet control. This can be set by setting it to an existing worksheet in the <ApiLink pkg="spreadsheet" type="Workbook" /> attached to the spreadsheet.

## Code Snippet

The following code snippet shows setting activation of the cell and worksheet in the <ApiLink pkg="spreadsheet" type="Spreadsheet" /> control:

```ts
this.spreadsheet.activeWorksheet = this.spreadsheet.workbook.worksheets(1);

this.spreadsheet.activeCell = new SpreadsheetCell("C5");
```
