# {Platform} Displaying Imagery from Open Street Maps

The {Platform} <ApiLink pkg="maps" type="OpenStreetMapImagery" /> is a free geographic imagery mapping service created collaboratively by OpenStreetMap© contributors from around the world. It provides geographic imagery tiles of the world only in road map style without any configuration options. This geographic imagery service can be accessed directly on <a href="http://www.openstreetmap.org" target="_blank">www.OpenStreetMap.org</a> web site.
By the default, the {ProductName} map component already displays geographic imagery from the Open Street Maps. Therefore, there is no need to configure the control to display geographic imagery from the Open Street Maps.

## Code Snippet
This code example explicitly sets <ApiLink pkg="maps" type="GeographicMap" member="BackgroundContent" label="BackgroundContent" /> of the map component to the <ApiLink pkg="maps" type="OpenStreetMapImagery" /> object which provides geographic imagery from  OpenStreetMap© contributors.

<PlatformBlock for="Angular">

```html
<igx-geographic-map #map
    width="100%"
    height="100%"
    zoomable="true" >
</igx-geographic-map>
```

</PlatformBlock>

<PlatformBlock for="Angular">

```ts
import { IgxGeographicMapComponent } from 'igniteui-angular-maps';
import { IgxOpenStreetMapImagery } from 'igniteui-angular-maps';
// ...
public map: IgxGeographicMapComponent;

const tileSource = new IgxOpenStreetMapImagery();
this.map.backgroundContent = tileSource;
```

</PlatformBlock>

<PlatformBlock for="React">

```ts
import { IgrGeographicMap } from 'igniteui-react-maps';
import { IgrOpenStreetMapImagery } from 'igniteui-react-maps';
// ...
const tileSource = new IgrOpenStreetMapImagery();

const geoMap = new IgrGeographicMap({ name: "geoMap" });
geoMap.backgroundContent = tileSource;
```

</PlatformBlock>
