# {Platform} {ComponentTitle} Clipboard Overview

Copy to clipboard operations are now available in the <ApiLink pkg="grids" type="{ComponentName}" prefixed={false} />. This functionality provides a fast, easy and customizable way to copy data of the {Platform} <ApiLink pkg="grids" type="{ComponentName}" prefixed={false} /> through the current multi cell data select. System Clipboard behavior gives the user ability to copy data from the <ApiLink pkg="grids" type="{ComponentName}" prefixed={false} /> into Excel or other external programs.

## Functionality

Copy behavior is working with the default interaction defined by the browser and operating system. Thus for the copy and paste behaviors, these are:

- Windows/Unix based
  - <kbd>CTRL</kbd> + <kbd>C</kbd> / <kbd>CTRL</kbd> + <kbd>Ins</kbd> as a keyboard shortcut
  - <kbd>CTRL</kbd> + <kbd>V</kbd> / <kbd>SHIFT</kbd> + <kbd>Ins</kbd> as a keyboard shortcut
  - Copy action through the browser menu
- macOS
  - <kbd>CMD</kbd> + <kbd>C</kbd> as a keyboard shortcut
  - <kbd>CMD</kbd> + <kbd>V</kbd> as a keyboard shortcut
  - Copy action through the browser menu


## Limitations

- Both the **cut** and **copy** events are not natively supported in Internet Explorer. The exception is the

**paste** event (IE 11) which is emitted but does not expose the <PlatformBlock for="Angular"><ApiLink pkg="grids" type="IClipboardOptions" kind="interface" label="ClipboardData" prefixed={false} /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink pkg="grids" type="ClipboardOptions" kind="interface" label="ClipboardData" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" type="ClipboardOptions" kind="class" label="ClipboardData" /></PlatformBlock> property in the event.
<DocsAside type="note">
In order to **copy** cells in IE 11, you can use the keyboard selection. Hold the <kbd>SHIFT</kbd> key in order to make a multi-cell selection, press <kbd>CTRL</kbd> + <kbd>C</kbd> in order to copy.
</DocsAside>

- The copy behavior is disabled while the grid is in edit mode.
- The current version of this feature covers only the **copy** from grid behavior. Later on we plan to expose `paste` within grid behavior.

## API Usage

We expose <ApiLink pkg="grids" type="{ComponentName}" member="clipboardOptions" prefixed={false} /> property, which handles the following options:
- <PlatformBlock for="Angular"><ApiLink pkg="grids" type="IClipboardOptions" kind="interface" member="enabled" prefixed={false} /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink pkg="grids" type="ClipboardOptions" kind="interface" member="enabled" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" type="ClipboardOptions" kind="class" member="enabled" /></PlatformBlock> Enables/disables copying of selected cells.
- <PlatformBlock for="Angular"><ApiLink pkg="grids" type="IClipboardOptions" kind="interface" member="copyHeaders" prefixed={false} /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink pkg="grids" type="ClipboardOptions" kind="interface" member="copyHeaders" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" type="ClipboardOptions" kind="class" member="copyHeaders" /></PlatformBlock> Include the associated headers when copying.
- <PlatformBlock for="Angular"><ApiLink pkg="grids" type="IClipboardOptions" kind="interface" member="copyFormatters" prefixed={false} /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink pkg="grids" type="ClipboardOptions" kind="interface" member="copyFormatters" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" type="ClipboardOptions" kind="class" member="copyFormatters" /></PlatformBlock> Apply any existing column formatters to the copied data.
- <PlatformBlock for="Angular"><ApiLink pkg="grids" type="IClipboardOptions" kind="interface" member="separator" prefixed={false} /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink pkg="grids" type="ClipboardOptions" kind="interface" member="separator" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" type="ClipboardOptions" kind="class" member="separator" /></PlatformBlock> The string separator to use the for formatting the data in the clipboard. Default is `/t`

<DocsAside type="note">
Excel can automatically detect text that is separated by tabs (tab-delimited `/t`) and properly paste the data into separate columns. When the paste format doesn't work, and everything you paste appears in a single column, then Excel's delimiter is set to another character, or your text is using spaces instead of tabs.
</DocsAside>

- <PlatformBlock for="WebComponents"><ApiLink pkg="grids" kind="interface" type="GridComponentEventMap" member="gridCopy" label='gridCopy' /></PlatformBlock><PlatformBlock for="React"><ApiLink pkg="grids" kind="class" type="Grid" member="onGridCopy" label='gridCopy' suffix={false} /></PlatformBlock><PlatformBlock for="Angular"><ApiLink pkg="grids" kind="class" type="Grid" member="gridCopy" label='gridCopy' /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink pkg="grids" kind="class" type="Grid" suffix={false} member="GridCopy" label='gridCopy' /></PlatformBlock> Emitted when a copy operation is executed. Fired only if copy behavior is enabled through the <ApiLink pkg="grids" type="{ComponentName}" member="clipboardOptions" prefixed={false} />.
