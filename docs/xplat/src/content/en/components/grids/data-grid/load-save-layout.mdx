# {Platform} Grid Load & Save Layout

The {ProductName} Data Table / Data Grid supports loading and saving the grid layout. This is performed by calling the <ApiLink pkg="grids" type="DataGrid" member="loadLayout" label="LoadLayout" /> and <ApiLink pkg="grids" type="DataGrid" member="saveLayout" label="SaveLayout" /> methods of the grid. These features are useful when an end user can move, sort, and group columns, and wants to preserve the state of the grid and be able to recover the layout and resume work at a later time.
