# {Platform} Bubble Chart

The {ProductName} Bubble Chart is a type of [Scatter Chart](scatter-chart.md) that show markers with variable scaling to represent the relationship among items in several distinct series of data or to plot data items using x and y coordinates. These coordinates of the data point are determined by two numeric data columns. The Bubble Chart draws attention to uneven intervals or clusters of data. This chart is often used to plot scientific data, and can highlight the deviation of collected data from predicted results. The Bubble Chart has many of the characteristics of the [Scatter Marker Chart](scatter-chart.md#{PlatformLower}-scatter-marker-chart) but with the option to have various radius scale sizes.

## {Platform} Bubble Chart Styling

In {Platform} Bubble Chart, you can customize shape of bubble markers using <ApiLink pkg="charts" type="BubbleSeries" member="markerType" label="MarkerType" /> property, their size with <ApiLink pkg="charts" type="BubbleSeries" member="radiusScale" label="RadiusScale" /> property, and their appearance using <ApiLink pkg="charts" type="BubbleSeries" member="markerBrush" label="MarkerBrush" />, <ApiLink pkg="charts" type="BubbleSeries" member="markerOutline" label="MarkerOutline" />, <ApiLink pkg="charts" type="BubbleSeries" member="markerThickness" label="MarkerThickness" /> properties. In addition, you can also color bubble markers based on a data column using <ApiLink pkg="charts" type="BubbleSeries" member="fillMemberPath" label="FillMemberPath" /> and <ApiLink pkg="charts" type="BubbleSeries" member="fillScale" label="FillScale" /> properties.
