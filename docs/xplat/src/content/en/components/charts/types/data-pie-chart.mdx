# {Platform} Data Pie Chart
The {ProductName} Data Pie Chart is a part-to-whole chart that shows how categories (parts) of a data set add up to a total (whole) value. Categories are rendered as sections in a circular, or pie-shaped graph. Each section, or pie slice, has an arc length proportional to its underlying data value. Categories are shown in proportion to other categories based on their value percentage to the total value being analyzed, as parts of 100 or 100%.

## {Platform} Data Pie Chart Example
You can create the {Platform} Pie Chart in the <ApiLink type="DataPieChart" /> by binding your data items with a string and a numeric data value. These data values will add up to a value of 100% of visualization.

## {Platform} Data Pie Chart Recommendations
Pie Charts are appropriate for small data sets and are easy to read at a glance. Pie charts are just one type of part-to-whole visualization such as Doughnut (Ring) Chart, Funnel Chart, Stacked Area Chart, Stacked Bar Chart, and Treemap.

The {Platform} Data Pie Chart includes interactive features that give the viewer tools to analyze data, like:

- Legends
- Slice Selection
- Slice Highlighting
- Chart Animations

Best Practices for a Pie Chart:

- Comparing slices or segments as percentage values in proportion to a total value or whole.
- Showing how a group of categories is broken into smaller segments.
- Presenting small, non-hierarchical data sets (less than 6 to 8 segments of data).
- Ensuring data segments add up to 100%.
- Arranging the order of data from largest (highest) to smallest (least).
- Using standard presentation techniques such as starting in the 12 o'clock position and continuing clockwise.
- Ensuring the color palette is distinguishable for segments/slices of the parts.
- Considering data labels in segments vs. legends for ease of reading.
- Choosing an alternative chart to Pie such as Bar or Ring based on ease of comprehension.
- Avoiding positioning multiple pie charts next to each other for comparative analysis.

Do Not Use Pie Chart When:

- Comparing change over time —use a Bar, Line or Area chart.
- Requiring precise data comparison —use a Bar, Line or Area chart.
- You have more than 6 or 8 segments (high data volume) — consider a Bar, Line or Area chart if it works for your data story.
- It would be easier for the viewer to perceive the value difference in a Bar chart.

## {Platform} Data Pie Chart Legend
Legends are used to show information about each point, to know about its contribution towards the total sum.

In order to display a legend next to the pie chart an ItemLegend needs to be created and assigned to the <ApiLink type="DataPieChart" member="legend" label="Legend" /> property. The ItemLegend will display its items in vertical orientation as a default, but this can be changed by setting its <ApiLink type="DataPieChart" member="orientation" label="Orientation" /> property.

The labels shown on the legend will display the same content as the label that is shown for each slice in the <ApiLink type="DataPieChart" /> by default, but this can be modified by utilizing the <ApiLink type="DataPieChart" member="legendSliceLabelContentMode" label="LegendSliceLabelContentMode" /> property on the chart. This exposes an enumeration that allows you to show the label, value, percentage, or any combination of those as the legend's content for each slice in the chart.

You can also modify the ItemLegend badge. By default, it appears as a filled circle corresponding to the color of the associated chart slice. You can configure this by using the <ApiLink type="DataPieChart" member="legendItemBadgeShape" label="LegendItemBadgeShape" /> property on the chart, and you can set this to be a circle, line, bar, column, and more.

## {Platform} Pie Chart Others Category
Sometimes, the underlying data for the pie chart will contain many items with small values. In this case, the Others category will permit automatic aggregation of several data values into a single slice.

The Others category in the <ApiLink type="DataPieChart" /> has three main, configurable properties - <ApiLink type="DataPieChart" member="othersCategoryType" label="OthersCategoryType" />, <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" />, and <ApiLink type="DataPieChart" member="othersCategoryText" label="OthersCategoryText" /> that allow you to configure how the Others slice in the chart is shown. These are each described below:

The <ApiLink type="DataPieChart" member="othersCategoryType" label="OthersCategoryType" /> property works in tandem with the <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" /> property of the <ApiLink type="DataPieChart" />. For the <ApiLink type="DataPieChart" member="othersCategoryType" label="OthersCategoryType" />, you can define whether you want the <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" /> to be evaluated as a number or a percentage. For example, if you decide on number and set the <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" /> to 5, any slices that have a value less than 5 will become part of the Others category. Using the same value of 5 with a percent type, any values that are less than 5 percent of the total values of the <ApiLink type="DataPieChart" /> will become part of the Others category.

To get the underlying data items that are contained within the Others slice in the chart, you can utilize the <ApiLink type="DataPieChart" member="getOthersContext" label="GetOthersContext" /> method on the chart. This return type of this method is an <ApiLink type="OthersCategoryContext" /> which exposes an <ApiLink type="OthersCategoryContext" member="items" label="Items" /> property. The <ApiLink type="OthersCategoryContext" member="items" label="Items" /> property returns an array that will contain the items in the Others slice. Additionally, when clicking the Others slice, the `Item` property of the event arguments for the `SeriesClick` event will be will also return this <ApiLink type="OthersCategoryContext" />.

By default, the Others slice will be represented by a label of "Others." You can change this by modifying the <ApiLink type="DataPieChart" member="othersCategoryText" label="OthersCategoryText" /> property of the chart.

### {Platform} Styling the Others Slice
You can style the aggregated Others slice separately from other slices by using these properties:

- <ApiLink type="DataPieChart" member="othersCategoryBrush" label="OthersCategoryBrush" />
  Sets the fill (brush) used for the Others slice.

- <ApiLink type="DataPieChart" member="othersCategoryOutline" label="OthersCategoryOutline" />
  Sets the outline (stroke) used for the Others slice.

These properties only affect the Others slice (when it exists). All other slices continue to use the normal palette and item-wise coloring behavior.

<DocsAside type="info">
The Others slice is only rendered when the chart is configured to create it (for example, with <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" /> greater than `0` and an appropriate <ApiLink type="DataPieChart" member="othersCategoryType" label="OthersCategoryType" />). If the Others slice is not present, <ApiLink type="DataPieChart" member="othersCategoryBrush" label="OthersCategoryBrush" /> and <ApiLink type="DataPieChart" member="othersCategoryOutline" label="OthersCategoryOutline" /> have no visible effect.
</DocsAside>

If you want to ensure that the Others category does not show up in the <ApiLink type="DataPieChart" />, you can set the <ApiLink type="DataPieChart" member="othersCategoryThreshold" label="OthersCategoryThreshold" /> to 0.

## {Platform} Data Pie Chart Selection
The <ApiLink type="DataPieChart" /> supports slice selection by mouse click on the slices plotted in the chart. This can be configured by utilizing the <ApiLink type="DataPieChart" member="selectionBehavior" label="SelectionBehavior" /> and <ApiLink type="DataPieChart" member="selectionMode" label="SelectionMode" /> properties of the chart, described below:

The main two options of the <ApiLink type="DataPieChart" member="selectionBehavior" label="SelectionBehavior" /> are <ApiLink type="SeriesSelectionBehavior" member="PerDataItemSingleSelect" label="PerDataItemSingleSelect" /> and <ApiLink type="SeriesSelectionBehavior" member="PerDataItemMultiSelect" label="PerDataItemMultiSelect" />, which will enable single and multiple selection, respectively.

The <ApiLink type="DataPieChart" member="selectionMode" label="SelectionMode" /> property exposes an enumeration that determines how the pie chart slices respond to being selected. The following are the options of that enumeration and what they do:

- <ApiLink type="SeriesSelectionMode" member="Brighten" label="Brighten" />: The selected slices will be highlighted.
- <ApiLink type="SeriesSelectionMode" member="FadeOthers" label="FadeOthers" />: The selected slices will remain their same color and others will fade.
- <ApiLink type="SeriesSelectionMode" member="FocusColorFill" label="FocusColorFill" />: The selected slices will change their background to the FocusBrush of the chart.
- <ApiLink type="SeriesSelectionMode" member="FocusColorOutline" label="FocusColorOutline" />: The selected slices will have an outline with the color defined by the FocusBrush of the chart.
- <ApiLink type="SeriesSelectionMode" member="FocusColorThickOutline" label="FocusColorThickOutline" />: The selected slices will have an outline with the color defined by the FocusBrush of the chart. The thickness of this outline can be configured via the Thickness property of the control as well.
- <ApiLink type="SeriesSelectionMode" member="GrayscaleOthers" label="GrayscaleOthers" />: The unselected slices will have a gray color filter applied to them.
- <ApiLink type="SeriesSelectionMode" member="None" label="None" />: There is no effect on the selected slices.
- <ApiLink type="SeriesSelectionMode" member="SelectionColorFill" label="SelectionColorFill" />: The selected slices will change their background to the SelectionBrush of the chart.
- <ApiLink type="SeriesSelectionMode" member="SelectionColorOutline" label="SelectionColorOutline" />: The selected slices will have an outline with the color defined by the SelectionBrush of the chart.
- <ApiLink type="SeriesSelectionMode" member="SelectionColorThickOutline" label="SelectionColorThickOutline" />: The selected slices will have an outline with the color defined by the FocusBrush of the chart. The thickness of this outline can be configured via the Thickness property of the control as well.
- <ApiLink type="SeriesSelectionMode" member="ThickOutline" label="ThickOutline" />: The selected slices will apply an outline with the thickness dependent on the Thickness property of the chart.

When a slice is selected, its underlying data item will be added to the SelectedSeriesItems collection of the chart. As such, the DataPieChart exposes the SelectedSeriesItemsChanged event to detect when a slice has been selected and this collection is changed.

## {Platform} Data Pie Chart Highlighting
The <ApiLink type="DataPieChart" /> supports mouse over highlighting, as well as a highlighting overlay that can be configured by providing a separate data source.

First, the <ApiLink type="DataPieChart" member="highlightingBehavior" label="HighlightingBehavior" /> enumerated property determines how a slice will be highlighted. The following are the options of that property and what they do:

- <ApiLink type="SeriesHighlightingBehavior" member="DirectlyOver" label="DirectlyOver" />: The slices are only highlighted when the mouse is directly over them.
- <ApiLink type="SeriesHighlightingBehavior" member="NearestItems" label="NearestItems" />: The nearest slice to the mouse position will be highlighted.
- <ApiLink type="SeriesHighlightingBehavior" member="NearestItemsAndSeries" label="NearestItemsAndSeries" />: The nearest slice and series to the mouse position will be highlighted.
- <ApiLink type="SeriesHighlightingBehavior" member="NearestItemsRetainMainShapes" label="NearestItemsRetainMainShapes" />: The nearest items to the mouse position will be highlighted and the main shapes of the series will not be de-emphasized.

The <ApiLink type="DataPieChart" member="highlightingMode" label="HighlightingMode" /> enumerated property determines how the data pie chart slices respond to being highlighted. The following are the options of that property and what they do:

- <ApiLink type="SeriesSelectionMode" member="Brighten" label="Brighten" />: The series will have its color brightened when the mouse position is over or near it.
- <ApiLink type="SeriesSelectionMode" member="FadeOthers" label="FadeOthers" />: The series will retain its color when the mouse position is over or near it, while the others will appear faded.
- <ApiLink type="SeriesSelectionMode" member="None" label="None" />: The series and slices will not be highlighted.

The following example demonstrates the mouse highlighting behaviors of the <ApiLink type="DataPieChart" /> component:

In addition to the mouse highlighting, the <ApiLink type="DataPieChart" /> exposes a highlight filter capability that can display a subset of your data. This is applied by specifying a <ApiLink type="DataPieChart" member="highlightedDataSource" label="HighlightedDataSource" /> for the control and by setting the <ApiLink type="DataPieChart" member="highlightedValuesDisplayMode" label="HighlightedValuesDisplayMode" /> property to `Overlay`. The <ApiLink type="DataPieChart" member="highlightedDataSource" label="HighlightedDataSource" /> expects a subset of the data assigned to the <ApiLink type="DataPieChart" member="dataSource" label="DataSource" /> property of the <ApiLink type="DataPieChart" />.

When these conditions are met, the values of the subset will be highlighted, while the remainder of the full set of data will be faded - effectively creating a highlight for the subset and allowing easier visualization of a subset of your data within the same control.

## {Platform} Data Pie Chart Animation
The <ApiLink type="DataPieChart" /> supports animating its slices into view, as well as when a value changes.

You can set the <ApiLink type="DataPieChart" member="isTransitionInEnabled" label="IsTransitionInEnabled" /> property to **true** to have the pie chart animate into view. The type of animation performed can be configured by setting the <ApiLink type="DataPieChart" member="transitionInMode" label="TransitionInMode" /> enumerated property to the type of animation you would like to see. Additionally, you can also set the <ApiLink type="DataPieChart" member="transitionInSpeedType" label="TransitionInSpeedType" /> property to scale with index, value, normal, or randomized. The duration of this animation can be controlled by the <ApiLink type="DataPieChart" member="transitionInDuration" label="TransitionInDuration" /> property, which takes a `TimeSpan`.

If you would like to animate data changes, this can also be done by setting the <ApiLink type="DataPieChart" member="animateSeriesWhenAxisRangeChanges" label="AnimateSeriesWhenAxisRangeChanges" /> property to **true**. The duration of this change can be configured by setting the <ApiLink type="DataPieChart" member="transitionDuration" label="TransitionDuration" /> property as well.
