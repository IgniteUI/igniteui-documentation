# {Platform} Chart Data Tooltip

In {ProductName}, the **DataToolTip** displays values and titles of series as well as legend badges of series in a tooltip. In addition, it provides many configuration properties of the <ApiLink pkg="charts" type="DataLegend" /> for filtering series rows and values columns, styling, and formatting values. This tooltip type updates while moving the mouse inside of the plot area of the <ApiLink pkg="charts" type="CategoryChart" />, <ApiLink pkg="charts" type="FinancialChart" />, and <ApiLink pkg="charts" type="DataChart" /> components.

## {Platform} Data Tooltip Properties

All properties of <ApiLink pkg="charts" type="DataToolTipLayer" /> are prefixed with **DataToolTip** and exposed on API of <ApiLink pkg="charts" type="CategoryChart" /> and <ApiLink pkg="charts" type="FinancialChart" /> components. However, you will need to create an instance of <ApiLink pkg="charts" type="DataToolTipLayer" /> and add it to series collection of <ApiLink pkg="charts" type="DataChart" /> component if you want to use it with Radial Charts, Polar Charts, Scatter Charts.

## {Platform} Data Tooltip Elements

The **DataToolTip** displays content using a set of three types of rows and four types of columns.

### {Platform} Data Tooltip Rows

The rows of the **DataToolTip** include the header row, series row(s), and the summary row.

The header row displays the axis label of the point that is hovered, and can be changed using the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipHeaderText" label="DataToolTipHeaderText" /> property.

The series row can actually be a set of rows corresponding to each series plotted in the chart. These rows will display the legend badge, series title, actual/abbreviated value of the the series, and abbreviation symbol and unit, if specified.

Finally, there is a summary row that displays the total of all series values. The default summary title can be changed using the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipSummaryTitleText" label="DataToolTipSummaryTitleText" /> property of the legend. Also, you can use the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipSummaryType" label="DataToolTipSummaryType" /> property to customize whether you display the Total, Min, Max, or Average of series values in the summary row.

### {Platform} Data Tooltip Columns

The columns of the <ApiLink pkg="charts" type="DataToolTipLayer" /> include the title, label, value, and units columns. Each series in the chart can have multiple columns for label, value, and units depending on the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipIncludedColumns" label="DataToolTipIncludedColumns" /> or <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipExcludedColumns" label="DataToolTipExcludedColumns" /> collections of the chart.

The title column displays legend badges and series titles, which come from the `Title` property of the different `Series` plotted in the chart.

The label column displays the name or abbreviation of the different property paths in the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipIncludedColumns" label="DataToolTipIncludedColumns" /> or <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipExcludedColumns" label="DataToolTipExcludedColumns" /> collections of the tooltip.

The value column displays series values as abbreviated text which can be formatted using the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatAbbreviation" label="DataToolTipValueFormatAbbreviation" /> property to apply the same abbreviation for all numbers by setting this property to `Auto` or `Shared`. Alternatively, a user can select other abbreviations such as `Independent`, `Kilo`, `Million`, etc. Precision of abbreviated values is controlled using the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatMinFractions" label="DataToolTipValueFormatMinFractions" /> and <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatMaxFractions" label="DataToolTipValueFormatMaxFractions" /> for minimum and maximum digits, respectively.

The units column displays an abbreviation symbol and/or unit text, which can be set either on the **DataToolTip** by setting the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipUnitsText" label="DataToolTipUnitsText" /> for all columns or using the following properties on each series in the chart:

- Category Series (e.g. ColumnSeries)
  - ValueMemberAsLegendUnit="K"
- Financial Price Series:
  - OpenMemberAsLegendUnit="K"
  - LowMemberAsLegendUnit="K"
  - HighMemberAsLegendUnit="K"
  - CloseMemberAsLegendUnit="K"
- Range Series:
  - LowMemberAsLegendUnit="K"
  - HighMemberAsLegendUnit="K"
- Radial Series:
  - ValueMemberAsLegendUnit="km"
- Polar Series:
  - RadiusMemberAsLegendUnit="km"
  - AngleMemberAsLegendUnit="degrees"

For the above-listed properties, there are corresponding properties ending with **MemberAsLegendLabel** to determine the text in the label columns mentioned previously.

The columns included in the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipIncludedColumns" label="DataToolTipIncludedColumns" /> and <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipExcludedColumns" label="DataToolTipExcludedColumns" /> collections generally correspond to the value paths of your underlying data items, but the financial series has the option to include some special ones in addition to the `High`, `Low`, `Open`, and `Close` paths that are required for the financial series to plot correctly. You have the ability to show `TypicalPrice`, `Change`, and `Volume` options within the tooltip.

## {Platform} Data Tooltip Grouping for Data Chart

`DataLegendGroup` can be set, on all types of series, to a string that will categorize a group of series in Data Legend. Each group will have its own summary row displayed before another group of series is displayed:
By default, DataLegend will hide names of groups, but you can display group names by setting the `GroupRowVisible` property to true. `GroupingMode` should be set to "Grouped" and `LabelDisplayMode` should be set to "Visible" on the Data Tooltip Layer.

## {Platform} Data Tooltip Grouping & Positioning for Category Chart & Financial Chart

You can set <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipGroupingMode" label="DataToolTipGroupingMode" /> property to either `Grouped` or `Individual` to group content for multiple series into single tooltip or separate content for each series in multiple tooltips. In the `Grouped` mode, you can customize where the tooltip is shown by setting the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipGroupedPositionModeX" label="DataToolTipGroupedPositionModeX" /> and <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipGroupedPositionModeY" label="DataToolTipGroupedPositionModeY" /> properties. This essentially allows you to customize the horizontal and vertical alignments of the tooltip and whether you want it to track to the closest series points to the mouse position or pin the tooltip to edge of plot area.

## {Platform} Data Tooltip Value Formatting

The **DataToolTip** provides automatic abbreviation of large numbers using its <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatAbbreviation" label="DataToolTipValueFormatAbbreviation" /> property. This adds a multiplier in the units column such as kilo, million, billion, etc. You can customize the number of fractional digits that are displayed by setting the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatMinFractions" label="DataToolTipValueFormatMinFractions" /> and <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatMaxFractions" label="DataToolTipValueFormatMaxFractions" />. This will allow you to determine the minimum and maximum number of digits that appear after the decimal point, respectively.

## {Platform} Data Tooltip Value Mode

You can change the default decimal display of values within the **DataToolTip** to be currency by changing the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatMode" label="DataToolTipValueFormatMode" /> property of the layer. The **DataToolTip** also exposes the ability to modify the culture of the displayed currency symbol by using its <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueFormatCulture" label="DataToolTipValueFormatCulture" /> property and setting it to its corresponding culture tag. ## {Platform} Data Tooltip Styling

The **DataToolTip** provides properties for styling each type of column. Each of these properties begins with Title, Label, Value, or Units, and you can style the text's color, font, and margin. For example, if you wanted to set the text color of each of these, you would set the <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipTitleTextColor" label="DataToolTipTitleTextColor" />, <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipLabelTextColor" label="DataToolTipLabelTextColor" />, <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipValueTextColor" label="DataToolTipValueTextColor" />, and <ApiLink pkg="charts" type="CategoryChart" member="dataToolTipUnitsTextColor" label="DataToolTipUnitsTextColor" /> properties.
