# {Platform} Chart Markers

In {ProductName}, markers are visual elements that display the values of data points in the chart's plot area. Markers help your end-users immediately identify a data point's value even if the value falls between major or minor grid lines.

## {Platform} Chart Marker Example

In the following example, the [Line Chart](../types/line-chart.md) is comparing the generation of renewable electricity for the countries Europe, China, and USA over the years of 2009 to 2019 with markers enabled by setting the <ApiLink pkg="charts" type="CategoryChart" member="markerType" label="MarkerType" /> property to <ApiLink pkg="charts" kind="enum" type="MarkerType" member="Circle" label="Circle" prefixed={false} /> enum value.

The colors of the markers are also managed by setting the <ApiLink pkg="charts" type="CategoryChart" member="markerBrushes" label="MarkerBrushes" /> and <ApiLink pkg="charts" type="CategoryChart" member="markerOutlines" label="MarkerOutlines" /> properties in the sample below. The markers and <ApiLink pkg="charts" type="CategoryChart" member="chartType" label="CategoryChart.ChartType" /> is configurable in this sample by using the drop-downs as well.
