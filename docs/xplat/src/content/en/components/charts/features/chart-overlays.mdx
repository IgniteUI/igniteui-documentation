# {Platform} Chart Overlays

The {Platform} <ApiLink type="DataChart" /> allows for placement of horizontal or vertical lines at a single numeric value that you define through usage of the <ApiLink type="ValueOverlay" />. This can help you to visualize data such as the mean or median of a particular series.

## {Platform} Value Overlay Properties

Unlike other series types that use a <ApiLink type="ValueOverlay" member="dataSource" label="ItemsSource" /> for data binding, the value overlay uses a <ApiLink type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" pkg="charts" /> property to bind a single numeric value. In addition, the value overlay requires you to define a single <ApiLink type="ValueOverlay" member="axis" label="Axis" /> to use. If you use an X-axis, the value overlay will be a vertical line, and if you use a Y-axis, it will be a horizontal line.

When using a numeric X or Y axis, the <ApiLink type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" pkg="charts" /> property should reflect the actual numeric value on the axis where you want the value overlay to be drawn. When using a category X or Y axis, the <ApiLink type="ValueOverlay" member="valueMemberPath" label="ValueMemberPath" pkg="charts" /> should reflect the index of the category at which you want the value overlay to appear.

When using the value overlay with a numeric angle axis, it will appear as a line from the center of the chart and when using a numeric radius axis, it will appear as a circle.

<ApiLink type="ValueOverlay" /> appearance properties are inherited from <ApiLink type="DataChart" member="Series" label="Series" /> and so <ApiLink type="ValueOverlay" member="brush" label="Brush" /> and <ApiLink type="ValueOverlay" member="thickness" label="Thickness" /> for example are available and work the same way they do with other types of series.

It is also possible to show an axis annotation on a <ApiLink type="ValueOverlay" /> to show the value of the overlay on the owning axis. In order to show this, you can set the <ApiLink type="ValueOverlay" member="isAxisAnnotationEnabled" label="IsAxisAnnotationEnabled" /> property to true.

## {Platform} Value Layer

The {Platform} charting components also expose the ability to use value lines to call out different focal points of your data, such as minimum, maximum, and average values.

Applying the <ApiLink type="ValueLayer" /> in the <ApiLink type="CategoryChart" /> and <ApiLink type="FinancialChart" /> components is done by setting the <ApiLink type="CategoryChart" member="valueLines" label="ValueLines" /> property on the chart. This property takes a collection of the <ApiLink type="ValueLayerValueMode" /> enumeration. You can mix and match multiple value layers in the same chart by adding multiple <ApiLink type="ValueLayerValueMode" /> enumerations to the <ApiLink type="CategoryChart" member="valueLines" label="ValueLines" /> collection of the chart.

In the <ApiLink type="DataChart" />, this is done by adding a <ApiLink type="ValueLayer" /> to the <ApiLink type="DataChart" member="Series" label="Series" /> collection of the chart and then setting the <ApiLink type="ValueLayer" label="ValueMode" /> property to one of the <ApiLink type="ValueLayerValueMode" /> enumerations. Each of these enumerations and what they mean is listed below:

- <ApiLink type="ValueLayerValueMode" member="Auto" label="Auto" />: The default value mode of the <ApiLink type="ValueLayerValueMode" /> enumeration.
- <ApiLink type="ValueLayerValueMode" member="Average" label="Average" />: Applies potentially multiple value lines to call out the average value of each series plotted in the chart.
- <ApiLink type="ValueLayerValueMode" member="GlobalAverage" label="GlobalAverage" />: Applies a single value line to call out the average of all of the series values in the chart.
- <ApiLink type="ValueLayerValueMode" member="GlobalMaximum" label="GlobalMaximum" />: Applies a single value line to call out the absolute maximum value of all of the series values in the chart.
- <ApiLink type="ValueLayerValueMode" member="GlobalMinimum" label="GlobalMinimum" />: Applies a single value line to call out the absolute minimum value of all of the series values in the chart.
- <ApiLink type="ValueLayerValueMode" member="Maximum" label="Maximum" />: Applies potentially multiple value lines to call out the maximum value of each series plotted in the chart.
- <ApiLink type="ValueLayerValueMode" member="Minimum" label="Minimum" />: Applies potentially multiple value lines to call out the minimum value of each series plotted in the chart.

If you want to prevent any particular series from being taken into account when using the <ApiLink type="ValueLayer" /> element, you can set the <ApiLink type="ValueLayer" member="targetSeries" label="TargetSeries" /> property on the layer. This will force the layer to target the series that you define. You can have as many <ApiLink type="ValueLayer" /> elements within a single <ApiLink type="DataChart" /> as you want.

## Chart Overlay Text <Badge variant="preview" />

The {Platform} <ApiLink type="ValueOverlay" />, <ApiLink type="ValueLayer" />, and all Data Annotation Layers can render custom overlay text inside plot area of the DataChart component. You can use this overlay text to annotate important events (e.g. company quarter reports) on x-axis or important values on y-axis in relationship to the layers.

For example, you can use <ApiLink type="DataAnnotationSliceLayer" />, <ApiLink type="ValueOverlay" />, and <ApiLink type="ValueLayer" /> to show overlay text.

### Styling Overlay Text

This code example shows how to style and customize Overlay Text on
the <ApiLink type="DataAnnotationSliceLayer" />, <ApiLink type="ValueOverlay" />, and <ApiLink type="ValueLayer" />.


<PlatformBlock for="Blazor">

```razor
public Series StylingOverlayText()
{
    var annoLayer = new IgbDataAnnotationSliceLayer();

    // styling optional overlay text
    annoLayer.OverlayTextColor = Brushes.White;
    annoLayer.OverlayTextBackground = Brushes.Green;
    annoLayer.OverlayTextBorderColor = Brushes.Black;
    annoLayer.OverlayTextBorderThickness = 1;
    annoLayer.OverlayTextBorderRadius = 4;
    annoLayer.OverlayTextHorizontalMargin = 5;
    annoLayer.OverlayTextHorizontalPadding = 2;
    annoLayer.OverlayTextVerticalMargin = 5;
    annoLayer.OverlayTextVerticalPadding = 2;

    return annoLayer;
}
```

</PlatformBlock>

