# {Platform} Chart Tooltips

In {Platform} charts, tooltips provide details about bound data and they are rendered in popups when the end-user hovers over data points. Tooltips are supported by the <ApiLink type="CategoryChart" />, <ApiLink type="FinancialChart" />, and <ApiLink type="DataChart" /> controls.

## {Platform} Chart Tooltip Types

{Platform} Chart provide three types of tooltips that you can with tooltips enabled by setting the <ApiLink type="CategoryChart" member="toolTipType" label="ToolTipType" /> property. The following example shows the [Column Chart](../types/column-chart.mdx) with a combo-box that you can use to change type of tooltips.

<Sample src="/charts/category-chart/column-chart-with-tooltips" height={500} alt="{Platform} Tooltip Types Example" />




The <ApiLink type="CategoryChart" member="toolTipType" label="ToolTipType" /> property is configurable and can be set to one of the following options:

| Property Value     | Description  |
| -------------------|----------------|
| <ApiLink type="ToolTipType" member="Default" label="Default" />  Tooltip | Display a tooltip for a single item when the pointer is positioned over it.  |
| <ApiLink type="ToolTipType" member="Data" label="Data" /> Tooltip | Display the data tooltips for all series in the chart.  |
| <ApiLink type="ToolTipType" member="Item" label="Item" />  Tooltip    | Display a tooltip for each data item in the category that the pointer is positioned over.  |
| <ApiLink type="ToolTipType" member="Category" label="Category" /> Tooltip | Display a grouped tooltip for all data points in the category that the pointer is positioned over.  |
