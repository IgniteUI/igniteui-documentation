# {Platform} Chart Title and Subtitle

The title and subtitle feature of the chart control allows you to add information to the top section of the {Platform} charts.

## API References

When adding a title or subtitle to the chart control, the content of the chart automatically resizes allowing for the title and subtitle information.

| Property Name         | Property Type   |     Description  |
| ----------------------|------------------|------------|
| <ApiLink type="ChartTitle" />          | string |  Title's text content.  |
| <ApiLink type="CategoryChart" member="titleTextColor" label="TitleTextColor" />      | string |  Title's text. color  |
| <ApiLink type="CategoryChart" member="titleAlignment" label="TitleAlignment" />      | HorizontalAlignment |  Title's horizontal alignment.  |
| <ApiLink type="CategoryChart" member="titleTextStyle" label="TitleTextStyle" />      | string | Title's font style, e.g. Italic Bold 8pt Times New Roman  |
| <ApiLink type="CategoryChart" member="titleTopMargin" label="TitleTopMargin" />      | number | Title's top margin.  |
| <ApiLink type="CategoryChart" member="titleLeftMargin" label="TitleLeftMargin" />     | number | Title's left margin.  |
| <ApiLink type="CategoryChart" member="titleRightMargin" label="TitleRightMargin" />    | number | Title's right margin.  |
| <ApiLink type="CategoryChart" member="titleBottomMargin" label="TitleBottomMargin" />   | number | Title's bottom margin.  |
| <ApiLink type="CategoryChart" member="subtitle" label="Subtitle" />            | string |  Title's text content.  |
| <ApiLink type="CategoryChart" member="subtitleTextColor" label="SubtitleTextColor" />   | string |  Title's text. color  |
| <ApiLink type="CategoryChart" member="subtitleAlignment" label="SubtitleAlignment" />   | HorizontalAlignment |  Title's horizontal alignment.  |
| <ApiLink type="CategoryChart" member="subtitleTextStyle" label="SubtitleTextStyle" />   | string | Title's font style, e.g. Italic Bold 8pt Times New Roman  |
| <ApiLink type="CategoryChart" member="subtitleTopMargin" label="SubtitleTopMargin" />   | number | Title's top margin.  |
| <ApiLink type="CategoryChart" member="subtitleLeftMargin" label="SubtitleLeftMargin" />  | number | Title's left margin.  |
| <ApiLink type="CategoryChart" member="subtitleRightMargin" label="SubtitleRightMargin" /> | number | Title's right margin.  |
| <ApiLink type="CategoryChart" member="subtitleBottomMargin" label="SubtitleBottomMargin" />| number | Title's bottom margin.  |
