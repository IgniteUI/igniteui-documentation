### Initials

If the <ApiLink pkg="core" type="Avatar" member="initials" label="Initials" /> attribute is set all children elements of the avatar will be ignored and the string passed to this attribute will be displayed.

<PlatformBlock for="WebComponents">

```html
<!-- Initials("AZ") will be displayed instead of the icon. -->

<igc-avatar initials="AZ">
  <igc-icon name="home"></igc-icon>
</igc-avatar>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrAvatar initials="AZ">
    <IgrIcon name="home" />
</IgrAvatar>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<!-- Initials("AZ") will be displayed instead of the icon. -->

<IgbAvatar Initials="AZ">
  <IgbIcon Name="home" />
</IgbAvatar>
```

</PlatformBlock>

### Image

The avatar can also display an image when the <ApiLink pkg="core" type="Avatar" member="src" label="Src" /> attribute is assigned a valid URL to a static asset. In that case the <ApiLink pkg="core" type="Avatar" member="initials" label="Initials" /> value will be ignored and children elements will not be rendered.

<PlatformBlock for="WebComponents">

```html
<igc-avatar
  initials="AZ"
  src="https://www.infragistics.com/angular-demos/assets/images/men/1.jpg"
  alt="A photo of a man.">
  <igc-icon name="home"></igc-icon>
</igc-avatar>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrAvatar initials="AZ"
           src="https://static.infragistics.com/xplatform/images/people/men/1.jpg"
           alt="A photo of a man.">
    <IgrIcon name="home" />
</IgrAvatar>
```

</PlatformBlock>


<PlatformBlock for="Blazor">

```razor
<IgbAvatar Initials="AZ"
           Src="https://static.infragistics.com/xplatform/images/people/GUY01.png"
           Alt="A photo of a man.">
  <IgbIcon Name="home" />
</IgbAvatar>
```

</PlatformBlock>

### Shape

The avatar supports three shapes - `circle`, `rounded`, and `square`. The default shape of the avatar is `square` and it can be changed via the `shape` attribute.

<Sample src="/layouts/avatar/shape" height={80} alt="{Platform} Avatar Example" />




### Size

Apart from the shape, the size of the avatar can also be changed by utilizing the `--ig-size` CSS variable. The supported sizes are `small` (default), `medium`, and `large`. The following code snippet shows how to use a different component size:

```css
igc-avatar {
  --ig-size: var(--ig-size-large);
}
```

<Sample src="/layouts/avatar/size" height={130} alt="{Platform} Avatar Example" />




### Styling

The <ApiLink pkg="core" type="Avatar" /> component exposes several CSS parts, giving you full control over its style:

|Name|Description|
|--|--|
| `base` | The base wrapper of the avatar. |
| `initials` | The initials wrapper of the avatar. |
| `image` | The image wrapper of the avatar. |
| `icon` | The icon wrapper of the avatar. |

```css
igc-avatar {
  --ig-avatar-background: var(--ig-success-500);
}

igc-avatar::part(base) {
  --size: 60px;
  color: var(--ig-success-500-contrast);
  border-radius: 20px;
}
```
