## Examples

### Display Time

Use the <ApiLink pkg="core" type="Snackbar" member="displayTime" label="DisplayTime" /> property to configure how long the snackbar component is visible. By default, it's set to 4000 milliseconds.

<Sample src="/notifications/snackbar/display-time" height={230} alt="{Platform} Snackbar Display Time Example" />




### Action Text

By default, the snackbar component is hidden automatically after a period specified by the <ApiLink pkg="core" type="Snackbar" member="displayTime" label="DisplayTime" />. You can use <ApiLink pkg="core" type="Snackbar" member="keepOpen" label="KeepOpen" /> property to change this behavior. In this way, the snackbar will remain visible. Using the snackbar <ApiLink pkg="core" type="Snackbar" member="actionText" label="ActionText" /> you can display an action button inside the component.

<Sample src="/notifications/snackbar/action-text" height={230} alt="{Platform} Snackbar Action Text Example" />




## Styling

The <ApiLink pkg="core" type="Snackbar" /> component exposes several CSS parts to give you full control over its styling:

|Name|Description|
|--|--|
| `base` | The base wrapper of the snackbar component. |
| `message` | The snackbar message. |
| `action` | The default snackbar action button. |
| `action-container` | The area holding the actions. |

```css
igc-snackbar::part(base) {
  background: var(--ig-primary-500);
  border-color: var(--ig-primary-800);
  color: white;
}
```
