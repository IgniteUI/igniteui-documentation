## 使用方法

<PlatformBlock for="WebComponents">

## 例

### バリエーション

{ProductName} Chip は、事前定義された複数のスタイル バリエーションをサポートしています。サポートされている値のいずれか (`Primary`、`Info`、`Success`、`Warning`、または `Danger`) を <ApiLink pkg="core" type="Chip" member="variant" label="Variant" /> プロパティに割り当てることで、バリエーションを変更できます。

<PlatformBlock for="React">

```tsx
<IgrChip variant="success"></IgrChip>
```

</PlatformBlock>

<PlatformBlock for="WebComponents">

```html
<igc-chip variant="success"></igc-chip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbChip Variant="ChipVariant.Success"></IgbChip>
```

</PlatformBlock>

### スロット

公開されているコンポーネント スロットを使用すると、<ApiLink pkg="core" type="Chip" /> のさまざまな部分にカスタム コンテンツを追加できます。コンポーネントには既定の選択アイコンと削除アイコンが用意されていますが、`select` スロットと `remove` スロットを使用してカスタマイズできます。また、`start` スロットと `end` スロットを使用して、メイン コンテンツの前後に追加のコンテンツを追加することもできます。

単純なテキスト、記号、または絵文字を追加する場合は `<span>` 要素を使用し、`<igc-icon>`、`remove`、`select`、および `start` スロットにアイコンを追加する場合は [`end`](../layouts/icon.md) コンポーネントを使用することをお勧めします。

<PlatformBlock for="WebComponents">

```html
<igc-chip selectable removable>
    <igc-icon slot="select" name="verified-account"></igc-icon>
    <igc-icon slot="start" name="brush"></igc-icon>
    Chip
    <igc-icon slot="end" name="blood"></igc-icon>
    <igc-icon slot="remove" name="pacifier"></igc-icon>
</igc-chip>
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
<IgbChip Selectable="true" Removable="true">
    <IgbIcon slot="select" IconName="verified-account" Collection="material"></IgbIcon>
    <IgbIcon slot="start" IconName="brush" Collection="material"></IgbIcon>
    Chip
    <IgbIcon slot="end" IconName="blood" Collection="material"></IgbIcon>
    <IgbIcon slot="remove" IconName="pacifier" Collection="material"></IgbIcon>
</IgbChip>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrChip selectable={true} removable={true}>
    <IgrIcon slot="select" name="verified-account"></IgrIcon>
    <IgrIcon slot="start" name="brush"></IgrIcon>
    <span>Chip</span>
    <IgrIcon slot="end" name="blood"></IgrIcon>
    <IgrIcon slot="remove" name="pacifier"></IgrIcon>
</IgrChip>
```

</PlatformBlock>

## サイズ

`<ApiLink pkg="core" type="Chip" />` CSS 変数を利用することで、--ig-size のサイズを選択できます:

<PlatformBlock for="React">

```tsx

    <span>Chip</span>
</IgrChip>
<IgrChip className="size-medium" selectable={true} removable={true}>
    <span>Chip</span>
</IgrChip>
<IgrChip className="size-large" selectable={true} removable={true}>
    <span>Chip</span>
</IgrChip>
```

</PlatformBlock>

```css
.size-small {
  --ig-size: var(--ig-size-small);
}

.size-medium {
  --ig-size: var(--ig-size-medium);
}

.size-large {
  --ig-size: var(--ig-size-large);
}
```

## スタイル設定

<ApiLink pkg="core" type="Chip" /> コンポーネントは、すべてのスタイル プロパティを変更するために使用できる `base`、`prefix`、`suffix` CSS part を公開しています。

```css
igc-chip::part(base) {
  background: var(--ig-primary-500);
  color: var(--ig-primary-500-contrast);
}

igc-chip::part(suffix) {
  color: var(--ig-gray-400);
}
```
