# {Platform} グリッドのテーマ

{Platform} テーマ設定 css を使用すると、{ProductName} グリッドをブランド アイデンティティに合わせて簡単にカスタマイズできます。
定義済みのテーマとパレットに加えて、別の CSS カスタム プロパティ セットを使用することで、データ グリッドのルック アンド フィールをさらにカスタマイズできます。

<Sample src="/grids/grid/overview-dark" height={500} alt="{Platform} Grid Dark Theme Example" />

<PlatformBlock for="Blazor">

```razor
<head>
    <link href="_content/IgniteUI.Blazor/themes/grid/dark/bootstrap.css" rel="stylesheet" />
</head>
```

</PlatformBlock>

## 使用可能なテーマ設定プロパティ

| プロパティ名                            | 型                   | 説明                                                                                  |
| ---------------------------------------- | ---------------------- | -------------------------------------------------------------------------------------------- |
| --grid-elevation                         | 0～24 の数値    | グリッドに使用するエレベーション レベル                                                 |
| --drag-elevation                         | 0～24 の数値    | 移動可能な要素 (例: 列ヘッダー) に使用するエレベーション レベル                     |
| --grouparea-color                        | 色                  | グリッドのグループ領域のテキスト色                                                               |
| --grouparea-background                   | 色                  | グリッドのグループ領域のテキスト色                                                               |
| --drop-area-text-color                   | 色                  | ドロップ領域のテキスト色                                                                     |
| --drop-area-icon-color                   | 色                  | ドロップ領域のアイコン色                                                                     |
| --drop-area-background                   | 色                  | ドロップ領域の背景色                                                               |
| --drop-area-on-drop-background           | 色                  | ドロップ時のドロップ領域の背景色                                                       |
| --header-background                      | 色                  | グリッド ヘッダーの背景色                                                             |
| --header-text-color                      | 色                  | グリッド ヘッダーのテキスト色                                                                   |
| --header-selected-background             | 色                  | 選択時 (例: 列選択) のグリッド ヘッダーの背景色。                       |
| --header-selected-text-color             | 色                  | 選択時 (例: 列選択) のグリッド ヘッダーのテキスト色。                             |
| --sortable-header-icon-hover-color       | 色                  | 列がソート可能な場合の、グリッド ヘッダー内のホバー時のアイコン色。                          |
| --sorted-header-icon-color               | 色                  | ソート済みテーブル ヘッダーのアイコン色                                                           |
| --header-border-width                    | border-width           | ヘッダー境界線に使用する境界線の幅。                                                    |
| --header-border-style                    | border-style           | ヘッダー境界線に使用する境界線のスタイル。'dotted'、'dashed'、または 'solid' を指定できます               |
| --header-border-color                    | 色                  | ヘッダー境界線に使用する色。                                                           |
| --ghost-header-background                | 色                  | ドラッグされたヘッダーの背景色。                                                         |
| --ghost-header-text-color                | 色                  | ドラッグされたヘッダーのテキスト色。                                                               |
| --ghost-header-icon-color                | 色                  | ドラッグされたヘッダーのアイコン色。                                                               |
| --filtering-background-and               | 色                  | 高度なフィルタリングの "AND" 条件の背景色。                                  |
| --filtering-background-or                | 色                  | 高度なフィルタリングの "AND" 条件がフォーカス/選択されたときの背景色。                |
| --filtering-background-and--focus        | 色                  | 高度なフィルタリングの "OR" 条件の背景色。                                   |
| --filtering-background-or--focus         | 色                  | 高度なフィルタリングの "OR" 条件がフォーカス/選択されたときの背景色。                 |
| --grid-border-color                      | 色                  | グリッドを囲む境界線の色。                                                       |
| --content-background                     | 色                  | グリッド本体の背景色。                                                              |
| --content-text-color                     | 色                  | グリッド本体のテキスト色。                                                                    |
| --row-odd-background                     | 色                  | 奇数行の背景色。                                                            |
| --row-even-background                    | 色                  | 偶数行の背景色。                                                           |
| --row-odd-text-color                     | 色                  | 奇数行のテキスト色。                                                                  |
| --row-even-text-color                    | 色                  | 偶数行のテキスト色。                                                                 |
| --row-selected-background                | 色                  | 選択された行の背景色。                                                           |
| --tree-selected-filtered-row-text-color  | 色                  | 選択/フィルターされた行のテキスト色                                                         |
| --row-selected-text-color                | 色                  | 選択された行のテキスト色。                                                                 |
| --cell-selected-within-background        | 色                  | 選択された行/列内にある選択されたセルの背景。                            |
| --cell-selected-within-text-color        | 色                  | 選択された行/列内にある選択されたセルの色。                                 |
| --row-selected-hover-background          | 色                  | 選択された行のホバー時の背景。                                                           |
| --row-selected-hover-text-color          | 色                  | 選択された行のホバー時のテキスト                                                                   |
| --row-hover-background                   | 色                  | ホバー時の行の背景色。                                                              |
| --row-hover-text-color                   | 色                  | ホバー時の行のテキスト色。                                                                    |
| --row-border-color                       | 色                  | グリッド本体の行間の境界線の色。                                                 |
| --pinned-border-width                    | border-width           | 固定行/列の境界線の幅。                                                 |
| --pinned-border-style                    | border-style           | 固定行/列の境界線のスタイル。'dotted'、'dashed'、または 'solid' を指定できます            |
| --pinned-border-color                    | 色                  | 固定行/列の境界線の色。                                                 |
| --cell-active-border-color               | 色                  | アクティブ (フォーカスされた) セルの境界線の色。                                                       |
| --cell-selected-background               | 色                  | 選択されたセルの背景色。                                                          |
| --tree-selected-filtered-cell-text-color | 色                  | 選択/フィルターされたセルのテキスト色                                                        |
| --cell-selected-text-color               | 色                  | 選択されたセルのテキスト色。                                                                |
| --cell-editing-background                | 色                  | 編集モードのセルの背景色。                                                 |
| --edit-mode-color                        | 色                  | 編集モードのテキスト色                                                                  |
| --edited-row-indicator                   | 色                  | 編集済み行のインジケーターの色。                                                         |
| --cell-new-color                         | 色                  | 新しい未編集セルのテキスト色。グリッドで新しい行を追加するときに使用されます。                  |
| --cell-edited-value-color                | 色                  | セルの編集済み値の色。                                                              |
| --cell-disabled-color                    | 色                  | 無効なセルのテキスト色。                                                           |
| --resize-line-color                      | 色                  | グリッド ヘッダーのサイズ変更線の色。                                                           |
| --drop-indicator-color                   | 色                  | ドロップ インジケーターの色。                                                             |
| --group-label-column-name-text           | 色                  | グリッドのグループ行の列名のテキスト色。                                                   |
| --group-label-icon                       | 色                  | グリッドのグループ行のアイコン色。                                                               |
| --group-label-text                       | 色                  | グリッドのグループ行のテキスト色。                                                               |
| --expand-all-icon-color                  | 色                  | グリッド ヘッダーのすべてのグループ行を展開するアイコンの色。                                            |
| --expand-all-icon-hover-color            | 色                  | グリッド ヘッダーのすべてのグループ行を展開するアイコンのホバー色。                                      |
| --expand-icon-color                      | 色                  | グリッド行の展開アイコンの色。                                                              |
| --expand-icon-hover-color                | 色                  | グリッド行の展開アイコンのホバー色。                                                        |
| --active-expand-icon-color               | 色                  | ドロップ時のドロップ領域の背景色。                                                      |
| --active-expand-icon-hover-color         | 色                  | ドロップ時のドロップ領域の背景色。                                                      |
| --group-count-background                 | 色                  | グリッドのグループ行のカウント バッジの背景色。                                              |
| --group-count-text-color                 | 色                  | グリッドのグループ行のカウント バッジのテキスト色。                                                    |
| --group-row-background                   | 色                  | グリッドのグループ行の背景色。                                                         |
| --group-row-selected-background          | 色                  | ドロップ時のドロップ領域の背景色。                                                      |
| --filtering-header-background            | 色                  | フィルターされた列ヘッダーの背景色。                                          |
| --filtering-header-text-color            | 色                  | フィルターされた列ヘッダーのテキスト色。                                          |
| --filtering-row-background               | 色                  | フィルタリング行の背景色。                                                   |
| --filtering-row-text-color               | 色                  | フィルタリング行のテキスト色。                                                   |
| --tree-filtered-text-color               | 色                  | フォーカス時のグループ化行の背景色。                                                  |
| --body-summaries-background              | 色                  | 本体に配置された集計グループの背景色。                                 |
| --body-summaries-text-color              | 色                  | 本体に配置された集計グループのテキスト色。                                       |
| --root-summaries-background              | 色                  | フッターに配置された集計グループの背景色。                               |
| --root-summaries-text-color              | 色                  | フッターに配置された集計グループのテキスト色。                                     |
| --row-highlight                          | 色                  | グリッド行のハイライト色。                                                                |
| --row-ghost-background                   | 色                  | ドラッグされた行の背景色。                                                     |
| --row-drag-color                         | 色                  | ドラッグ ハンドルの色。                                                                |
| --drop-area-border-radius                | 0～1 の数値 | ドロップ領域に使用する境界線の半径。0～1 の小数、ピクセル、またはパーセントを指定できます。 |
