# 最速の {Platform} グリッド

{ProductName} は、世界最速の仮想化された {Platform} データ グリッドを含む、{Platform} ネイティブで Material ベースの UI コンポーネントの完全なライブラリを提供します。

## {Platform} グリッドの例

この {Platform} グリッドの例では、データ検索とフィルタリング、列の並べ替え、サイズ変更、固定と非表示、行選択、Excel と CSV へのエクスポート、水平および垂直スクロールなど、グリッドに組み込まれているさまざまな機能を活用して、ユーザーが *データ ビュー* をカスタマイズする方法を確認できます。線形プログレス バー インジケーターやスパークラインなどのコンポーネントを含むセル テンプレートの例も用意しています。その他の機能については、この[トピック](#{PlatformLower}-grid-features)を参照してください。

## {Platform} グリッドとは

{Platform} データ グリッドは、一連の行と列で表形式データを表示するために使用されるコンポーネントです。テーブルとも呼ばれるデータ グリッドは、Microsoft Excel などの一般的なソフトウェアにより、デスクトップの世界ではよく知られています。グリッドは長い間デスクトップ プラットフォームで利用できましたが、最近では {Platform} UI などの Web アプリ UI の一部にもなっています。最新のグリッドは複雑であり、データバインディング、編集、Excel のようなフィルタリング、カスタム並べ替え、グループ化、行の並べ替え、行と列の固定、行の集計、Excel、CSV、PDF 形式へのエクスポートなど、さまざまな機能を含むことがあります。

## {Platform} グリッドを使用する理由

{Platform} データ グリッドは、大量のデータを保存し、すばやく並べ替える必要があるユース ケースに不可欠です。これには、大容量かつ高速なデータを頻繁に使用する金融業界や保険業界などが含まれます。多くの場合、これらの企業の成功は、こうしたデータ グリッドの機能性とパフォーマンスに依存しています。たとえば、株式に関する判断をマイクロ秒単位で行う必要がある場合、データ グリッドが遅延やちらつきなしに動作することが不可欠です。

### データ仮想化とパフォーマンス

データ グリッドの列レベルおよび行レベルの仮想化により、{Platform} グリッド内の無制限の行と列をシームレスにスクロールできます。ローカルまたはリモートのデータ ソースのサポートにより、データがどこに存在していても最高のパフォーマンスを得られます。ユーザーは、エンタープライズ レベルの速度で Excel のようなスクロールを体験できます。遅延、画面のちらつき、視覚的な遅れがなく、パフォーマンスを損なうことなく最適なユーザー エクスペリエンス (UX) を提供します。
