# {Platform} グリッド アイコンのカスタマイズ

{ProductName} <ApiLink type="Grid" /> アイコンは、公開されている API メソッドを使用して、別のコレクション セットのカスタム アイコンを使用するようにカスタマイズできます。

<PlatformBlock for="Angular,React,WebComponents">

- <ApiLink type="registerIconFromText" />
- <ApiLink type="registerIcon" />
- <ApiLink type="setIconRef" />

</PlatformBlock>

<PlatformBlock for="Blazor">

- <ApiLink pkg="core" type="Icon" member="RegisterIconFromText(string, string, string)" label="RegisterIconFromText" />
- <ApiLink pkg="core" type="Icon" member="RegisterIcon(string, string, string)" label="RegisterIcon" />
- <ApiLink pkg="core" type="Icon" member="SetIconRefAsync(string, string, IgbIconMeta)" label="SetIconRefAsync" />

</PlatformBlock>

アイコンを登録するとローカルにキャッシュされるため、そのアイコンを名前とコレクション名で参照するコンポーネント間で再利用できます。参照を設定すると、そのアイコンを名前で参照したときに、どのコレクションのどのアイコンが使用されるかが変更されます。

<PlatformBlock for="WebComponents">

```ts
// Add a new 'material' icon called 'filter_list' from string
registerIconFromText("filter_list", '<svg>...</svg>', "material");

// Add a new 'material' icon called 'my-filter_list' from svg url
registerIcon("filter_list", "url" , "material")

// Sets the icon reference to the new registered icon.
setIconRef('filter_list', 'default', {
            name: 'filter_list',
            collection: 'material',
});

```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
@code {
    private IgbIcon icon;

    protected override async Task OnAfterRenderAsync(bool firstRender)
    {
        if (this.icon != null && firstRender)
        {
            this.icon.EnsureReady().ContinueWith(new Action<Task>((e) =>
            {
                // Add a new 'material' icon called 'filter_list' from string
                this.icon.RegisterIconFromText("filter_list", '<svg>...</svg>', "material");

                // Add a new 'material' icon called 'filter_list' from svg url
                this.icon.RegisterIcon("filter_list", "url" , "material")

                // Sets the icon reference to the new registered icon.
                this.icon.SetIconRef("filter_list", "default", new IgbIconMeta()
                {
                    Name = "filter_list",
                    Collection = "material",
                });
            }));
        }
    }
}

```

</PlatformBlock>

## グリッド

| アイコン              | 説明                                                                       |
| --------------------- | -------------------------------------------------------------------------- |
| **add**               | excel-filter メニューでフィルター エントリを追加するために使用されます。  |
| **arrow\_back**       | 列を後方に移動するために、さまざまな UI 要素で使用されます。              |
| **arrow\_drop\_down** | 切り替え可能なメニューを示すために、さまざまなボタンで使用されます。      |
| **arrow\_forward**    | 列を前方に移動するために、さまざまな UI 要素で使用されます。              |
| **cancel**            | 操作をキャンセルするために、さまざまな UI 要素で使用されます。            |
| **chevron\_right**    | excel スタイル フィルタリングのような展開可能なメニューを示すために使用されます。 |
| **close**             | 展開されたメニューを閉じるために使用されます。                            |
| **confirm**           | 操作を確定するために使用されます。                                        |
| **drag\_indicator**   | 項目をドラッグできることを示すハンドルを表示するために使用されます。      |
| **error**             | 編集可能なセルで、誤ったデータ入力を示すために使用されます。              |
| **expand\_more**      | excel フィルタリング メニューで、さらにフィルターを追加できることを示すために使用されます。 |
| **file\_download**    | excel フィルター エクスポーターで使用されます。                           |
| **filter\_**          | さまざまなフィルタリング オペランドに使用されます。                       |
| **group\_work**       | group-by ドロップ領域で使用されます。                                      |
| **hide**              | 列を非表示にするために、さまざまな UI 要素で使用されます。                |
| **import\_export**    | ピボット データ セレクターで移動のために使用されます。                    |
| **input\_clear**      | 入力データをクリアするために、入力フィールドで使用されます。              |
| **next**              | チップ間を移動するために、フィルタリング行メニューで使用されます。        |
| **pin**               | 列のピン固定のために、さまざまな UI 要素で使用されます。                  |
| **prev**              | チップ間を移動するために、フィルタリング行メニューで使用されます。        |
| **remove**            | 削除インジケーターとして、さまざまな UI 要素で使用されます。              |
| **refresh**           | フィルターを再読み込みするために、フィルタリング行メニューで使用されます。 |
| **selected**          | アクティブな選択を示すために、さまざまな UI 要素で使用されます。          |
| **show**              | 列を表示するために、さまざまな UI 要素で使用されます。                    |
| **sort\_asc**         | 並べ替え方向を示すために、さまざまな UI 要素で使用されます。              |
| **sort\_desc**        | 並べ替え方向を示すために、さまざまな UI 要素で使用されます。              |
| **functions**         | ピボット グリッドおよびデータ セレクターで使用されます。                  |
| **table\_rows**       | ピボット グリッド データ セレクターで使用されます。                       |
| **tree\_collapse**    | ツリー状構造で、詳細を少なく表示するために使用されます。                  |
