# {Platform} マスター詳細グリッド

<ApiLink type="Grid" /> コンポーネントは、コンテンツを展開/縮小することで特定の行の追加詳細を表示する詳細テンプレートの指定をサポートします。指定すると、各レコードはマスターとして機能し、展開時に現在のレコードのコンテキスト データを含むカスタマイズ可能な詳細テンプレートが表示されます。

このモードは、マスター詳細形式のデータを階層構造で表示する必要がある場合に便利です。

## 構成

<ApiLink type="Grid" /> をマスター詳細モードで表示するように構成するには、グリッドのテンプレートを指定する必要があります。

<PlatformBlock for="Angular">

```html
<{ComponentSelector}>
         <ng-template GridDetail let-dataItem>
              <!-- Custom detail template content here -->
        </ng-template>
</{ComponentSelector}>
```

</PlatformBlock>

<PlatformBlock for="WebComponents">

```ts
constructor() {
  var grid = this.grid = document.getElementById('grid') as IgcGridComponent;
  this._bind = () => {
    grid.data = this.customersData;
    grid.detailTemplate = this.masterDetailTemplate;
  }
  this._bind();
}
```

</PlatformBlock>

<PlatformBlock for="Blazor">

```razor
    <IgbGrid DetailTemplateScript="DetailTemplate"  AutoGenerate=true  Data=northwindEmployees></IgbGrid>
```

</PlatformBlock>

<PlatformBlock for="React">

```tsx
<IgrGrid detailTemplate={masterDetailTemplate} autoGenerate={false} data={nwindData}></IgrGrid>
```

</PlatformBlock>

テンプレートのコンテキストはマスター レコード データであるため、マスター レコードの値を詳細テンプレートに表示できます。例:

<PlatformBlock for="Angular">

```html
<{ComponentSelector}>
         <ng-template GridDetail let-dataItem>
            <div *ngIf="dataItem.Category">
                <header>{{dataItem.Category.CategoryName}}</header>
                <span>{{dataItem.Category.Description}}</span>
            </div>
        </ng-template>
</{ComponentSelector}>
```

</PlatformBlock>

## キーボード ナビゲーション

- 詳細行にフォーカスがある場合:

  - <kbd>🡑</kbd> - 1 行上に移動し、前の行のセルにフォーカスします。
  - <kbd>🡓</kbd> - 1 行下に移動し、次の行のセルにフォーカスします。
  - <kbd>タブ</kbd> - テンプレート内にフォーカス可能な要素がある場合は、フォーカスを次のフォーカス可能な要素に移動できます。それ以外の場合は次のグリッド行に移動します。
  - <kbd>シフト</kbd> + <kbd>タブ</kbd> - フォーカスを前の行に移動します。

- エキスパンダー付きのデータ行にフォーカスがある場合:
  - <kbd>代替</kbd> + <kbd>🡒</kbd> または <kbd>代替</kbd> + <kbd>🡓</kbd> - 行を展開します。
  - <kbd>代替</kbd> + <kbd>🡐</kbd> または <kbd>代替</kbd> + <kbd>🡑</kbd> - 行を縮小します。

## 既知の問題と制限

| 既知の制限                                                                                                                                                  | 説明                                                                                          |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| カスタム詳細テンプレート内の Tab ナビゲーションでは、次にフォーカスされる要素が表示ビューポートの外にある場合、マスター グリッドのスクロール位置が更新されないことがあります。 | カスタム詳細テンプレート内の Tab ナビゲーションはブラウザーに委ねられます。                          |
| 詳細テンプレートは Excel にエクスポートされません。                                                                                                                    | 詳細テンプレートには任意の種類のコンテンツを含めることができるため、そのままでは Excel にエクスポートできません。 |
| 検索機能では、詳細テンプレートの要素は強調表示されません。                                                                                          |                                                                                                      |
