# {Platform} {ComponentTitle} 列タイプの概要

{Platform} {ComponentTitle} は、**number**、**string**、**date**、**boolean**、**currency**、**percent** の列データ型の既定の処理を提供し、それに基づいて既定テンプレートと編集テンプレートの外観が表示されます。

## {Platform} {ComponentTitle} 既定テンプレート

データ型固有のテンプレートを有効にする場合は、列の <ApiLink type="Column" member="dataType" /> 入力を設定する必要があります。設定しない場合、列の <ApiLink type="Column" member="dataType" /> の既定値がそれであるため、その列は文字列列として扱われます。

以下のセクションでは、各 <ApiLink type="Column" member="dataType" /> の既定テンプレートについて説明します。

### 文字列

この列 <ApiLink type="Column" member="dataType" /> は、セル値の外観や形式を変更しません。

### 数値

<ApiLink type="Column" member="dataType" /> が **number** に設定されている場合、セル値はアプリケーションまたはグリッドの <ApiLink pkg="core" type="DatePicker" member="locale" label="Locale" /> 設定に基づいて書式設定されます。また、<ApiLink type="Column" member="pipeArgs" /> プロパティが指定されている場合も同様です。その後、それらに基づいて数値形式が変更されます。たとえば、次のような変更が行われる場合があります。

- 小数点以下の桁数
- `,` または `.` による小数点区切り文字

### DateTime、Date、Time

日付部分の外観（例: 日、月、年）は、<ApiLink pkg="core" type="DatePicker" member="locale" label="Locale" /> 形式または <ApiLink type="Column" member="pipeArgs" /> 入力に基づいて設定されます。パイプ引数を使用して、カスタムの日付形式またはタイムゾーンを指定できます。

- **format** - 日付を書式設定するための既定値は `'mediumDate'` です。その他に使用可能なオプションは、`'short'`、`'long'`、`'shortDate'`、`'fullDate'`、`'longTime'`、`'fullTime'` などです。
- **timezone** - ユーザーのローカル システム タイムゾーンが既定値です。タイムゾーン オフセット、標準 GMT/UTC、または米国本土のタイムゾーン略語も渡すことができます。世界中の任意の場所に対応する時刻を表示する、さまざまなタイムゾーンの例:

#### セル編集

列タイプに基づくセル編集では、異なるエディターが表示されます。

- `DateTime` - <ApiLink pkg="core" type="DatePicker" /> が使用されます。このエディターは、`DateTime` オブジェクトの一部である入力要素に対するマスクの指示を提供します。
- `Date` - <ApiLink pkg="core" type="DatePicker" /> が使用されます。
- `Time` - <ApiLink pkg="core" type="DateTimeInput" /> が使用されます。

#### フィルタリング

クイック フィルタリング/Excel スタイル フィルタリングでは、上記と同じエディターが使用されます。各タイプが公開するフィルタリング オペランドは次のとおりです。

- `DateTime` および `Date` - 等しい、等しくない、前、後、今日、昨日、今月、先月、来月、今年、昨年、来年、空、空ではない、Null、Null ではない;
- `Time` - 時刻、時刻ではない、前、後、同時または前、同時または後、空、空ではない、Null、Null ではない;

#### 集計

使用可能な集計オペランドは、**Count**、**Earliest**（日付/時刻）、**Latest**（日付/時刻）です。

#### 並べ替え

- `Time`: 列はオブジェクトの時刻部分に基づいて並べ替えられ、ms は無視されます。
- `Date`: 列は日付部分に基づいて並べ替えられ、時刻部分は無視されます。
- `DateTime`: 列は完全な日付に基づいて並べ替えられます。

### 画像

既定テンプレートは、データから渡される値を既定の画像テンプレートの画像ソースとして使用します。既定の画像テンプレートは、アクセシビリティ要件を満たすために画像ファイルの名前を抽出し、画像の `alt` 属性として設定します。表示されるセルのサイズは、レンダリングされた画像のサイズに合わせて調整されます。そのため、大きな画像もそのままレンダリングされ、画像列内の画像と同じ大きさまでグリッド行が大きくなることに注意してください。画像タイプの列では、フィルタリング、並べ替え、グループ化は既定でオフになります。これらを有効にする場合は、データ操作を実行するカスタム ストラテジを提供する必要があります。

### パーセント

既定テンプレートは、基になる数値のパーセント相当を表示します。表示されるセル値は、表示係数 '100' によって乗算された結果です。たとえば、既定の係数が 100 で、セルに渡される "value" が 0.123 の場合、表示されるセル値は "12.3%" になります。

セル編集では、値はデータ ソースの値と同じになります。表示係数は '1' です。セルの編集中は、パーセント値のプレビューがサフィックス要素として表示されます。たとえば、'0.0547' を編集している間、プレビュー要素には '5.47%' と表示されます。

    * `minFractionDigits`: The minimum number of digits after the decimal point. Default is 0.
    * `maxFractionDigits`: The maximum number of digits after the decimal point. Default is 3.
    */
