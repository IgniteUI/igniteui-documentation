<ComponentBlock for="Grid">

</ComponentBlock>

<ComponentBlock for="HierarchicalGrid">

`HierarchicalTransactionService` を <ApiLink type="{ComponentName}" /> と一緒に使用しつつ、各アイランドごとに個別のトランザクション ログを蓄積させるには、代わりにサービス ファクトリを提供する必要があります。`HierarchicalTransactionServiceFactory` としてエクスポートされており、すぐに使用できます。

</ComponentBlock>

```ts
constructor() {
    var grid = this.grid = document.getElementById('grid') as IgcGridComponent;
    var undo = this.undo = document.getElementById('undo') as HTMLButtonElement;
    var redo = this.redo = document.getElementById('redo') as HTMLButtonElement;
    var commit = this.commit = document.getElementById('commit') as HTMLButtonElement;
}

this._bind = () => {
    grid.data = this.data;
    undo.addEventListener('click', this.OnUndoClick);
    redo.addEventListener('click', this.OnRedoClick);
    commit.addEventListener('click', this.OnCommitClick);
}
this._bind();

private OnCommitClick() {
    if (this.grid.transactions.getAggregatedChanges(false).length > 0)
        this.grid.transactions.commit(this.data);
}

private OnUndoClick() {
    if (this.grid.transactions.canUndo)
        this.grid.transactions.undo();
}

private OnRedoClick() {
    if (this.grid.transactions.canRedo)
        this.grid.transactions.redo();
}
```

```typescript
export class GridBatchEditingSampleComponent {

    public redo() {
        /* exit edit mode and commit changes */
        this.grid.endEdit(true);
```

```typescript
export class HierarchicalGridBatchEditingSampleComponent {
    public undo(grid: IgxHierarchicalGridComponent) {
        /* exit edit mode and commit changes */
        grid.endEdit(true);
        grid.transactions.undo();
    }

    public redo(grid: IgxHierarchicalGridComponent) {
        /* exit edit mode and commit changes */
        grid.endEdit(true);
        grid.transactions.redo();
    }

    public commit() {
        this.hierarchicalGrid.transactions.commit(this.data);
        this.childGrid.hgridAPI.getChildGrids().forEach((grid) => {
            grid.transactions.commit(grid.data);
        });
        this.dialogChanges.close();
    }
}
```

<DocsAside type="info">

トランザクション API は編集の終了を処理しないため、自分で処理する必要があります。そうしないと、`{ComponentName}` は編集モードのままになります。これを行う方法の 1 つは、該当するメソッドで `EndEdit` を呼び出すことです。

</DocsAside>

<ComponentBlock for="TreeGrid">

<ApiLink type="{ComponentName}" /> で親ノードを削除する場合には、いくつかの特性があります。階層データを使用している場合、親を削除すると子も削除されます。フラット データを使用している場合は、<ApiLink type="{ComponentName}" /> の `CascadeOnDelete` プロパティを使用して目的の動作を設定できます。このプロパティは、親が削除されたときに子レコードも削除するかどうかを示します（既定では **true** に設定されています）。

</ComponentBlock>

<DocsAside type="info">

<ApiLink type="{ComponentName}" member="rowEditable" /> プロパティを無効にすると、`{ComponentName}` が変更され、セルの変更時にトランザクションが作成され、UI に行編集オーバーレイが表示されなくなります。

</DocsAside>
