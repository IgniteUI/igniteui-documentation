# {Platform} {ComponentTitle} クリップボードの概要

クリップボードへのコピー操作が <ApiLink type="{ComponentName}" /> で利用できるようになりました。この機能は、現在の複数セル データ選択を通じて {Platform} <ApiLink type="{ComponentName}" /> のデータをコピーするための高速で簡単かつカスタマイズ可能な方法を提供します。システム クリップボードの動作により、ユーザーは <ApiLink type="{ComponentName}" /> から Excel またはその他の外部プログラムにデータをコピーできます。

## 機能

コピー動作は、ブラウザーとオペレーティング システムによって定義された既定の操作で動作します。したがって、コピーと貼り付けの動作は次のとおりです。

- Windows/Unix ベース
  - キーボード ショートカットとして <kbd>コントロール</kbd> + <kbd>シー</kbd> / <kbd>コントロール</kbd> + <kbd>挿入</kbd>
  - キーボード ショートカットとして <kbd>コントロール</kbd> + <kbd>V</kbd> / <kbd>シフト</kbd> + <kbd>挿入</kbd>
  - ブラウザー メニューからのコピー操作
- macOS
  - キーボード ショートカットとして <kbd>コマンド プロンプト</kbd> + <kbd>シー</kbd>
  - キーボード ショートカットとして <kbd>コマンド プロンプト</kbd> + <kbd>V</kbd>
  - ブラウザー メニューからのコピー操作

## 制限事項

- **切り取り** イベントと **コピー** イベントはどちらも Internet Explorer でネイティブにサポートされていません。例外は

**貼り付け** イベント (IE 11) で、これは発行されますが、イベント内の <PlatformBlock for="Angular"><ApiLink type="IClipboardOptions" label="ClipboardData" /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink type="ClipboardOptions" label="ClipboardData" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="ClipboardOptions" label="ClipboardData" /></PlatformBlock> プロパティを公開しません。

<DocsAside type="info">

IE 11 でセルを **コピー** するには、キーボード選択を使用できます。複数セル選択を行うには <kbd>シフト</kbd> キーを押したままにし、コピーするには <kbd>コントロール</kbd> + <kbd>シー</kbd> を押します。

</DocsAside>

- グリッドが編集モードの場合、コピー動作は無効になります。
- この機能の現在のバージョンは、グリッドからの **コピー** 動作のみを対象としています。今後、グリッド動作内で `paste` を公開する予定です。

## API の使用方法

次のオプションを処理する <ApiLink type="{ComponentName}" member="clipboardOptions" /> プロパティを公開しています。

- <PlatformBlock for="Angular"><ApiLink type="IClipboardOptions" member="enabled" /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink type="ClipboardOptions" member="enabled" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="ClipboardOptions" member="enabled" /></PlatformBlock> 選択したセルのコピーを有効/無効にします。
- <PlatformBlock for="Angular"><ApiLink type="IClipboardOptions" member="copyHeaders" /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink type="ClipboardOptions" member="copyHeaders" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="ClipboardOptions" member="copyHeaders" /></PlatformBlock> コピー時に関連するヘッダーを含めます。
- <PlatformBlock for="Angular"><ApiLink type="IClipboardOptions" member="copyFormatters" /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink type="ClipboardOptions" member="copyFormatters" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="ClipboardOptions" member="copyFormatters" /></PlatformBlock> コピーされたデータに既存の列フォーマッターを適用します。
- <PlatformBlock for="Angular"><ApiLink type="IClipboardOptions" member="separator" /></PlatformBlock><PlatformBlock for="React,WebComponents"><ApiLink type="ClipboardOptions" member="separator" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="ClipboardOptions" member="separator" /></PlatformBlock> クリップボード内のデータの書式設定に使用する文字列区切り文字です。既定値は `/t` です。

<DocsAside type="info">

Excel は、タブで区切られたテキスト (タブ区切り `/t`) を自動的に検出し、データを別々の列に適切に貼り付けることができます。貼り付け形式が機能せず、貼り付けた内容がすべて単一の列に表示される場合は、Excel の区切り文字が別の文字に設定されているか、テキストがタブではなくスペースを使用しています。

</DocsAside>

- <PlatformBlock for="WebComponents"><ApiLink type="GridComponentEventMap" member="gridCopy" label="gridCopy" /></PlatformBlock><PlatformBlock for="React"><ApiLink type="Grid" member="onGridCopy" label="gridCopy" /></PlatformBlock><PlatformBlock for="Angular"><ApiLink type="Grid" member="gridCopy" label="gridCopy" /></PlatformBlock><PlatformBlock for="Blazor"><ApiLink type="Grid" member="gridCopy" label="gridCopy" /></PlatformBlock> コピー操作が実行されたときに発行されます。コピー動作が <ApiLink type="{ComponentName}" member="clipboardOptions" /> を通じて有効になっている場合にのみ発生します。
