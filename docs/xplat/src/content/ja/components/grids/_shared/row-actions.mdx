# {Platform} {ComponentTitle} の行アクション

{Platform} {ComponentTitle} の {ProductName} 行アクション機能により、開発者は <ApiLink type="ActionStrip" /> を使用し、行/セル コンポーネントの CRUD と行のピン固定を利用できます。<ApiLink type="{ComponentName}" /> 内の特定の行に適用できるこれらの操作 (編集とピン固定) 用に、いくつかの定義済み UI コントロールがあります。

## 使用方法

<PlatformBlock for="Angular">

最初の手順は、**app.module.ts** ファイルで **IgxActionStripModule** をインポートすることです:

```typescript
// app.module.ts
import { IgxActionStripModule } from 'igniteui-angular';

@NgModule({
    imports: [..., IgxActionStripModule],
})
```

</PlatformBlock>

定義済みのアクション UI コンポーネントは次のとおりです:

- <ApiLink type="GridEditingActions" /> - <ApiLink type="{ComponentName}" /> の編集向けに特別に設計された機能と UI が含まれています。<ApiLink type="{ComponentName}" member="rowEditable" /> オプションに応じてセルまたは行の編集モードをすばやく切り替え、<ApiLink type="{ComponentName}" /> の行を削除できます。

- <ApiLink type="GridPinningActions" /> - <ApiLink type="{ComponentName}" /> の行ピン固定向けに特別に設計された機能と UI が含まれています。行をすばやくピン固定し、ピン固定された行と無効化された対応行の間を移動できます。

これらは <ApiLink type="{ComponentName}" /> 内に追加され、デフォルトのインタラクションを提供する <ApiLink type="ActionStrip" /> を使用するために必要なのはこれだけです。

<PlatformBlock for="Angular">

```html
<{ComponentSelector} [data]="data" [rowEditable]="true" [primaryKey]="'ID'">
    <igx-column *ngFor="let c of columns" [field]="c.field">
    </igx-column>

    <igx-action-strip #actionStrip>
        <igx-grid-pinning-actions></igx-grid-pinning-actions>
        <igx-grid-editing-actions></igx-grid-editing-actions>
    </igx-action-strip>
</{ComponentSelector}>

```

</PlatformBlock>

## カスタム実装

これらのコンポーネントはテンプレートを公開しており、カスタマイズに柔軟性を提供します。たとえば、**削除**、**編集** などの行アクションを持つ Gmail シナリオで <ApiLink type="ActionStrip" /> を使用したい場合は、アイコン付きのボタン コンポーネントを作成し、それにクリック イベントを追加して <ApiLink type="ActionStrip" /> に挿入するだけです。

<PlatformBlock for="Angular">

```html
<{ComponentSelector}>
    <igx-action-strip #actionstrip>
        <igx-grid-pinning-actions></igx-grid-pinning-actions>
        <button title="Edit" igxButton="icon" igxRipple (click)='startEdit(actionstrip.context)'>
            <igx-icon>edit</igx-icon>
        </button>
        <button title="Delete" igxButton="icon" igxRipple *ngIf='!isDeleted(actionstrip.context)' (click)='actionstrip.context.delete()'>
            <igx-icon>delete</igx-icon>
        </button>
    </igx-action-strip>
</{ComponentSelector}>
```

</PlatformBlock>

<ComponentBlock for="Grid, TreeGrid, HierarchicalGrid">

<PlatformBlock for="Blazor">

```razor
<div class="grid__wrapper">
    <{ComponentSelector} Data=northwindEmployees>
        <IgbActionStrip @ref=actionstrip>
            <IgbGridPinningActions></IgbGridPinningActions>
            <IgbGridEditingActions
                EditRow="true"
                DeleteRow="true"
                AddRow="true">
            </IgbGridEditingActions>
        </IgbActionStrip>
    </{ComponentSelector}>
</div>
```

</PlatformBlock>
