# {Platform} グリッドのテーマ

{Platform} テーマ設定 CSS を使用すると、{ProductName} グリッドをブランド アイデンティティに合わせて簡単にカスタマイズできます。
定義済みのテーマとパレットに加えて、別の CSS カスタム プロパティ セットを使用することで、データ グリッドの外観と操作感をさらにカスタマイズできます。

## 利用可能なテーマ設定プロパティ

| プロパティ名                           | 型                     | 説明                                                                                         |
| ---------------------------------------- | ---------------------- | -------------------------------------------------------------------------------------------- |
| --grid-elevation                         | 0～24 の数値           | グリッドに使用するエレベーション レベル                                                      |
| --drag-elevation                         | 0～24 の数値           | 移動可能な要素 (例: 列ヘッダー) に使用するエレベーション レベル                              |
| --grouparea-color                        | color                  | グリッド グループ領域のテキスト色                                                            |
| --grouparea-background                   | color                  | グリッド グループ領域のテキスト色                                                            |
| --drop-area-text-color                   | color                  | ドロップ領域のテキスト色                                                                     |
| --drop-area-icon-color                   | color                  | ドロップ領域のアイコン色                                                                     |
| --drop-area-background                   | color                  | ドロップ領域の背景色                                                                         |
| --drop-area-on-drop-background           | color                  | ドロップ時のドロップ領域の背景色                                                             |
| --header-background                      | color                  | グリッド ヘッダーの背景色                                                                    |
| --header-text-color                      | color                  | グリッド ヘッダーのテキスト色                                                                |
| --header-selected-background             | color                  | 選択時 (例: 列選択) のグリッド ヘッダーの背景色。                                            |
| --header-selected-text-color             | color                  | 選択時 (例: 列選択) のグリッド ヘッダーのテキスト色。                                        |
| --sortable-header-icon-hover-color       | color                  | 列が並べ替え可能な場合の、グリッド ヘッダー内のホバー時アイコン色。                         |
| --sorted-header-icon-color               | color                  | 並べ替え済みテーブル ヘッダーのアイコン色                                                    |
| --header-border-width                    | border-width           | ヘッダーの境界線に使用する境界線の幅。                                                       |
| --header-border-style                    | border-style           | ヘッダーの境界線に使用する境界線スタイル。'dotted'、'dashed'、または 'solid' を使用できます |
| --header-border-color                    | color                  | ヘッダーの境界線に使用する色。                                                               |
| --ghost-header-background                | color                  | ドラッグ中のヘッダーの背景色。                                                               |
| --ghost-header-text-color                | color                  | ドラッグ中のヘッダーのテキスト色。                                                           |
| --ghost-header-icon-color                | color                  | ドラッグ中のヘッダーのアイコン色。                                                           |
| --filtering-background-and               | color                  | 高度なフィルタリングの "AND" 条件の背景色。                                                  |
| --filtering-background-or                | color                  | 高度なフィルタリングの "AND" 条件のフォーカス時/選択時の背景色。                            |
| --filtering-background-and--focus        | color                  | 高度なフィルタリングの "OR" 条件の背景色。                                                   |
| --filtering-background-or--focus         | color                  | 高度なフィルタリングの "OR" 条件のフォーカス時/選択時の背景色。                             |
| --grid-border-color                      | color                  | グリッドを囲む境界線の色。                                                                   |
| --content-background                     | color                  | グリッド本体の背景色。                                                                       |
| --content-text-color                     | color                  | グリッド本体のテキスト色。                                                                   |
| --row-odd-background                     | color                  | 奇数行の背景色。                                                                             |
| --row-even-background                    | color                  | 偶数行の背景色。                                                                             |
| --row-odd-text-color                     | color                  | 奇数行のテキスト色。                                                                         |
| --row-even-text-color                    | color                  | 偶数行のテキスト色。                                                                         |
| --row-selected-background                | color                  | 選択行の背景色。                                                                             |
| --tree-selected-filtered-row-text-color  | color                  | 選択/フィルター済み行のテキスト色                                                            |
| --row-selected-text-color                | color                  | 選択行のテキスト色。                                                                         |
| --cell-selected-within-background        | color                  | 選択された行/列内の選択セルの背景。                                                          |
| --cell-selected-within-text-color        | color                  | 選択された行/列内の選択セルの色。                                                            |
| --row-selected-hover-background          | color                  | 選択行のホバー時の背景。                                                                     |
| --row-selected-hover-text-color          | color                  | 選択行のホバー時のテキスト                                                                   |
| --row-hover-background                   | color                  | ホバー行の背景色。                                                                           |
| --row-hover-text-color                   | color                  | ホバー行のテキスト色。                                                                       |
| --row-border-color                       | color                  | グリッド本体の行間の境界線の色。                                                             |
| --pinned-border-width                    | border-width           | 固定された行/列の境界線の幅。                                                                |
| --pinned-border-style                    | border-style           | 固定された行/列の境界線スタイル。'dotted'、'dashed'、または 'solid' を使用できます          |
| --pinned-border-color                    | color                  | 固定された行/列の境界線の色。                                                                |
| --cell-active-border-color               | color                  | アクティブ (フォーカス) セルの境界線の色。                                                    |
| --cell-selected-background               | color                  | 選択セルの背景色。                                                                           |
| --tree-selected-filtered-cell-text-color | color                  | 選択/フィルター済みセルのテキスト色                                                          |
| --cell-selected-text-color               | color                  | 選択セルのテキスト色。                                                                       |
| --cell-editing-background                | color                  | 編集モードのセルの背景色。                                                                   |
| --edit-mode-color                        | color                  | 編集モードのテキスト色                                                                       |
| --edited-row-indicator                   | color                  | 編集された行のインジケーターの色。                                                           |
| --cell-new-color                         | color                  | 新しい未編集セルのテキスト色。グリッドに新しい行を追加する際に使用されます。                |
| --cell-edited-value-color                | color                  | 編集されたセル値の色。                                                                       |
| --cell-disabled-color                    | color                  | 無効化されたセルのテキスト色。                                                               |
| --resize-line-color                      | color                  | グリッド ヘッダーのサイズ変更ラインの色。                                                    |
| --drop-indicator-color                   | color                  | ドロップ インジケーターの色。                                                                |
| --group-label-column-name-text           | color                  | グリッド グループ行の列名テキスト色。                                                        |
| --group-label-icon                       | color                  | グリッド グループ行のアイコン色。                                                            |
| --group-label-text                       | color                  | グリッド グループ行のテキスト色。                                                            |
| --expand-all-icon-color                  | color                  | グリッド ヘッダーのすべてのグループ行を展開するアイコンの色。                                |
| --expand-all-icon-hover-color            | color                  | グリッド ヘッダーのすべてのグループ行を展開するアイコンのホバー色。                          |
| --expand-icon-color                      | color                  | グリッド行の展開アイコンの色。                                                               |
| --expand-icon-hover-color                | color                  | グリッド行の展開アイコンのホバー色。                                                         |
| --active-expand-icon-color               | color                  | ドロップ時のドロップ領域の背景色。                                                           |
| --active-expand-icon-hover-color         | color                  | ドロップ時のドロップ領域の背景色。                                                           |
| --group-count-background                 | color                  | グリッド グループ行のカウント バッジの背景色。                                               |
| --group-count-text-color                 | color                  | グリッド グループ行のカウント バッジのテキスト色。                                           |
| --group-row-background                   | color                  | グリッド グループ行の背景色。                                                                |
| --group-row-selected-background          | color                  | ドロップ時のドロップ領域の背景色。                                                           |
| --filtering-header-background            | color                  | フィルター済み列ヘッダーの背景色。                                                           |
| --filtering-header-text-color            | color                  | フィルター済み列ヘッダーのテキスト色。                                                       |
| --filtering-row-background               | color                  | フィルター行の背景色。                                                                       |
| --filtering-row-text-color               | color                  | フィルター行のテキスト色。                                                                   |
| --tree-filtered-text-color               | color                  | フォーカス時のグループ化行の背景色。                                                         |
| --body-summaries-background              | color                  | 本体に配置された集計グループの背景色。                                                       |
| --body-summaries-text-color              | color                  | 本体に配置された集計グループのテキスト色。                                                   |
| --root-summaries-background              | color                  | フッターに配置された集計グループの背景色。                                                   |
| --root-summaries-text-color              | color                  | フッターに配置された集計グループのテキスト色。                                               |
| --row-highlight                          | color                  | グリッド行のハイライト色。                                                                   |
| --row-ghost-background                   | color                  | ドラッグ中の行の背景色。                                                                     |
| --row-drag-color                         | color                  | ドラッグ ハンドルの色。                                                                      |
| --drop-area-border-radius                | 0～1 の数値            | ドロップ領域に使用する境界線の半径。0～1 の小数、ピクセル、またはパーセントを使用できます。 |
