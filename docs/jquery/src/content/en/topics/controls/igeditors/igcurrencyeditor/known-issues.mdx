# igCurrencyEditor Known Issues

**Known Issues**

Currently there are no known issues or limitations specific to the `igCurrencyEditor`.
Since the control inherits from the `igNumericEditor`, common issues and limitations listed in the [igNumericEditor Known Issues](/ignumericeditor/known-issues.mdx) are also likely to be valid for the Currency editor, unless stated otherwise.
