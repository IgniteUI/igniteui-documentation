# Property Reference (igPopover)


## igPopover Property Reference
### Introduction

This topic explains the properties of the `igPopover`™ control and lists their default values.

### igPopover property reference chart

The following table explains the properties of the `igPopover` control and lists the default values. The properties are listed alphabetically.


| Property | Type | Description | Default Value |
| --- | --- | --- | --- |
| <ApiLink type="igpopover" member="animationDuration" section="options" label="animationDuration" /> | number | Duration of the animated showing/hiding of the popover. | 150 |
| <ApiLink type="igpopover" member="closeOnBlur" section="options" label="closeOnBlur" /> | bool | Controls whether the popover will close on blur or not | true |
| <ApiLink type="igpopover" member="containment" section="options" label="containment" /> | object | The containment setting for the popover. Containment works by specifying an object (like DIV container, for example) whose boundaries will serve to restrict the area within which the popover can be shown. | null |
| <ApiLink type="igpopover" member="contentTemplate" section="options" label="contentTemplate" /> | string\|function | The content for the popover container. If left null the content will be retrieved from the target. | null |
| <ApiLink type="igpopover" member="direction" section="options" label="direction" /> | enumerable | The position of the popover relative to the target element. The direction is the side of the target on which the popover container will be shown. The valid settings are: "auto” “left” ”right” “top” “bottom" | “auto” |
| <ApiLink type="igpopover" member="headerTemplate.closeButton" section="options" label="headerTemplate.closeButton" /> | bool | Controls whether the popover renders a functional Close button. | false |
| <ApiLink type="igpopover" member="headerTemplate.title" section="options" label="headerTemplate.title" /> | string | The content of the popover header. | null |
| <ApiLink type="igpopover" member="height" section="options" label="height" /> | number | Height for the popover. Leave null for auto | null |
| <ApiLink type="igpopover" member="maxHeight" section="options" label="maxHeight" /> | string\|number | Maximum height of the popover if no <ApiLink type="igpopover" member="height" section="options" label="height" /> has been specified. | 200 |
| <ApiLink type="igpopover" member="maxWidth" section="options" label="maxWidth" /> | string\|number | Maximum width of the popover if no <ApiLink type="igpopover" member="width" section="options" label="width" /> has been specified. | 200 |
| <ApiLink type="igpopover" member="position" section="options" label="position" /> | enumerable | The position of the popover relative to the target element in case the popover is larger than the target. If the popover is smaller, it will always be shown in the middle of the visible area on the target element side set as `direction`. The valid settings are: "auto” The popover position will be determined automatically. “balanced” The popover will be positioned at the middle of the target element side set as `direction`. “start” The popover will be positioned at the beginning of the target element side set as `direction`. The beginning of the side is considered its left-hand section at “top”/“bottom” directions and its upper section at “left”/“right” directions. “end" The popover will be positioned at the end (bottom or right) of the target element side set as `direction`. The end of the side is considered its right-hand section at “top”/“bottom” directions and its lower section at “left”/“right” directions. | “auto” |
| <ApiLink type="igpopover" member="selectors" section="options" label="selectors" /> | string | Selectors indicating which items should show popovers. Any jQuery selector is a valid setting and the pre-defined setting is [title] which means that the title attribute of the item will be shown. Customize if you need to display something different than the title attribute or if you need a different selector for event delegation. When changing this option, you will likely need to also change also the <ApiLink type="igpopover" member="contentTemplate" section="options" label="contentTemplate" /> option. | null |
| <ApiLink type="igpopover" member="showOn" section="options" label="showOn" /> | enumerable | The event on which the popover will be shown. The valid settings are: "mouseenter" (fired on mouseover) "click" "focus" (The target elements gets the focus.) Note: The <ApiLink type="igpopover" member="showOn" section="options" label="showOn" /> setting has no effect in touch environment. On touch devices, the popover appears on tap and this activation is not configurable. | "mouseenter" |
| <ApiLink type="igpopover" member="width" section="options" label="width" /> | number | Width of the popover. Leave null for auto. | null |
