# Property Reference (igPopover)


## igPopover Property Reference
### Introduction

This topic explains the properties of the `igPopover`™ control and lists their default values.

### igPopover property reference chart

The following table explains the properties of the `igPopover` control and lists the default values. The properties are listed alphabetically.


| Property | Type | Description | Default Value |
| --- | --- | --- | --- |
| [animationDuration](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:animationDuration) | number | Duration of the animated showing/hiding of the popover. | 150 |
| [closeOnBlur](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:closeOnBlur) | bool | Controls whether the popover will close on blur or not | true |
| [containment](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:containment) | object | The containment setting for the popover. Containment works by specifying an object (like DIV container, for example) whose boundaries will serve to restrict the area within which the popover can be shown. | null |
| [contentTemplate](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:contentTemplate) | string\|function | The content for the popover container. If left null the content will be retrieved from the target. | null |
| [direction](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:direction) | enumerable | The position of the popover relative to the target element. The direction is the side of the target on which the popover container will be shown. The valid settings are: "auto” “left” ”right” “top” “bottom" | “auto” |
| [headerTemplate.closeButton](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:headerTemplate.closeButton) | bool | Controls whether the popover renders a functional Close button. | false |
| [headerTemplate.title](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:headerTemplate.title) | string | The content of the popover header. | null |
| [height](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:height) | number | Height for the popover. Leave null for auto | null |
| [maxHeight](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:maxHeight) | string\|number | Maximum height of the popover if no [height](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:height) has been specified. | 200 |
| [maxWidth](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:maxWidth) | string\|number | Maximum width of the popover if no [width](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:width) has been specified. | 200 |
| [position](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:position) | enumerable | The position of the popover relative to the target element in case the popover is larger than the target. If the popover is smaller, it will always be shown in the middle of the visible area on the target element side set as `direction`. The valid settings are: "auto” The popover position will be determined automatically. “balanced” The popover will be positioned at the middle of the target element side set as `direction`. “start” The popover will be positioned at the beginning of the target element side set as `direction`. The beginning of the side is considered its left-hand section at “top”/“bottom” directions and its upper section at “left”/“right” directions. “end" The popover will be positioned at the end (bottom or right) of the target element side set as `direction`. The end of the side is considered its right-hand section at “top”/“bottom” directions and its lower section at “left”/“right” directions. | “auto” |
| [selectors](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:selectors) | string | Selectors indicating which items should show popovers. Any jQuery selector is a valid setting and the pre-defined setting is [title] which means that the title attribute of the item will be shown. Customize if you need to display something different than the title attribute or if you need a different selector for event delegation. When changing this option, you will likely need to also change also the [`contentTemplate`](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:contentTemplate) option. | null |
| [showOn](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:showOn) | enumerable | The event on which the popover will be shown. The valid settings are: "mouseenter" (fired on mouseover) "click" "focus" (The target elements gets the focus.) Note: The [`showOn`](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:showOn) setting has no effect in touch environment. On touch devices, the popover appears on tap and this activation is not configurable. | "mouseenter" |
| [width](&#123;environment:jQueryApiUrl&#125;/ui.igpopover#options:width) | number | Width of the popover. Leave null for auto. | null |
