# Method Reference (Column Fixing, igGrid)

## Column Fixing Method Reference

Fixed and Non-Fixed Columns areas in the `igGrid`™ control are implemented as a two separate table DOM elements. This requires all grid Document Object Model (DOM) manipulation methods to be duplicated for the Fixed Columns area. Usually, the Fixed Columns area APIs are prefixed with the word fixed.

#### Column Fixing method reference chart

The following table summarizes the purpose and functionality of the `igGrid` methods related to the Column Fixing feature.

Method | Description
-------|------------
<ApiLink type="iggrid" member="allFixedRows" section="methods" label="allFixedRows" /> | Gets the data for all rows in the Fixed Columns area.
<ApiLink type="iggrid" member="cellAt" section="methods" label="cellAt" /> | Gets a specific cell from the grid. Returns the DOM element of the cell at specified location. A new Boolean parameter is added to the API.
<ApiLink type="iggrid" member="fixedBodyContainer" section="methods" label="fixedBodyContainer" /> | Gets the body container of the Fixed Columns area. Returns the DOM of the body element containing the fixed columns.
<ApiLink type="iggrid" member="fixedContainer" section="methods" label="fixedContainer" /> | Gets the container of the Fixed Columns area. Returns the DOM container element of the fixed columns.
<ApiLink type="iggrid" member="fixedFooterContainer" section="methods" label="fixedFooterContainer" /> | Gets the container of the footer in the Fixed Columns area. Returns the DOM container element of the footer in the Fixed Columns area.
<ApiLink type="iggrid" member="fixedFootersTable" section="methods" label="fixedFootersTable" /> | Gets the table of the footer in the Fixed Columns area. Returns the DOM table element of the footer in the Fixed Columns area.
<ApiLink type="iggrid" member="fixedHeaderContainer" section="methods" label="fixedHeaderContainer" /> | Gets the container of the header in the Fixed Columns area. Returns the DOM container element of the header in the Fixed Columns area.
<ApiLink type="iggrid" member="fixedRowAt" section="methods" label="fixedRowAt" /> | Gets a specific row from the Fixed Columns area. Returns the DOM element of the row at the specified index.
<ApiLink type="iggrid" member="fixedRows" section="methods" label="fixedRows" /> | Gets the fixed rows. Returns an array of all DOM’s `&lt;tr&gt;` elements.
<ApiLink type="iggrid" member="fixedTable" section="methods" label="fixedTable" /> | Gets the fixed table. Returns the DOM element of the fixed table.
<ApiLink type="iggrid" member="hasFixedColumns" section="methods" label="hasFixedColumns" /> | Checks whether there are fixed columns in the grid. Returns true or false.
<ApiLink type="iggrid" member="isFixedColumn" section="methods" label="isFixedColumn" /> | Checks whether the column is fixed either by column key or index. Returns true or false.
