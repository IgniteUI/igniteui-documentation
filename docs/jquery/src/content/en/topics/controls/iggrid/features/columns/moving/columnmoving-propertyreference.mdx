# Property Reference (igGrid)


## igGrid Column Moving Properties

The following table explains some of the properties of the `igGrid` Column Moving feature and lists the default and recommended values. The properties are sorted alphabetically.


| Property | Type | Description | Default Value |
| --- | --- | --- | --- |
| [addMovingDropdown](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:addMovingDropdown) | bool | When false, hides the column moving menu for all columns. | true |
| [columnSettings](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings) | array | If column moving is enabled, configures individual columns for which moving will be disabled. The columns are specified through the [`columnSettings.allowMoving`](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.allowMoving) and [`columnSettings.columnIndex`](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnIndex) /[`columnSettings.columnKey`](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnKey) properties. | null |
| [columnSettings.allowMoving](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.allowMoving) | bool | When false, disables column moving for a particular column in the grid. The configured column is specified by either [`columnSettings.columnIndex`](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnIndex) or [`columnSettings.columnKey`](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnKey) . | true |
| [columnSettings.columnIndex](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnIndex) | number | The index of the column that is being configured. Use the column index to specify the column when the grid columns do not have column keys defined (e.g. when [`autoGenerateColumns`](&#123;environment:jQueryApiUrl&#125;/ui.iggrid#options:autoGenerateColumns) is set to true). | null |
| [columnSettings.columnKey](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:columnSettings.columnKey) | string | The column key of the column that is being configured. Use the column key to configure the column when the grid columns have column keys defined (e.g. when the grid columns are defined manually). | null |
| [mode](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:mode) | string | Sets the [Column Moving mode](./00_igGrid_ColumnMoving_Overview.mdx#mode). The valid values are: “immediate” – sets Column Moving to Immediate mode (As the user draggs the header, it is moved and the columns swap animatedly.) “deferred” – sets Column Moving to Deferred mode (Dragging does not move the column header but, instead, arrowheads appear to mark the position where the column would be dropped if the user releases the mouse button.) | “immediate” |
| [moveType](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:moveType) | string | Sets the [Column Moving type](./00_igGrid_ColumnMoving_Overview.mdx#type). The valid values are: “dom” – sets the type to DOM Manipulation (The column will be moved by detaching the column DOM and then re-attaching it back in the DOM tree.) “render” – sets the type to Grid Re-rendering (The grid will be re-rendered with the new column order.) | “dom” |
| [movingDialogAnimationDuration](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:movingDialogAnimationDuration) | number | Sets the timeDuration in milliseconds to of the animation showing or hiding the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). | 200 |
| [movingDialogHeight](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:movingDialogHeight) | number | Sets the height of the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). By default, the height is not set, so that the dialog height stretches to show all the grid columns. | N/A |
| [movingDialogWidth](&#123;environment:jQueryApiUrl&#125;/ui.iggridcolumnmoving#options:movingDialogWidth) | number | Sets the width of the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). | 400 |
