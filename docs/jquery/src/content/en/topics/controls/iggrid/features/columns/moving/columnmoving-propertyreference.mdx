# Property Reference (igGrid)


## igGrid Column Moving Properties

The following table explains some of the properties of the `igGrid` Column Moving feature and lists the default and recommended values. The properties are sorted alphabetically.


| Property | Type | Description | Default Value |
| --- | --- | --- | --- |
| <ApiLink type="iggridcolumnmoving" member="addMovingDropdown" section="options" label="addMovingDropdown" /> | bool | When false, hides the column moving menu for all columns. | true |
| <ApiLink type="iggridcolumnmoving" member="columnSettings" section="options" label="columnSettings" /> | array | If column moving is enabled, configures individual columns for which moving will be disabled. The columns are specified through the <ApiLink type="iggridcolumnmoving" member="columnSettings.allowMoving" section="options" label="columnSettings.allowMoving" /> and <ApiLink type="iggridcolumnmoving" member="columnSettings.columnIndex" section="options" label="columnSettings.columnIndex" /> /<ApiLink type="iggridcolumnmoving" member="columnSettings.columnKey" section="options" label="columnSettings.columnKey" /> properties. | null |
| <ApiLink type="iggridcolumnmoving" member="columnSettings.allowMoving" section="options" label="columnSettings.allowMoving" /> | bool | When false, disables column moving for a particular column in the grid. The configured column is specified by either <ApiLink type="iggridcolumnmoving" member="columnSettings.columnIndex" section="options" label="columnSettings.columnIndex" /> or <ApiLink type="iggridcolumnmoving" member="columnSettings.columnKey" section="options" label="columnSettings.columnKey" /> . | true |
| <ApiLink type="iggridcolumnmoving" member="columnSettings.columnIndex" section="options" label="columnSettings.columnIndex" /> | number | The index of the column that is being configured. Use the column index to specify the column when the grid columns do not have column keys defined (e.g. when <ApiLink type="iggrid" member="autoGenerateColumns" section="options" label="autoGenerateColumns" /> is set to true). | null |
| <ApiLink type="iggridcolumnmoving" member="columnSettings.columnKey" section="options" label="columnSettings.columnKey" /> | string | The column key of the column that is being configured. Use the column key to configure the column when the grid columns have column keys defined (e.g. when the grid columns are defined manually). | null |
| <ApiLink type="iggridcolumnmoving" member="mode" section="options" label="mode" /> | string | Sets the [Column Moving mode](./00_igGrid_ColumnMoving_Overview.mdx#mode). The valid values are: “immediate” – sets Column Moving to Immediate mode (As the user draggs the header, it is moved and the columns swap animatedly.) “deferred” – sets Column Moving to Deferred mode (Dragging does not move the column header but, instead, arrowheads appear to mark the position where the column would be dropped if the user releases the mouse button.) | “immediate” |
| <ApiLink type="iggridcolumnmoving" member="moveType" section="options" label="moveType" /> | string | Sets the [Column Moving type](./00_igGrid_ColumnMoving_Overview.mdx#type). The valid values are: “dom” – sets the type to DOM Manipulation (The column will be moved by detaching the column DOM and then re-attaching it back in the DOM tree.) “render” – sets the type to Grid Re-rendering (The grid will be re-rendered with the new column order.) | “dom” |
| <ApiLink type="iggridcolumnmoving" member="movingDialogAnimationDuration" section="options" label="movingDialogAnimationDuration" /> | number | Sets the timeDuration in milliseconds to of the animation showing or hiding the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). | 200 |
| <ApiLink type="iggridcolumnmoving" member="movingDialogHeight" section="options" label="movingDialogHeight" /> | number | Sets the height of the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). By default, the height is not set, so that the dialog height stretches to show all the grid columns. | N/A |
| <ApiLink type="iggridcolumnmoving" member="movingDialogWidth" section="options" label="movingDialogWidth" /> | number | Sets the width of the [Move Columns dialog](./00_igGrid_ColumnMoving_Overview.mdx#column-moving-dialog). | 400 |
