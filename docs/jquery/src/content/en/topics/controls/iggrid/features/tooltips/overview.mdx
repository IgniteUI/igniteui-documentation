# Tooltips Overview (igGrid)

## Topic Overview

### Purpose

### In this topic
This topic contains the following sections:

- [Introduction](#introduction)

## <a id="introduction"></a> Introduction 

In the `igGrid`, the main purpose of tooltips is to make the entire cell content visible and also to enable the user to select and copy the text that is inside the tooltip container (very useful when the text is too long to fit in a grid cell, see the picture below). Tooltips are displayed when the user hovers the mouse pointer over `igGrid` cells.

![](images/igGrid_Tooltips_Overview__01.png)

The tooltip functionality of the `igGrid` is provided through the Tooltips widget. Its features are listed in the [Main Features](#features) block below.

## <a id="features"></a> Main Features 

The table below briefly explains the main features of the Tooltips widget and the properties that manage them.

| Feature | Description | jQuery Property | MVC Property |
| --- | --- | --- | --- |
| Visibility management | Operational mode of the Tooltips widget (i.e. whether the tooltips show or not). | [visibility](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:visibility), [style](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:style) | [Visibility](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~Visibility.html), [Style](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~Style.html) |
| Configurable columns | Enables configuring the visibility of the tooltips on per-column basis (i.e. whether the cells of a column should display tooltips or not). | [columnSettings](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:columnSettings) | [ColumnSettings](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~ColumnSettings.html) |
| Animation timing management | Configurable show and delay times (in milliseconds) for the tooltips and duration of the animation that displays and hides them. | [showDelay](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:showDelay), [hideDelay](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:hideDelay), [fadeTimespan](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:fadeTimespan) | [ShowDelay](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~ShowDelay.html), [HideDelay](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~HideDelay.html), [FadeTimespan](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~FadeTimespan.html) |
| Custom positioning | Configurable tooltip position specified as an offset from the mouse pointer. | [cursorLeftOffset](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:cursorLeftOffset), [cursorTopOffset](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:cursorTopOffset) | [CursorLeftOffset](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~CursorLeftOffset.html), [CursorTopOffset](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~CursorTopOffset.html) |
| Client events | Special events that can be handled during the lifecycle of the Tooltips widget. There are events for every state of the tooltip:, showing, shown, hiding, hidden | [tooltipShowing](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#events:tooltipShowing), [tooltipShown](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#events:tooltipShown), [tooltipHiding](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#events:tooltipHiding), [tooltipHidden](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#events:tooltipHidden) |  |
| Appearance | Configures how the tooltips are styled and positioned. There are two modes: "tooltip" and "popover"., The "tooltip" style will position the tooltip according to the mouse cursor (it will display next to the mouse cursor). The tooltip content will be rendered as plain text., The "popover" style initializes an instance of the igPopover control. It will position the tooltip according to the target TD element with an arrow pointing to it. The tooltip content will be rendered as HTML., Appearance can be customized additionally by changing the jQuery UI CSS Framework classes. | [style](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options:style) | [Style](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips~Style.html) |


For the descriptions of the Tooltip widget’s properties, refer to the [jQuery](&#123;environment:jQueryApiUrl&#125;/ui.iggridtooltips#options) and [MVC](Infragistics.Web.Mvc~Infragistics.Web.Mvc.GridTooltips_members.html) API documentations.
