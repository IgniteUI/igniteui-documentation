# Adding igPivotGrid

##In This Group of Topics

### Introduction

This is a group of topics demonstrating how to add the `igPivotGrid`™ control to an HTML page and to an ASP.NET MVC application.

- [Adding igPivotGrid to an HTML Page](/igpivotgrid-adding-to-an-html-page.mdx): This topic explains, in both conceptual and step-by-step form, how to add the `igPivotGrid` control to an HTML page.
