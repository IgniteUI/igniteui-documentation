# Drag-and-Drop Property API Reference (igTree)

## Drag-and-Drop Property Reference
### Introduction

This topic provides reference information about the properties of the `igTree`™ control related to its Drag-and-Drop feature. The properties are listed alphabetically.

### Drag-and-Drop property reference chart

The following table explains the properties of the drag-and-drop feature of `igTree` and lists the default and recommended values.


| Property | Type | Description | Default Value |
| --- | --- | --- | --- |
| [allowDrop](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.allowDrop) | Boolean | Specifies whether the control will accept drops from other controls. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | false |
| [containment](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.containment) | Boolean/Selector/Element/String/Array | When true, the area inside which the helper is contained is scrollable while dragging. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | false |
| []()[copyAfterMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.copyAfterMarkup) | String | Hint markup upon “Copy after”. This hint appears when the copied node is about to be dropped after the currently highlighted node (the node over which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Copy after &#123;0&#125; |
| [copyBeforeMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.copyBeforeMarkup) | String | Hint markup upon “Copy before”. This hint appears when the copied node is about to be dropped before the currently highlighted node (the node over which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Copy before &#123;0&#125; |
| [copyBetweenMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.copyBetweenMarkup) | String | Hint markup upon “Copy between”. This hint appears when the copied node is about to be dropped between the two currently highlighted adjacent nodes (the nodes between which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Copy between &#123;0&#125; and &#123;1&#125; |
| [copyToMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.copyToMarkup) | String | Hint markup upon “Copy to”. This hint appears during dragging when the copied node is about to be dropped over the currently highlighted node (the node over which the mouse pointer is hovering at the moment). In this case, the copied node will become a child node of the target (highlighted) node. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Copy to &#123;0&#125; |
|  |  |  |  |
| [customDropValidation](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.customDropValidation) | Function | Provides an entry point for custom drop validation. The custom validation must have a Boolean return value. It makes the drop point not valid if false is returned. Custom drop validation function is overridden by the internal tree validation, so you can invalidate a drop-and-drop action only at the drop point. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | null |
| [dragAndDrop](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDrop) | Boolean | Enables/disables the Drag-and-Drop feature. | false |
| [dragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) | Object | Complex option for the settings managing the drag-and-drop feature. | none |
| [dragAndDropMode](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragAndDropMode) | String | Specifies the Drag-and-Drop mode. The valid settings are: [default](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragAndDropMode) [move](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragAndDropMode) [copy](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragAndDropMode) Each of them sets the respective Drag-and-Drop mode. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | default |
| [dragOpacity](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragOpacity) | Float | Opacity/transparency level of the dragged node and the hint while dragging. The range of valid values is between 0 and 1 with 1 meaning the hint is fully opaque and 0 meaning the hint is fully transparent. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | 0.75 |
| [dragStartDelay](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.dragStartDelay) | Number | Time to elapse (in milliseconds) after a `mousedown` event after which dragging will be allowed. The purpose of this option is to prevent unwanted drags when clicking on an element. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | 200 |
| [expandDelay](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.expandDelay) | Number | Time to elapse (in milliseconds) after a `mousedown` event after which the node will expand after hovering it during drag. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | 1000 |
| [helper](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.helper) | String | Allows for a helper element to be used for displaying the dragging. The valid settings are: [default](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.helper) `;` means you are providing a function that is returning a DOM element. The default setting means the the helper element uses a function that is defined internally. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | default |
| [invalidMoveToMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.invalidMoveToMarkup) | String | Hint markup for invalid drop location. This hint appears when the moved node is about to be dropped over an invalid drop location. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. |  |
| [moveAfterMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.moveAfterMarkup) | String | Hint markup upon “Move after”. This hint appears when the moved node is about to be dropped after the currently highlighted node (the node over which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Move after &#123;0&#125; |
| [moveBeforeMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.moveBeforeMarkup) | String | Hint markup upon “Move before”. This hint appears when the moved node is about to be dropped before the currently highlighted node (the node over which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings property.](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings property.) | Move before &#123;0&#125; |
| [moveBetweenMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.moveBetweenMarkup) | String | Hint markup upon “Move between”. This hint appears when the moved node is about to be dropped between the two currently highlighted adjacent nodes (the nodes between which the mouse pointer is hovering at the moment). This property is set inside the [DragAndDropSettings property.](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings property.) | Move between &#123;0&#125; and &#123;1&#125; |
| [moveToMarkup](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.moveToMarkup) | String | Hint markup upon “Move to”. This hint appears during dragging when the moved node is about to be dropped over the currently highlighted node (the node over which the mouse pointer is hovering at the moment). In this case, the moved node will become a child node of the target (highlighted) node. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | Move to &#123;0&#125; |
| [revert](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.revert) | Boolean | If set to true, the dragged node will return to its original locations when dragging stops. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | true |
| [revertDuration](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.revertDuration) | Number | Duration of the revert animation (in milliseconds). Setting this property to 0 disables the animation. The [revertDuration](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.revertDuration) setting is ignored if the [revert](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.revert) property is false. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | 500 |
| [zIndex](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings.zIndex) | Number | z-index for the helper while the element being dragged. The z-index specifies the stack order of an element. An element with a greater stack index is always in front of an element with a lower stack order. This property is set inside the [DragAndDropSettings](&#123;environment:jQueryApiUrl&#125;/ui.igtree#options:dragAndDropSettings) property. | 10 |
