# Drag-and-Drop (igTree)

## In This Group of Topics
### Introduction

The topics in this group explain the drag-and-drop functionality of the `igTree`™ control.

### Topics

Detailed information regarding Drag-and-Drop is covered in the following topics:

- [Drag-and-Drop Overview (igTree)](/igtree-drag-and-drop-overview.mdx): This topic provides an overview the Drag-and-Drop feature of the `igTree` control.

- [Enabling Drag-and-Drop (igTree)](/igtree-drag-and-drop-enabling.mdx): This topic explains, with code examples, how to enable the Drag-and-Drop feature in the `igTree` control, in both JavaScript and MVC.

- [Configuring Drag-and-Drop (igTree)](/configuring/igtree-drag-and-drop-configuring.mdx): This group of topics explain, with code examples, how to configure the various aspects of the Drag-and-Drop feature of the `igTree` control.

- [Drag-and-Drop Handling Events (igTree)](./03_ Events/~igTree_Drag-and-Drop_Handling_Events.mdx): This topic explains, with code examples, how to handle the most frequently used events related to the Drag-and-Drop feature of the `igTree` control.

- [Drag-and-Drop API Reference (igTree)](./04_API Reference/~igTree_Drag-and-Drop_API_Reference.mdx): The topics in this group provide reference information about the events, and properties related to the Drag-and-Drop feature of the `igTree` control.
