# Configuring Drag-and-Drop (igTree)

## In This Group of Topics
### Introduction

The topics in this group explain how to configure some aspects of the Drag-and-Drop feature of the `igTree`™ control.

### Topics

Detailed information regarding Drag-and-Drop is covered in the following topics:

- [Configuring the Drag-and-Drop Mode (igTree)](//walkthroughs/igtree-drag-and-drop-configuring-mode.mdx): This topic explains, with code examples, how to configure the Drag-and-Drop mode of the `igTree` control, in both JavaScript and MVC.

- [Configuring Custom Drop Validation (igTree)](//walkthroughs/igtree-drag-and-drop-configuring-custom-drop-validation.mdx): This topic explains, with code examples, how to configure custom drop validation of the Drag-and-Drop feature of the `igTree`™ control, in both JavaScript and MVC.

- [Configuring the Look of the Drag Visual Tokens (igTree)](/igtree-drag-and-drop-configuring-tokens.mdx): This topic explains, with code examples, how to configure the look of the drag visual tokens of the `igTree` control, in both JavaScript and MVC.
