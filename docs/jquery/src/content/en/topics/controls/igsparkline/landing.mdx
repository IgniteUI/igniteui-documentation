# igSparkline

## In This Group of Topics
### Introduction

The [`igSparkline`](&#123;environment:jQueryApiUrl&#125;/ui.igSparkline.html)™ is a lightweight charting control that can render the following chart types:

-   Line
-   Area
-   Column
-   Win/Loss

It is intended for rendering in a small scale layout such as a grid Cell. It can also be rendered stand alone. The following previews illustrate the four Sparkline types.
