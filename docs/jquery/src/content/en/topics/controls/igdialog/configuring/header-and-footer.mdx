## <a id="header"></a> Enable and Configure igDialog Header

The `igDialog` API proposes several API properties for modifying its header.

### <a id="header-property-settings"></a> Property Settings

The following table maps the desired header features to property settings:

In order to: | Use this property: | And set it to:
--- | --- | ---
Show the igDialog header | <ApiLink type="igDialog" member="showHeader" section="options" label="showHeader" /> | true
Set the igDialog header title | <ApiLink type="igDialog" member="headerText" section="options" label="headerText" /> | “HEADER”
Set the igDialog header image | <ApiLink type="igDialog" member="imageClass" section="options" label="imageClass" /> | “clsImage”
Set the igDialog close button title | <ApiLink type="igDialog" member="locale.closeButtonTitle" section="options" label="closeButtonTitle" /> | “X”
Set the igDialog minimize button title | <ApiLink type="igDialog" member="locale.minimizeButtonTitle" section="options" label="minimizeButtonTitle" /> | “MIN”
Set the igDialog maximize button title | <ApiLink type="igDialog" member="locale.maximizeButtonTitle" section="options" label="maximizeButtonTitle" /> | “MAX”
Set the igDialog pin button title | <ApiLink type="igDialog" member="locale.pinButtonTitle" section="options" label="pinButtonTitle" /> | “PIN”
Set the igDialog unpin button title | <ApiLink type="igDialog" member="locale.unpinButtonTitle" section="options" label="unpinButtonTitle" /> | “UNPIN”
Set the igDialog restore button title | <ApiLink type="igDialog" member="locale.restoreButtonTitle" section="options" label="restoreButtonTitle" /> | “RESTORE”

## <a id="footer"></a> Enable and Configure igDialog Footer

The `igDialog` API proposes two API properties for modifying its footer.

### <a id="footer-property-settings"></a> Property Settings

The following table maps the desired header features to property
settings:

In order to: | Use this property: | And set it to:
--- | --- | ---
Show the `igDialog` footer | <ApiLink type="igDialog" member="showFooter" section="options" label="showFooter" /> | true
Set the `igDialog` footer title | <ApiLink type="igDialog" member="footerText" section="options" label="footerText" /> | “FOOTER”
