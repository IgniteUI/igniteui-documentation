## <a id="header"></a> Enable and Configure igDialog Header

The `igDialog` API proposes several API properties for modifying its header.

### <a id="header-property-settings"></a> Property Settings

The following table maps the desired header features to property settings:

In order to: | Use this property: | And set it to:
--- | --- | ---
Show the igDialog header | [showHeader](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:showHeader) | true
Set the igDialog header title | [headerText](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:headerText) | “HEADER”
Set the igDialog header image | [imageClass](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:imageClass) | “clsImage”
Set the igDialog close button title | [closeButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.closeButtonTitle) | “X”
Set the igDialog minimize button title | [minimizeButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.minimizeButtonTitle) | “MIN”
Set the igDialog maximize button title | [maximizeButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.maximizeButtonTitle) | “MAX”
Set the igDialog pin button title | [pinButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.pinButtonTitle) | “PIN”
Set the igDialog unpin button title | [unpinButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.unpinButtonTitle) | “UNPIN”
Set the igDialog restore button title | [restoreButtonTitle](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:locale.restoreButtonTitle) | “RESTORE”

## <a id="footer"></a> Enable and Configure igDialog Footer

The `igDialog` API proposes two API properties for modifying its footer.

### <a id="footer-property-settings"></a> Property Settings

The following table maps the desired header features to property
settings:

In order to: | Use this property: | And set it to:
--- | --- | ---
Show the `igDialog` footer | [showFooter](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:showFooter) | true
Set the `igDialog` footer title | [footerText](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#options:footerText) | “FOOTER”
