## Method Reference

The following table summarizes the purpose and functionality of the igDialog control’s featured methods:

Method | Description
--- | ---
<ApiLink type="igDialog" member="state" section="methods" label="state" /> | Gets the state of the editor.
<ApiLink type="igDialog" member="mainElement" section="methods" label="mainElement" /> | Gets the reference to a dynamically created DIV element, which represents dialog.
<ApiLink type="igDialog" member="content" section="methods" label="content" /> | Retrieves the `igDialog` content container or sets its content to be the new content provided. This is the inner container of the dialog window excluding headers, resizing handlers, etc.
<ApiLink type="igDialog" member="close" section="methods" label="close" /> | Closes the `igDialog`, if it is opened.
<ApiLink type="igDialog" member="open" section="methods" label="open" /> | Opens the `igDialog`, if it is closed.
<ApiLink type="igDialog" member="minimize" section="methods" label="minimize" /> | Minimizes the `igDialog`, if it is opened.
<ApiLink type="igDialog" member="maximize" section="methods" label="maximize" /> | Maximizes the `igDialog`, if it is opened.
<ApiLink type="igDialog" member="pin" section="methods" label="pin" /> | Pins the `igDialog`, if it is not pinned.
<ApiLink type="igDialog" member="unpin" section="methods" label="unpin" /> | Unpins the `igDialog`, if it is pinned.
<ApiLink type="igDialog" member="getTopModal" section="methods" label="getTopModal" /> | Gets reference to the top modal dialog.
<ApiLink type="igDialog" member="isTopModal" section="methods" label="isTopModal" /> |Returns if dialog is modal and is currently active.
<ApiLink type="igDialog" member="moveToTop" section="methods" label="moveToTop" /> | Moves dialog to the top.
<ApiLink type="igDialog" member="destroy" section="methods" label="destroy" /> | Destroys the `igDialog` control.
