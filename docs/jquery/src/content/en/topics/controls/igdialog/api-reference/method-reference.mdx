## <a id="methods"></a> Method Reference

The following table summarizes the purpose and functionality of the igDialog control’s featured methods:

Method | Description
--- | ---
[state](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:state) | Gets the state of the editor.
[mainElement](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:mainElement) | Gets the reference to a dynamically created DIV element, which represents dialog.
[content](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:content) | Retrieves the `igDialog` content container or sets its content to be the new content provided. This is the inner container of the dialog window excluding headers, resizing handlers, etc.
[close](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:close) | Closes the `igDialog`, if it is opened.
[open](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:open) | Opens the `igDialog`, if it is closed.
[minimize](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:minimize) | Minimizes the `igDialog`, if it is opened.
[maximize](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:maximize) | Maximizes the `igDialog`, if it is opened.
[pin](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:pin) | Pins the `igDialog`, if it is not pinned.
[unpin](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:unpin) | Unpins the `igDialog`, if it is pinned.
[getTopModal](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:getTopModal) | Gets reference to the top modal dialog.
[isTopModal](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:isTopModal) |Returns if dialog is modal and is currently active.
[moveToTop](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:moveToTop) | Moves dialog to the top.
[destroy](&#123;environment:jQueryApiUrl&#125;/ui.igDialog#methods:destroy) | Destroys the `igDialog` control.
