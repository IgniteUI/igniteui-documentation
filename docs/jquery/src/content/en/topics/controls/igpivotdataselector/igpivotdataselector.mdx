# igPivotDataSelector


##In This Group of Topics

### Introduction

The topics in this group explain the `igPivotDataSelector`™ control and its use.

`igPivotDataSelector` is an interactive UI control (a jQuery UI widget) that enables users to select data slices, typically, when data is being visualized in an [igPivotGrid](/igpivotgrid/igpivotgrid.mdx)™.
