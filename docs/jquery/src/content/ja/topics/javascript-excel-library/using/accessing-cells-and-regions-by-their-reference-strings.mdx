# 参照文字列による Cells および Regions のアクセス

このトピックでは、Excel® ワークブックのワークシート内でセルと範囲にアクセスする方法をわかりやすく紹介します。Worksheet クラスの <ApiLink pkg="ig" type="excel.Worksheet" member="getCell" section="methods" label="getCell" /> メソッドと <ApiLink pkg="ig" type="excel.Worksheet" member="getRegion" section="methods" label="getRegion" /> メソッドは、パラメーターとしてセル参照文字列を受け取ります。

以下のコード例では、Worksheet セルに getCell メソッドでアクセスして、Excel 数式を適用します。同様に Excel 数式がワークシート内のセルの範囲に適用されます。

**JavaScript の場合:**


```js
'Accessing a single cell
worksheet.getCell("C5").applyFormula("=A5*B5");
'Accessing a range of cells
worksheet.getRegion("G1:G10").applyFormula("=E1*F1");
```
