# Excel ファイルから数式の値を取得

Formula で表現された Excel の Cell の評価した値を取得するには、単純に <ApiLink pkg="ig" type="excel.WorksheetCell" label="WorksheetCell" /> の <ApiLink pkg="ig" type="excel.WorksheetCell" member="value" section="methods" label="value" /> プロパティにアクセスします。

以下のコード例は、Excel 数式によって移植される特定の Excel ワークシート セルのアクセス方法を示します。

**JavaScript の場合:**
```js
var calculatedValue = theWorksheet.rows(1).cells(4).value();
```
