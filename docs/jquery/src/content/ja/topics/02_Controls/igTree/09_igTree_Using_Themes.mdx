# igTree でのテーマの使用

## テーマの使用
### 概要
`igTree`™ コントロールはスタイル設定やテーマの設定に jQuery UI CSS Framework を利用します。`igTree` コントロールはデフォルトで、アプリケーションが使用するようインフラジスティックスが提供している jQuery UI テーマとして IG Theme を使用します。

テーマのカスタマイズには、jQuery UI ThemeRoller を使用できます。ThemeRoller は jQuery UI が提供するツールで、これを使用すると、jQuery UI ウィジェットと互換性のあるカスタム テーマを作成できるようになります。数々のビルド済みテーマをご自分の Web サイトにダウンロードして使用できます。`igTree` コントロールは ThemeRoller テーマの使用をサポートしています。
