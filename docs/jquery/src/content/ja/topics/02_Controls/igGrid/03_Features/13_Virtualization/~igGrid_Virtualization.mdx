# 仮想化 (igGrid)

## 概要

仮想化は、アクティブなメモリ内 DOM オブジェクトの数を減らす `igGrid`™ コントロールの機能です。仮想化は、仮想化されていないグリッドと比較してユーザーからは何も変わりません。

## トピック

仮想化に関する詳細情報は、以下のトピックで解説します。

- [仮想化の概要](igGrid-Virtualization-Overview.html): このトピックは、`igGrid` コントロールの仮想化機能を紹介します。

- [仮想化を有効にし、構成する](igGrid-Enabling-and-Configuring-Virtualization.html): このトピックでは、コード例と共に、`igGrid` 内の仮想化機能を有効化し構成する方法について説明します。
