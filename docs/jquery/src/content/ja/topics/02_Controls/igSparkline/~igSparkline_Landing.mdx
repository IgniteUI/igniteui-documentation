# igSparkline

## このグループのトピックについて
### 概要

[`igSparkline`](%%jQueryApiUrl%%/ui.igSparkline.html)™ は、以下のチャート タイプを描画できる軽量チャート作成コントロールです。

-   折れ線
-   エリア
-   柱状
-   Win/Loss

グリッド セルなど小さなレイアウトで描画する目的で作成されています。スタンドアロンでも描画できます。以下のプレビューは、4 種類のスパークラインを示しています。
