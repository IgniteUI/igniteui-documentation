## <a id="introduction"></a>概要
### 中央ボタンの概要

中央ボタンは `igRadialMenu` コントロールが閉自邸ル場合にも唯一表示され、クリックでメニューを開くことができます。ルート レベルではメニューを閉じるために中央ボタンを使用し、サブ項目の他のすべてのレベルでは前の項目レベルに戻るために中央ボタンを使用します。

`igRadialMenu` の複数のプロパティでは、中央ボタンや「`CenterButton`」を使用したすべての起動を構成し、カスタマイズできます。



## <a id="configuration"></a>中央ボタンの構成の概要
### 中央ボタンの構成の概要表

以下の表は、`igRadialMenu` 中央ボタンの構成可能な主要な要素を簡単に説明し、それらを構成するプロパティにマップします。

構成可能な項目|詳細|プロパティ
---|---|---
コンテンツ幅|中央ボタンのコンテンツ幅の設定または取得|`centerButtonContentWidth`
コンテンツ高|中央ボタンのコンテンツ高さの設定または取得|`centerButtonContentHeight`
オープン/クローズ|`igRadialMenu` のオープン/クローズ|`isOpen`
