#キーボード ナビゲーション (igMaskEditor)

##トピックの概要
このトピックでは、エンドユーザーが入力操作を素早く簡単にナビゲートできるように、キーボード ナビゲーションを使用する方法を説明します。

### このトピックの内容

このトピックは、以下のセクションで構成されます。

-   [キーボード操作](#skbInteraction)
-   [エディターの選択](#skbInteraction)


## <a id="skbInteraction"></a>標準のキーボード操作

コピー、切り取り、貼り付け、元に戻す、やり直しなどの機能には、一般的なキーの組み合わせを使用できます。  

押す|動作
---|---
<kbd>Right Arrow</kbd>|マウス インジケータを 1 文字分右に移動します
<kbd>Left Arrow</kbd>|マウス インジケータを 1 文字分左に移動します
<kbd>Ctrl</kbd> + <kbd>Right Arrow</kbd>|マウス インジケータをチャンク 1 つ分右に移動します
<kbd>Ctrl</kbd> + <kbd>Left Arrow</kbd>|マウス インジケータをチャンク 1 つ分左に移動します
<kbd>Ctrl</kbd> + <kbd>Home</kbd> または <kbd>Home</kbd>|マウス インジケータを入力の先頭に移動します
<kbd>Ctrl</kbd> + <kbd>End</kbd> または <kbd>End</kbd>|マウス インジケータを入力の末尾に移動します
<kbd>Backspace</kbd>|前の文字を削除します
<kbd>Delete</kbd>|次の文字を削除します
<kbd>Ctrl</kbd> + <kbd>Backspace</kbd>|前のチャンクを削除します
<kbd>Ctrl</kbd> + <kbd>Delete</kbd>|次のチャンクを削除します
<kbd>Delete</kbd> または <kbd>Backspace</kbd>|選択した内容を削除します

## <a id="kbInteraction"></a>エディターの選択

このセクションでは、エディターのフォーカス、エディターの値の一部選択またはすべて選択に使用できるキーの組み合わせを示します。

押す|動作
---|---
<kbd>Tab</kbd> または <kbd>Shift</kbd> + <kbd>Tab</kbd>|エディターをフォーカスします
<kbd>Shift</kbd> + <kbd>Tab</kbd>|前のコンポーネントをフォーカスします
<kbd>Tab</kbd>|次のコンポーネントをフォーカスします
<kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|右方向に文字を選択します
<kbd>Shift</kbd> + <kbd>Left Arrow</kbd>|左方向に文字を選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|現在のチャンクの末尾を選択します、またはチャンク 1 つ分右に移動します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|現在のチャンクの先頭を選択します、またはチャンク 1 つ分左に移動します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Home</kbd> または <kbd>Shift</kbd> + <kbd>Home</kbd>|入力内の値を先頭まで選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>End</kbd> または <kbd>Shift</kbd> + <kbd>End</kbd>|入力内の値を末尾まで選択します
<kbd>Ctrl</kbd> + <kbd>A</kbd>|入力内のすべての値を選択します
