# igPercentEditor 既知の問題

##既知の問題

現在、`igPercentEditor` に関する既知の問題や制限はありません。
このコントロールは `igNumericEditor` を継承するため、記述がない限り「[igNumericEditor の既知の問題](igNumericEditor-Known-Issues.html)」の一般的な問題と制限がパーセンテージ エディターにも影響します。
