# igTimePicker 既知の問題

**既知の問題**

現時点で `igTimePicker` に関する既知の問題や制限はありません。
このコントロールは `igDateEditor` を継承するため、記述がない限り[「igDateEditor の既知の問題」](igdateeditor-known-issues.html)の一般的な問題と制限がタイムピッカーにも影響します。
