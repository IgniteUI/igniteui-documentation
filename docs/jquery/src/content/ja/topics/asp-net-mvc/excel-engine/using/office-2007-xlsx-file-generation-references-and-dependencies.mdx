# Office 2007 XLSX ファイル生成の参照と依存

**.NET CLR 4.0 以上のバージョンの使用**

Microsoft® Office® 2007 準拠の `xlsx` Excel ファイルを生成するために Infragistics® Excel® オブジェクト ライブラリおよびエクスポーターを使用する場合、.NET Framework CLR バージョン 4.0 以上が必要です。このシナリオでは、`Infragistics.Web.Mvc.Excel` アセンブリを参照します。xlsx ファイル作成ロジックに組み込まれる圧縮およびパッケージ機能は、.NET Framework CLR4.0 に依存します。
