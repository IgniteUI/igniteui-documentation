# レポート グラフィックス

レポート グラフィックスは、Brushes、Colors、Fonts、および Images などのいくつかのグラフィックス要素によってレポートを魅力的なものにします。レポート グラフィックスを使用して、レイアウト 要素を強化します。たとえば、ブラシとペンを使用して、要素の背景と境界線にそれぞれ色を付けます。Tables、Lists、Text などの[パターン コンテンツ](/documentengine-pattern-content)にカラーを追加してフォントを変更することも可能です。任意のレイアウトまたはコンテンツ 要素はレポート グラフィックスによって何らかの影響を受ける場合があります。
