# セクション
Section 要素は、Report 要素に追加できる唯一の要素です。Section 要素によって、ページ番号ならびに特にステーショナリやデコレーションを使用してドキュメントのルック アンド フィールを操作できます。

Section 要素についての詳細は、以下のリンクを参照してください。

-   [ページ番号の追加](/asp-net-mvc/document-engine/writing/elements/layout/section/add-page-numbering)
-   [Section 要素](/asp-net-mvc/document-engine/writing/elements/layout/section/section-element)
