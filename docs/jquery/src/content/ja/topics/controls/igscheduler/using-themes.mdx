# igScheduler のスタイル設定

## テーマの使用

### 概要

`igScheduler`™ はスタイルやテーマの設定に jQuery UI CSS Framework を使用します。特に指定しない限り、`igScheduler` は、デフォルトの jQuery UI テーマとして標準装備されている IG Theme を使用します。

テーマのカスタマイズには、ThemeRoller を使用することができます。ThemeRoller は jQuery UI が提供するツールで jQuery UI ウィジェットと互換性のあるカスタム テーマを簡単に作成できます。さまざまなビルド済みテーマを Web サイト用にダウンロードして使用できます。`igScheduler` コントロールは ThemeRoller テーマの使用をサポートしています。
