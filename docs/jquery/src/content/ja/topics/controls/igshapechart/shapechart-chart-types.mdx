# チャート タイプ 

## 概要

データの表示方法は、チャートの chartType プロパティで設定します。
以下は、シェープ チャートでサポートされるすべてのタイプです。

特別なケースにプロパティの `Auto` 設定があります。`Auto` を使用した場合、チャートがデータを分析し、最適なチャート タイプを割り当てます。

このプロパティのデフォルト値は、カテゴリ チャートにバインドされる基本の ItemsSource のサイズに基づいて決定されます。   

## サポートされるチャート タイプ

<div class="document-table-container">
    
|  |  |  |
| --- | --- | --- |
| プロパティ値 | 説明 | 例 |
| `Area` | 各ポイントに割り当てられた数値の X/Y データによる三角測量に基づいて色付きのサーフェスを含むエリア チャートを指定します。 | ![](../../images/images/shapechart-area-01.PNG) |
| `Bubble` | X/Y データに相対マーカーを使用するバブル チャートを指定します。 | ![](../../images/images/shapechart-bubble-01.PNG) |
| `Contour` | 各ポイントに割り当てられた数値の X/Y データによる三角測量に基づいて色付きの線を含むエリア チャートを指定します。 | ![](../../images/images/shapechart-contour-01.PNG) |
| `HighDensity` | 隣接するポイントの密度に基づいて、X/Y データに色付きのビットマップ ピクセルの高密度チャートを指定します。 | ![](../../images/images/shapechart-highdensity-01.PNG) |
| `Line` | X/Y データに線で接続される小さいマーカーを使用した折れ線チャートを指定します。 | ![](../../images/images/shapechart-line-01.PNG) |
| `Point` | X/Y データに小さいマーカーを使用するポイント チャートを指定します。 | ![](../../images/images/shapechart-point-01.PNG) |
| `Polygon` | X/Y データにより定義される多角形を持つ多角形チャートを指定します。 | ![](../../images/images/shapechart-polygon-01.PNG) |
| `Polyline` | X/Y データにより定義されるポリラインを持つポリライン チャートを指定します。 | ![](../../images/images/shapechart-polyline-01.PNG) |
| `Spline` | X/Y データにスプラインで接続される小さいマーカーを使用したスプライン チャートを指定します。 | ![](../../images/images/shapechart-spline-01.PNG) |
| `Auto` | 内部データ アダプターの候補に基づいてチャート タイプの自動選択を指定します。 |  |

</div>
