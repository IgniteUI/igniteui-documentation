# igSplitter



## このグループのトピックについて
### 概要

このグループのトピックでは、`igSplitter` コントロールとその使用方法について説明します。

`igSplitter` は、レイアウトを 2 つの異なるパネルに分けることにより HTML5 Web アプリケーションおよびサイトでレイアウトを管理するためのコンテナー コントロールです。
