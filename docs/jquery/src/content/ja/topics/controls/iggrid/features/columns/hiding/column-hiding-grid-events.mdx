## <a id="events-overview"></a> イベントの概要

列管理に関するイベントがいくつかあります。(以下の「イベント参照チャート」を参照してください)。

また、**-ing** サフィックスが付いたすべてのイベントはキャンセルでき、ハンドラーで false を返すよう設定されている場合にアクションを終了できます。

**JavaScript の場合:**

```js
$("#grid1").igGrid({
   features: [
	{
		name: 'Hiding',
		columnHiding: function (ui, args) {
			return  false;
		}
    }
  ]
});
```

引数のメソッドおよびプロパティのほとんどに、行固有のデータとそれらが属するグリッドの ID が入っています。

## <a id="attaching-events"></a> jQuery および MVC でイベントを添付する

イベントを添付するには、プロパティの定義と同様に、イベントのハンドラーを定義するだけです。イベントをトリガーすると、ハンドラーが呼び出されます。

**JavaScript の場合:**

```js
$("#grid1").igGrid({
    features: [{
        name: 'Hiding',
        columnHiding: function (e, args) {
            // Handle event  
        }
    }]
});
```

MVC でハンドラーを添付する場合、jQuery User Interface (UI) パターンを使用する必要があります。このパターンは、コントロールとイベントの名前を小文字で連結するよう指示します。

**JavaScript の場合:**

```js
$("#grid1").on("iggridhidingcolumnhiding ", function (e, args) {
     // Handle event  
});
```

> **注**: jQuery の `on` 関数は、ハンドラーに添付した後に一部の DOM 要素が追加される場合に使用されます。

[columnHidden](&#123;environment:jQueryApiUrl&#125;/ui.iggridhiding#events:columnHidden) |非表示が実行され、結果が描画された後に発生するイベント|False
[columnShowing](&#123;environment:jQueryApiUrl&#125;/ui.iggridhiding#events:columnShowing) |表示操作が実行される前に発生するイベント|True

[columnChooserOpening](&#123;environment:jQueryApiUrl&#125;/ui.iggridhiding#events:columnChooserOpening) |列チューザーが開く前に発生するイベント|True

[columnChooserContentsRendered](&#123;environment:jQueryApiUrl&#125;/ui.iggridhiding#events:columnChooserContentsRendered) |列チューザーのコンテンツが描画された後に発生するイベント|False
