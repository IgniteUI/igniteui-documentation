# プロパティ参照 (igGrid)



## igGrid 列移動のプロパティ

以下の表では、`igGrid` の列移動機能のプロパティを説明し、またデフォルト値と推奨値を紹介します。プロパティはアルファベット順に並び替えられます。


| プロパティ | タイプ | 説明 | デフォルト値 |
| --- | --- | --- | --- |
| [addMovingDropdown](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:addMovingDropdown) | bool | false の場合、すべての列に対して列移動メニューを非表示にします。 | true |
| [columnSettings](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings) | array | 列移動が有効になっていると、 移動が無効になる各列を構成します。列は、[`columnSettings.allowMoving`](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.allowMoving) および [`columnSettings.columnIndex`](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnIndex) /[`columnSettings.columnKey`](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnKey) プロパティを介して指定されます。 | null |
| [columnSettings.allowMoving](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.allowMoving) | bool | false である場合、グリッド内で特定の列に対して列移動を無効にします。構成済みの列は、 [`columnSettings.columnIndex`](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnIndex) または [`columnSettings.columnKey`](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnKey) により指定されます。 . | true |
| [columnSettings.columnIndex](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnIndex) | number | 構成されている列のインデックスグリッド列に列キーが定義されていない場合に列インデックスを使用して列を指定します (例: [`autoGenerateColumns`](environment:jQueryApiUrl/ui.iggrid#options:autoGenerateColumns) を true に設定する場合)。 | null |
| [columnSettings.columnKey](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:columnSettings.columnKey) | string | 構成されている列の列キーグリッド列に列キーが定義されている場合に列インデックスを使用して列を指定します (例: グリッド列が手動で定義される場合)。 | null |
| [mode](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:mode) | string | [列移動モード](/controls/iggrid/features/columns/moving/columnmoving-overview#mode)を設定します。有効な値： “immediate” - 列移動 を即時モードに設定します。 (ユーザーがヘッダーをドラッグすると、ヘッダーが移動されて列がアニメーションで置き換えされます) “deferred” - 列移動 を遅延モードに設定します。 (ドラッグでは列ヘッダーは移動しませんが、代わりにユーザーがマウス ボタンをリリースする場合に列がドロップされる位置をマーク付けします) | “immediate” |
| [moveType](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:moveType) | string | [列移動タイプ](/controls/iggrid/features/columns/moving/columnmoving-overview#type)を設定します。有効な値： “dom” - タイプを DOM 操作に設定します。 (列をデタッチし、DOM ツリーに再度アタッチすることで列を移動します) “render” - タイプをグリッドの再レンダリングに設定します。 (グリッドは、新しい列の順序で再描画されます) | “dom” |
| [movingDialogAnimationDuration](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:movingDialogAnimationDuration) | number | アニメーションで[列の移動ダイアログ](/controls/iggrid/features/columns/moving/columnmoving-overview#column-moving-dialog)を表示または非表示にする timeDuration をミリ秒で設定します。 | 200 |
| [movingDialogHeight](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:movingDialogHeight) | number | [列の移動ダイアログ](/controls/iggrid/features/columns/moving/columnmoving-overview#column-moving-dialog)の高さを設定します。デフォルトでは高さは設定されません。そのため、データの高さはすべてのグリッド列を表示するまで引き伸ばされます。 | なし |
| [movingDialogWidth](environment:jQueryApiUrl/ui.iggridcolumnmoving#options:movingDialogWidth) | number | [列の移動ダイアログ](/controls/iggrid/features/columns/moving/columnmoving-overview#column-moving-dialog)の幅を設定します。 | 400 |
