# igMap

##このグループのトピックについて


### 概要

このグループのトピックでは、\{environment:ProductName\}™ の `igMap`™ コントロールについて説明します。

`igMap` コントロールは、HTML5 キャンバス要素に基づいて多様なマップを視覚化し、すべてのレンダリングをクライアント側で実行するための機能を提供します。
