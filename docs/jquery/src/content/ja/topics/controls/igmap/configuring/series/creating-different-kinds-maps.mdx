# マップ シリーズの構成 (igMap)

##本トピック グループの内容

### 概要

このトピック グループでは、多種多様なマップを生成できるように `igMap`™ コントロールでサポートされる全マップ タイプ (マップ シリーズ) の構成方法を説明します。

#### トピック

このセクションには、`igMap` コントロールの機能に関するトピックが含まれています。

-	[地理シンボル シリーズの構成 (igMap)](/igmap-configuring-geographic-symbol-series): このトピックでは、`igMap` コントロールを使用して地理シンボル シリーズを構成する方法について説明します。

-	[地理図形シリーズの構成 (igMap)](/igmap-configuring-geographic-shapes): このトピックでは、`igMap`™ コントロールで地理図形シリーズを構成する方法について説明します。

-	[地理ポリライン シリーズの構成 (igMap)](/igmap-configuring-geographic-polyline-series): このトピックでは、`igMap`™ コントロールを使用して地理ポリライン シリーズを構成する方法について説明します。

-	[地理散布エリア シリーズの構成 (igMap)](/igmap-configuring-geographic-scatter-area-series): このトピックでは、`igMap`™ コントロールで地理散布エリア シリーズを構成する方法について説明します。

-	[地理等高線シリーズの構成 (igMap)](/igmap-configuring-geographic-contour-line-series): このトピックでは、`igMap`™ コントロールを使用して地理等高線シリーズを構成する方法について説明します。
