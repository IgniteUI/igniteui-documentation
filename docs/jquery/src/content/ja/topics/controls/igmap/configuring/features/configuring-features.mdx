# 機能の構成 (igMap)


##このグループのトピックについて

### 概要

このグループのトピックでは、`igMap`™ コントロールのさまざまな機能の構成方法を説明します。対応する機能としては、特定の地理領域へのナビゲーション、Overview Plus Detail パネルの有効または無効化、表示中の領域をマップ上で取得、パンとズームに関するユーザー インタラクションの構成、ツールチップ テンプレートの構成、カスタム マーカーの設定などがあります。

### トピック

このセクションには、`igMap` コントロールの機能に関するトピックが含まれています。

-	[ナビゲーション機能の構成 (igMap)](/igmap-configuring-navigation-features): このトピックでは、`igMap` コントロールのナビゲーション機能の構成方法と、その API を使用してマップ内の表示部分の位置とサイズを定義する方法を示します。

-	[ビジュアル機能の構成 (igMap)](/igmap-configuring-visual-features): このトピックでは、十字形、ツールチップ テンプレート、マーカー テンプレートなど、`igMap` コントロールのビジュアル機能の構成方法を示します。
