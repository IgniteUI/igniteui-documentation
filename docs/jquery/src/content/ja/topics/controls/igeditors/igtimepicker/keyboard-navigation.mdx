# キーボード ナビゲーション (igTimePicker)

#キーボード ナビゲーション (igTimePicker)

##トピックの概要
このトピックでは、エンドユーザーが入力操作を素早く簡単にナビゲートできるように、キーボード ナビゲーションを使用する方法を説明します。

### このトピックの内容

このトピックは、以下のセクションで構成されます。

-   [標準のキーボード操作](#standard_interactions)
-   [エディターの選択](#selection)
-   [igTimePicker 固有のナビゲーション](#specific_interactions)


## <a id="standard_interactions"></a>標準のキーボード操作

コピー、切り取り、貼り付け、元に戻す、やり直しの操作には、一般的なキーの組み合わせを使用できます。  

キー|動作
---|---
<kbd>右矢印</kbd>|カレットを 1 文字分右に移動します
<kbd>左矢印</kbd>|カレットを 1 文字分左に移動します
<kbd>Ctrl</kbd> + <kbd>右矢印</kbd>|カレットを時間入力の 1 区切り分右に移動します
<kbd>Ctrl</kbd> + <kbd>Left Arrow</kbd>|カレットを時間入力の 1 区切り分左に移動します
<kbd>Ctrl</kbd> + <kbd>Home</kbd> または <kbd>Home</kbd>|カレットを入力の先頭に移動します
<kbd>Ctrl</kbd> + <kbd>End</kbd> または <kbd>End</kbd>|カレットを入力の末尾に移動します
<kbd>Backspace</kbd>|前の文字を削除します
<kbd>Delete</kbd>|次の文字を削除します
<kbd>Ctrl</kbd> + <kbd>Backspace</kbd>|時間入力の前の部分を削除します
<kbd>Ctrl</kbd> + <kbd>Delete</kbd>|次のチャンクを削除します
<kbd>上矢印</kbd>|入力フィールドで現在の時間部分を増加します。
<kbd>下矢印</kbd>|入力フィールドで現在の時間部分を減らします。

## <a id="selection"></a>エディターの選択

このセクションでは、エディターのフォーカス、エディターの値の一部選択またはすべて選択に使用できるキーの組み合わせを示します。

キー|動作
---|---
<kbd>Tab</kbd> または <kbd>Shift</kbd> + <kbd>Tab</kbd>|エディターをフォーカスします
<kbd>Shift</kbd> + <kbd>Tab</kbd>|前のコンポーネントをフォーカスします
<kbd>Tab</kbd>|次のコンポーネントをフォーカスします
<kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|時間入力で右の記号を選択します
<kbd>Shift</kbd> + <kbd>Left Arrow</kbd>|時間入力で記号を左選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|時間の右部分を選択します。
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|時間の左部分を選択します。
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Home</kbd> または <kbd>Shift</kbd> + <kbd>Home</kbd>|入力値の先頭まで選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>End</kbd> または <kbd>Shift</kbd> + <kbd>End</kbd>|入力値の末尾まで選択します
<kbd>Ctrl</kbd> + <kbd>A</kbd>|入力したすべての値を選択します

## <a id="specific_interactions"></a> igTimePicker 固有のナビゲーション


キー|動作
---|---
<kbd>Alt</kbd> + <kbd>Down Arrow</kbd>|入力がフォーカスされている場合に、ドロップダウンを開きます
<kbd>Alt</kbd> + <kbd>Up Arrow</kbd>|入力がフォーカスされている場合に、ドロップダウンを閉じます
<kbd>下矢印</kbd>|マウス インジケーターの位置に従って、現在の時間部分をスピンダウンします
<kbd>上矢印</kbd>|マウス インジケーターの位置に従って、現在の時間部分をスピンアップします
<kbd>Ctrl</kbd> + <kbd>右矢印</kbd>|次の時間部分に移動します
<kbd>Ctrl</kbd> + <kbd>左矢印</kbd>|前の時間部分に移動します
<kbd>Enter</kbd>|フォーカスされた項目を選択済みとして確定します
<kbd>Enter</kbd> または <kbd>Alt</kbd> + <kbd>上矢印</kbd>|フォーカスされた項目を破棄し、以前の選択を保持します
