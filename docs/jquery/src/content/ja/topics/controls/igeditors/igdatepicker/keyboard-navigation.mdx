#キーボード ナビゲーション (igDatePicker)

##トピックの概要
このトピックでは、エンドユーザーが入力操作を素早く簡単にナビゲートできるように、キーボード ナビゲーションを使用する方法を説明します。

### このトピックの内容

このトピックは、以下のセクションで構成されます。

-   [キーボード操作](#skbInteraction)
-   [エディターの選択](#skbInteraction)
-   [igDatePicker 固有のナビゲーション](#ddInteraction)


## <a id="skbInteraction"></a>標準のキーボード操作

コピー、切り取り、貼り付け、元に戻す、やり直しなどの機能には、一般的なキーの組み合わせを使用できます。  

押す|動作
---|---
<kbd>Right Arrow</kbd>|カレットを 1 文字分右に移動します
<kbd>Left Arrow</kbd>|カレットを 1 文字分左に移動します
<kbd>Ctrl</kbd> + <kbd>Right Arrow</kbd>|カレットを日付の 1 区切り分右に移動します
<kbd>Ctrl</kbd> + <kbd>Left Arrow</kbd>|カレットを日付の 1 区切り分左に移動します
<kbd>Ctrl</kbd> + <kbd>Home</kbd> または <kbd>Home</kbd>|カレットを入力の先頭に移動します
<kbd>Ctrl</kbd> + <kbd>End</kbd> または <kbd>End</kbd>|カレットを入力の末尾に移動します
<kbd>Backspace</kbd>|前の文字を削除します
<kbd>Delete</kbd>|次の文字を削除します
<kbd>Ctrl</kbd> + <kbd>Backspace</kbd>|日付の前の部分を削除します
<kbd>Ctrl</kbd> + <kbd>Delete</kbd>|次のチャンクを削除します
<kbd>Delete</kbd> または <kbd>Backspace</kbd>|日付の後の部分を削除します
<kbd>Up Arrow</kbd>|入力内の日付の現在の部分を増分します
<kbd>Down Arrow</kbd>|入力内の日付の現在の部分を減分します

## <a id="kbInteraction"></a>エディターの選択

このセクションでは、エディターのフォーカス、エディターの値の一部選択またはすべて選択に使用できるキーの組み合わせを示します。

押す|動作
---|---
<kbd>Tab</kbd> または <kbd>Shift</kbd> + <kbd>Tab</kbd>|エディターをフォーカスします
<kbd>Shift</kbd> + <kbd>Tab</kbd>|前のコンポーネントをフォーカスします
<kbd>Tab</kbd>|次のコンポーネントをフォーカスします
<kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|終了日の現在の部分を選択します、または次の部分を選択します
<kbd>Shift</kbd> + <kbd>Left Arrow</kbd>|開始日の現在の部分を選択します、または前の部分を選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|現在のチャンクの末尾を選択します、またはチャンク 1 つ分右に移動します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Right Arrow</kbd>|現在のチャンクの先頭を選択します、またはチャンク 1 つ分左に移動します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Home</kbd> または <kbd>Shift</kbd> + <kbd>Home</kbd>|入力内の値を先頭まで選択します
<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>End</kbd> または <kbd>Shift</kbd> + <kbd>End</kbd>|入力内の値を末尾まで選択します
<kbd>Ctrl</kbd> + <kbd>A</kbd>|入力内のすべての値を選択します 

## <a id="ddInteraction"></a> igDatePicker 固有のナビゲーション


押す|動作
---|---
<kbd>Alt</kbd> + <kbd>Down Arrow</kbd>|入力がフォーカスされている場合に、ドロップダウンを開きます
<kbd>Alt</kbd> + <kbd>Up Arrow</kbd>|入力がフォーカスされている場合に、ドロップダウンを閉じます
<kbd>Down Arrow</kbd>|マウス インジケータの位置に従って、日付の現在の部分をスピン ダウンします
<kbd>Up Arrow</kbd>|マウス インジケータの位置に従って、日付の現在の部分をスピン アップします
<kbd>Ctrl</kbd> + <kbd>Right Arrow</kbd>|現在の日付の一部に関連し、カレンダーの次の項目に移動します。
<kbd>Ctrl</kbd> + <kbd>Left Arrow</kbd>|現在の日付の一部に関連し、カレンダーの前の項目に移動します。
<kbd>Ctrl</kbd> + <kbd>Down Arrow</kbd> |月がフォーカスされてる場合は月から日に移動、または日がフォーカスされている場合は日付の下の行に移動します。
<kbd>Ctrl</kbd> + <kbd>Up Arrow</kbd> |カレンダーの一番上の行の日付がフォーカスされている場合は日から月に移動、またはカレンダーの一番上の行以外の行の日付がフォーカスされている場合は上の行に移動します。
<kbd>Enter</kbd>|フォーカスされた項目を選択済みとして確定します
<kbd>Enter</kbd> または <kbd>Alt</kbd> + <kbd>Up Arrow</kbd>|フォーカスされた項目を破棄し、古い選択を維持します
