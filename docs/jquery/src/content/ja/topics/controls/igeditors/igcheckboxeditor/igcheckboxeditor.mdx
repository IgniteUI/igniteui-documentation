# igCheckboxEditor

### 概要

`igCheckboxEditor` は、1 つまたは複数の相互に排他的なオプションのどちらか一方の選択をユーザーに許可するほか、チェック ボックスの値をサーバーに送信できるようにするコントロールです。このコントロールは、2 つの状態のトグルとして機能します。1 つは、オンの状態を反映し、もう 1 つは、 
オフ の状態を反映します。
