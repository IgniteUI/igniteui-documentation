# 仮想化 (igHierarchicalGrid)

## 概要
仮想化は、アクティブなイン メモリー DOM オブジェクトの数を減らす igHierarchicalGrid™ コントロールの機能です。仮想化は、仮想化されていないグリッドと比較してユーザーからは何も変わりません。
