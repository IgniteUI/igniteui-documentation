### <a id="Details"></a>詳細
サンプルでは、`igHtmlEditor` が AngularJS ディレクティブで初期化されています。	`igHtmlEditor` のデータソースは AngularJS コントローラーの変数に保存されます。データを保持する同じ変数は HTML `textarea` にバインドされます。データを `textarea` から更新する場合、`igHtmlEditor` のコンテンツが直ちに更新されます。
