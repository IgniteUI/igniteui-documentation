# チャートのアニメーション化 (チャートの Motion Framework)

##トピックの概要
### 概要

Infragistics® Motion Framework™ は、\{environment:ProductName\}™ チャート コントロールのデータ変更をアニメーション化するためのフレームワークです。

このセクションには、Motion Framework によるチャートのアニメーション化に関するトピックが記載されています。

-	[ASP.NET MVC でのチャートのアニメーション化 (igDataChart)](./02_Animating Charts in ASP.NET MVC.mdx): このトピックでは、コード例を使用して、AJAX POST 要求を使用してデータを動的に追加しているMVC で、簡単なアニメーション化された柱状チャートを追加する方法を示します。
