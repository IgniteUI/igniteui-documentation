# igTileManager

## このグループのトピックについて
### 概要

`igTileManager`™ は、データをタイル表示にレンダリングして管理するためのレイアウト コントロールです。タイルは、レスポンシブ グリッド レイアウト (グリッド、チャート、マップなどの異なるコンポーネントを持つダッシュボードなど) で表示されます。
