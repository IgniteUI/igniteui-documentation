## <a id="methods"></a> メソッド リファレンス

igDialog コントロールの各種メソッドの目的と機能を表にまとめると、次のようになります。

メソッド|説明
--- | ---
<ApiLink type="igDialog" member="state" section="methods" label="state" />|エディターの状態を取得します。
<ApiLink type="igDialog" member="mainElement" section="methods" label="mainElement" /> |動的に作成される div 要素 (ダイアログを表す要素) への参照を取得します。
<ApiLink type="igDialog" member="content" section="methods" label="content" /> |`igDialog` コンテンツのコンテナーを取得するか、提供された新しいコンテンツを設定します。ヘッダー、サイズ変更のハンドルなど以外のダイアログ ウィンドウの内部コンテナーです。
<ApiLink type="igDialog" member="close" section="methods" label="close" /> |開いている `igDialog` を閉じます。
<ApiLink type="igDialog" member="open" section="methods" label="open" /> |閉じている `igDialog` を開きます。
<ApiLink type="igDialog" member="minimize" section="methods" label="minimize" /> |開いている `igDialog` を最小化します。
<ApiLink type="igDialog" member="maximize" section="methods" label="maximize" /> |開いている `igDialog` を最大化します。
<ApiLink type="igDialog" member="pin" section="methods" label="pin" /> |ピン固定されていない `igDialog` をピン固定します。
<ApiLink type="igDialog" member="unpin" section="methods" label="unpin" /> |ピン固定されている `igDialog` の固定を解除します。
<ApiLink type="igDialog" member="getTopModal" section="methods" label="getTopModal" /> |一番上のモーダル ダイアログへの参照を取得します。
<ApiLink type="igDialog" member="isTopModal" section="methods" label="isTopModal" /> |ダイアログがモーダルで現在アクティブであることを示します。
<ApiLink type="igDialog" member="moveToTop" section="methods" label="moveToTop" /> |ダイアログを一番上に移動させます。
<ApiLink type="igDialog" member="destroy" section="methods" label="destroy" />|`igDialog` コントロールを破棄します。
