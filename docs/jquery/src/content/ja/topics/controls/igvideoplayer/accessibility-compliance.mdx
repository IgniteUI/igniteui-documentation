# アクセシビリティの遵守 (igVideoPlayer)

## ビデオ プレーヤー アクセシビリティの遵守
すべての &#123;environment:ProductName&#125;™ コントロールおよびコンポーネントは、1973 年リハビリテーション法第 508 条第 1194 部 22 条を遵守しています。表 1 には、コントロールに関連する第 1194 部 22 条の特定の規則が記載されています。また、ビデオ プレーヤー コントロールが各規則を遵守するための詳しい方法も含んでいます。

各アクセシビリティ規則の要件を満たすために、場合によっては、コントロールを操作して特定のプロパティを設定する必要がありますが、それ以外の場合は、コントロール自身がこの作業を行います。

>**注:** jQuery コントロールはクライアント専用のため、一部の規則はサポートされておらず、制限として扱います。

**表 1: 第 508 条遵守の説明**


| 規則 | 規則を遵守する方法 |
| --- | --- |
| (c) | クライアント側イベントを使用すると、現在の状態を反映する任意のコントロールの DOM 要素の属性をユーザーは変更できます。 |
| (d)* | コントロールのマークアップは、クライアント側のコントロールであり、CSS の規則に依存するため、関連付けられたスタイル シートがないと読み取りできません。 |
| (j)* | コントロールは、画面のフリッカーが発生しないように設計されています。ただし、再生されるビデオにフリッカーに似た特定の効果が含まれている場合がありますが、コントロールでそれを制限したり、別の方法で再生を停止することはありません。 |
| (l) | コントロールのスクリプトがページ マークアップを変更している場合、規則 (c) で説明されている属性のために、新しく作成されたコンテンツはスクリーン リーダーに準拠します。 |
| (n) | jQuery Video Player は、ページのタブ シーケンスに含まれ、マウス クリックによるものを含むフォーカスを受け取ります。 右矢印 - 順方向に再生します。(スライダーにフォーカスがあるとき) 左矢印 - 逆方向に再生します。(スライダーにフォーカスがあるとき) SPACE - 現在の状態に応じてビデオを一時停止または再生します。 ESC - 全画面表示を終了します。 TAB - 再生コントロール間を移動し、それぞれのコントロール ボタンにフォーカスを設定します。これは、Infragistics 再生コントロールを使用しているときのみ機能します。埋め込みのブラウザー コントロールはキーボード ナビゲーションをサポートしていません。 |


\* - コントロールの制限
