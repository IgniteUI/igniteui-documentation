## igPopover のプロパティ リファレンス
### 概要

このトピックでは、`igPopover`™ コントロールのプロパティについて説明し、デフォルト値の一覧を示します。

以下の表では、`igPopover`™ コントロールのプロパティについて説明し、デフォルト値の一覧を示します。プロパティはアルファベット順に紹介します。

| <ApiLink type="igpopover" member="contentTemplate" section="options" label="contentTemplate" /> | string\|function | ポップオーバー コンテナーのコンテンツ。null 値のままの場合、コンテンツはターゲットから取り出されます。 | null |
