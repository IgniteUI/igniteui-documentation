# igPieChart



##このグループのトピックについて

### 概要

`igPieChart`™ コントロールは、HTML5 Canvas 要素に基づいて Web アプリケーションで各種のチャートを表示する機能を提供します。

![](../../images/images/igPieChart_Landing_Page_1.png)
