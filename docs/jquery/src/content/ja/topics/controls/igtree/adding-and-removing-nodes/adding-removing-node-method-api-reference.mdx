# ノードの追加/削除メソッド API のリファレンス (igTree)

## ノードの追加/削除メソッドのリファレンス
### 概要

ここでは、ノードの追加と削除に関する `igTree`™ コントロールのメソッドに関するリファレンス情報を紹介します。メソッドはアルファベット順に紹介します。

### 追加/削除メソッドのリファレンス チャート

以下の表では、`igTree` コントロールの追加/削除のメソッドの解説と、デフォルト値と推奨値をまとめました。


| メソッド | 説明 |
| --- | --- |
| [addNode](environment:jQueryApiUrl/ui.igtree#methods:addNode) | 新しいノード配列をツリーに追加します。新しいノードはルートか指定した親ノードに追加します。 引数名 |
| [node](environment:jQueryApiUrl/ui.igtree#methods:addNode) | 新しいノードを作成するデータを指定します。 |
| [parent](environment:jQueryApiUrl/ui.igtree#methods:addNode) | ノードを追加する、親ノードの jQuery オブジェクトを指定します。 |
