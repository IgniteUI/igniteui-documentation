# Angular Toolbar (ツールバー) の概要

Angular ツールバー コンポーネントは、主にチャート コンポーネントで使用される UI 操作のコンパニオン コンテナーです。ツールバーは、<ApiLink type="DataChart" /> または <ApiLink type="CategoryChart" /> コンポーネントにリンクされると、プロパティとツール項目のプリセットで動的に更新されます。プロジェクト用のカスタム ツールを作成して、エンド ユーザーが変更を提供できるようになり、無限のカスタマイズが可能になります。

## 依存関係


Ignite UI for Angular のレイアウト、入力、チャート、コア パッケージをインストールします。

```cmd
npm install igniteui-angular-layouts
npm install igniteui-angular-inputs
npm install igniteui-angular-charts
npm install igniteui-angular-core
```

<ApiLink type="DataChart" /> コンポーネントとその機能とともに <ApiLink type="Toolbar" /> を使用する場合、次のモジュールが必要です。



```ts
import { IgxToolbarModule } from 'igniteui-angular-layouts';
import { IgxDataChartToolbarModule, IgxDataChartCoreModule, IgxDataChartCategoryModule, IgxDataChartAnnotationModule, IgxDataChartInteractivityModule, IgxDataChartCategoryTrendLineModule  } from 'igniteui-angular-charts';

@NgModule({
    imports: [
        // ...
        IgxToolbarModule,
        IgxDataChartToolbarModule,
        IgxDataChartCoreModule,
        IgxDataChartCategoryModule,
        IgxDataChartAnnotationModule,
        IgxDataChartInteractivityModule,
        IgxDataChartCategoryTrendLineModule
        // ...
    ]
})
export class AppModule {}
```

## 使用方法

### ツール操作

以下は、ツールバーに追加できるさまざまな <ApiLink type="ToolAction" /> 項目のリストです。

- <ApiLink type="ToolActionButton" />
- <ApiLink type="ToolActionCheckbox" />
- <ApiLink type="ToolActionIconButton" />
- <ApiLink type="ToolActionIconMenu" />
- <ApiLink type="ToolActionLabel" />
- <ApiLink type="ToolActionNumberInput" />
- <ApiLink type="ToolActionRadio" />
- <ApiLink type="ToolActionSubPanel" />

これらのツールはそれぞれ、マウスのクリックによってトリガーされる `OnCommand` イベントを公開します。注: <ApiLink type="ToolActionIconMenu" /> は、<ApiLink type="ToolActionIconMenu" /> 内にラップすることもできる他のツールのラッパーです。

<ApiLink type="ToolAction" /> オブジェクトの <ApiLink type="ToolAction" member="overlayId" label="OverlayId" />、<ApiLink type="ToolAction" member="beforeId" label="BeforeId" />、および <ApiLink type="ToolAction" member="afterId" label="AfterId" /> プロパティを使用して、新規および既存のツールの位置を変更したり、非表示にマークしたりすることができます。ToolActions は <ApiLink type="ToolAction" member="visibility" label="Visibility" /> プロパティも公開します。

次の例は、いくつかの機能を示しています。まず、**ZoomReset** や **AnalyzeMenu** メニュー ツール操作などの組み込みツールを非表示にするなど、<ApiLink type="ToolActionSubPanel" /> でツールをグループ化できます。この例では、<ApiLink type="ToolAction" member="afterId" label="AfterId" /> プロパティを使用して **ZoomMenu** 内に **ZoomReset** ツール操作の新しいインスタンスを作成し、それを **ZoomOut** に割り当てて配置を正確にします。また、ツールの <ApiLink type="ToolAction" member="isHighlighted" label="IsHighlighted" /> プロパティによってもハイライト表示されます。

### Angular データ チャートの統合

Angular ツールバーには、<ApiLink type="Toolbar" member="Target" label="Target" /> プロパティが含まれています。これは、以下のコードに示すように、<ApiLink type="DataChart" /> などのコンポーネントをリンクするために使用されます。




```html
  <div class="legend">
    <igx-toolbar
      name="toolbar"
      [target]="chart"
      #toolbar>
    </igx-toolbar>
  </div>
  <div class="container fill">
    <igx-data-chart
    name="chart"
    #chart>
  </igx-data-chart>
```









<ApiLink type="DataChart" /> が Toolbar にリンクされると、いくつかの既存の <ApiLink type="ToolAction" /> 項目とメニューが使用可能になります。以下は、組み込みの Angular <ApiLink type="DataChart" /> ツール操作とそれに関連付けられた <ApiLink type="ToolAction" member="overlayId" label="OverlayId" /> のリストです。

ズーム操作

- `ZoomMenu`: チャートのズーム レベルを増減するための <ApiLink type="DomainChart" member="zoomIn" label="ZoomIn" /> および <ApiLink type="DomainChart" member="zoomOut" label="ZoomOut" /> メソッドを呼び出す 3 つの <ApiLink type="ToolActionLabel" /> 項目を公開する <ApiLink type="ToolActionIconMenu" /> には、チャートの <ApiLink type="DomainChart" member="resetZoom" label="ResetZoom" /> メソッドを呼び出してズーム レベルをデフォルトの位置にリセットする `ZoomReset` が含まれます。

トレンド操作

- `AnalyzeMenu`: チャートのさまざまなオプションを構成するためのいくつかのオプションを含む <ApiLink type="ToolActionIconMenu" />。
- `AnalyzeHeader`: サブ セクションのヘッダー。
  - `LinesMenu`: チャート上で水平破線を表示するためのさまざまなツールが含まれるサブ メニュー。
  - `LinesHeader`: 次の 3 つのツールのサブメニュー セクション ヘッダー:
   - `MaxValue`: シリーズの最大値で yAxis に沿って水平破線を表示する <ApiLink type="ToolActionCheckbox" />。
   - `MinValue`: シリーズの最小値で yAxis に沿って水平破線を表示する <ApiLink type="ToolActionCheckbox" />。
   - <ApiLink type="ValueLayerValueMode" member="Average" label="Average" />:  シリーズの平均値で yAxis に沿って水平破線を表示する <ApiLink type="ToolActionCheckbox" />。
  - `TrendsMenu`: さまざまな近似曲線を <ApiLink type="DataChart" /> プロット領域に適用するためのツールを含むサブ メニュー。
  - `TrendsHeader`: 次の 3 つのツールのサブメニュー セクション ヘッダー:
   - **Exponential**: チャート内の各シリーズの <ApiLink type="DomainChart" member="trendLineType" label="TrendLineType" /> を **ExponentialFit** に設定する <ApiLink type="ToolActionRadio" />。
   - **Linear**: チャート内の各シリーズの <ApiLink type="DomainChart" member="trendLineType" label="TrendLineType" /> を **LinearFit** に設定する <ApiLink type="ToolActionRadio" />。
   - **Logarithmic**: チャート内の各シリーズの <ApiLink type="DomainChart" member="trendLineType" label="TrendLineType" /> を **LogarithmicFit** に設定する <ApiLink type="ToolActionRadio" />。
- `HelpersHeader`: サブ セクションのヘッダー。
  - `SeriesAvg`: <ApiLink type="ValueLayerValueMode" member="Average" label="Average" /> タイプの <ApiLink type="ValueLayerValueMode" /> を使用して、チャートのシリーズ コレクションに <ApiLink type="ValueLayer" /> を追加または削除する <ApiLink type="ToolActionCheckbox" />。
  - `ValueLabelsMenu`: <ApiLink type="DataChart" /> のプロット領域に注釈を表示するためのさまざまなツールを含むサブ メニュー。
  - `ValueLabelsHeader`: 次のツールのサブ メニュー セクション ヘッダー:
   - `ShowValueLabels`: <ApiLink type="CalloutLayer" /> を使用してデータ ポイント値を切り替える <ApiLink type="ToolActionCheckbox" />。
   - `ShowLastValueLabel`: <ApiLink type="FinalValueLayer" /> を使用して最終値軸の注釈を切り替える <ApiLink type="ToolActionCheckbox" />。
- `ShowCrosshairs`: チャートの <ApiLink type="DomainChart" member="crosshairsDisplayMode" label="CrosshairsDisplayMode" /> プロパティを介してマウスオーバー十字線の注釈を切り替える <ApiLink type="ToolActionCheckbox" />。
- `ShowGridlines`: X-Axis に `MajorStroke` を適用することで追加のグリッド線を切り替える <ApiLink type="ToolActionCheckbox" />。

画像に保存アクション

- `CopyAsImage`: チャートをクリップボードにコピーするオプションを公開する <ApiLink type="ToolActionLabel" />。
- `CopyHeader`: サブ セクションのヘッダー。

### SVG アイコン

ツールを手動で追加する場合、`RenderIconFromText` メソッドを使用してアイコンを割り当てることができます。このメソッドには 3 つのパラメーターを渡す必要があります。1 つ目は、ツールで定義されたアイコン コレクション名です (例: <ApiLink type="ToolAction" member="iconCollectionName" label="IconCollectionName" />)。2 つ目は、ツールで定義されたアイコンの名前 (例: <ApiLink type="ToolAction" member="iconName" label="IconName" />) で、その後に SVG 文字列を追加します。

### データ URL アイコン

svg を追加するのと同様に、<ApiLink type="Toolbar" member="registerIconFromDataURL" label="RegisterIconFromDataURL" /> を介して URL からアイコン画像を追加することもできます。メソッドの 3 番目のパラメーターは、文字列 URL を入力するために使用されます。

次のスニペットは、アイコンを追加する両方の方法を示しています。



```html
<igx-tool-action-label
    title="Custom Icon"
    iconName="CustomIcon"
    iconCollectionName="CustomCollection">
</igx-tool-action-label>
```

### Vertical Orientation

By default the Angular Toolbar is shown horizontally, but it also has the ability to shown vertically by setting the <ApiLink type="Toolbar" member="orientation" label="Orientation" /> property.



```html
<igx-toolbar orientation="Vertical" />
```

### Color Editor

You can add a custom color editor tool to the the Angular Toolbar, which will also work with the Command event to perform custom styling to your application.



```html
<igx-toolbar
  name="toolbar"
  #toolbar>
      <igx-tool-action-color-editor
      title="Series Brush"
      name="colorEditorTool"
      #colorEditorTool>
      </igx-tool-action-color-editor>
</igx-toolbar>
```
