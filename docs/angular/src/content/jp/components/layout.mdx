# Angular Layout Manager (レイアウト マネジャー) ディレクティブの概要

<div class="highlight">
The Ignite UI for Angular Layout Directives allow developers to specify a layout direction for any children of the container it is applied to. Layout can flow vertically or horizontally, with controls for wrapping, justification, and alignment.
</div>
<div class="divider"></div>

## 使用方法

コンテナー要素で <ApiLink type="LayoutDirective" suffix={false} label="**igxLayout**" /> ディレクティブを使用して、子要素のレイアウト方向を指定します。

<hr/>

### 水平方向

<ApiLink type="LayoutDirective" suffix={false} member="dir" label="igxLayoutDir" />`="row"` を使用します。

<Sample src="/layouts/layout-direction-row" height={180} alt="" />

<hr/>

### 垂直方向

<ApiLink type="LayoutDirective" suffix={false} member="dir" label="igxLayoutDir" />`="column"` を使用します。

<Sample src="/layouts/layout-direction-column" height={300} alt="" />

<hr/>

<DocsAside type="note" title="注">
順序を反転するには、<ApiLink type="LayoutDirective" suffix={false} member="reverse" label="igxLayoutReverse" />`="true"` を使用します。
</DocsAside>

<hr/>

### 要素の順序をカスタマイズ

`igxFlexOrder` を使用して要素の順序をカスタマイズします。

<Sample src="/layouts/layout-custom-order" height={110} alt="" />

<hr/>

### 要素の間隔を変更

<ApiLink type="LayoutDirective" suffix={false} member="justify" label="igxLayoutJustify" />`="space-between | space-around"` を使用して、要素の間隔を変更します。

<Sample src="/layouts/layout-content-space" height={180} alt="" />

<hr/>

### 要素を主軸に沿って配置

<ApiLink type="LayoutDirective" suffix={false} member="justify" label="igxLayoutJustify" />`="flex-start | center | flex-end"` を使用して、要素を主軸に沿って配置します。

<Sample src="/layouts/layout-justify-content" height={180} alt="" />

<hr/>

### 要素を交差軸に沿って配置

<ApiLink type="LayoutDirective" suffix={false} member="itemalign" label="igxLayoutItemAlign" />`="flex-start | center | flex-end"` を使用して、要素を交差軸に沿って配置します。

<Sample src="/layouts/layout-align-items" height={400} alt="" />

<hr/>

### 要素をラップ

<ApiLink type="LayoutDirective" suffix={false} member="wrap" label="igxLayoutWrap" />`="wrap"` を使用して要素をラップすることもできます。

<Sample src="/layouts/layout-wrap" height={180} alt="" />

<DocsAside type="note" title="注">
<ApiLink type="LayoutDirective" suffix={false} label="igxLayout" /> ディレクティブはそのコンテナーの**直下**の子のフロー方向に影響します。
</DocsAside>

## ネスト

<ApiLink type="FlexDirective" suffix={false} label="igxFlex" /> ディレクティブを <ApiLink type="LayoutDirective" suffix={false} label="igxLayout" /> 親内の要素に使用して、特定のフレックスボックス プロパティを制御します。
