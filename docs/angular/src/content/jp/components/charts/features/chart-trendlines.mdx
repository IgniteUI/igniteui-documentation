# Angular チャート トレンドライン

Ignite UI for Angular チャートでは、トレンドラインはトレンドの識別やデータ内のパターンの検索に役立ちます。トレンドラインは、常にチャートにバインドされたデータ ポイントの前に描画されます。積層シリーズ、シェイプ シリーズ、および範囲シリーズを除き、これらは <ApiLink pkg="charts" type="CategoryChart" />、<ApiLink pkg="charts" type="FinancialChart" />、および <ApiLink pkg="charts" type="DataChart" /> (積層型シリーズ、シェイプ シリーズ、範囲シリーズを除く) でサポートされています。

トレンドラインはデフォルトでオフになっていますが、<ApiLink pkg="charts" type="CategoryChart" member="trendLineType" label="TrendLineType" /> プロパティを設定することで有効にできます。また、ブラシ、期間、太さなど、トレンドラインの複数の外観プロパティを変更できます。

トレンドラインを有効にすると、ダッシュ配列を適用することもできます。これを行うには、<ApiLink pkg="charts" type="CategoryChart" member="trendLineDashArray" label="TrendLineDashArray" /> プロパティを数値の配列に設定します。数値配列は、トレンドラインの破線の長さを表します。

## Angular チャートト レンドライン レイヤー

<ApiLink pkg="charts" type="TrendLineLayer" /> は、ターゲット シリーズに対して単一のトレンドライン タイプを表示するように設計されたシリーズ タイプです。これと既存のシリーズ タイプの既存のトレンド ライン機能との違いは、<ApiLink pkg="charts" type="TrendLineLayer" /> はシリーズ タイプであるため、チャートの <ApiLink pkg="charts" type="DataChart" member="Series" label="Series" /> コレクションに複数のトレンド ラインを追加して、同じシリーズに複数のトレンド ラインを添付できることです。また、これまでできなかったトレンドラインを凡例に表示する ことも可能です。

## トレンドライン レイヤーの使用

<ApiLink pkg="charts" type="TrendLineLayer" /> が正しく動作するには、<ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> と <ApiLink pkg="charts" type="TrendLineLayer" member="trendLineType" label="TrendLineType" /> を指定する必要があります。利用可能なさまざまなトレンドラインのタイプは、シリーズで利用可能なトレンドラインと同じです。

凡例に <ApiLink pkg="charts" type="TrendLineLayer" /> を表示する場合は、<ApiLink pkg="charts" type="TrendLineLayer" member="useLegend" label="UseLegend" /> プロパティを **true** に設定します。

## トレンドライン レイヤーのスタイル設定

デフォルトでは、<ApiLink pkg="charts" type="TrendLineLayer" /> は <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> と同じ色の破線で描画されます。これは、<ApiLink pkg="charts" type="TrendLineLayer" /> のさまざまなスタイル設定プロパティを使用して構成できます。

描画されるトレンドラインの色を変更するには、<ApiLink pkg="charts" type="TrendLineLayer" member="brush" label="Brush" /> プロパティを設定します。あるいは、<ApiLink pkg="charts" type="TrendLineLayer" member="useIndex" label="UseIndex" /> プロパティを **true** に設定することもできます。これにより、<ApiLink pkg="charts" type="TrendLineLayer" /> がチャートの <ApiLink pkg="charts" type="DataChart" member="Series" label="Series" /> コレクションに配置されているインデックスに基づいて、チャートの <ApiLink pkg="charts" type="DomainChart" member="brushes" label="Brushes" /> パレットからブラシが取得されます。

<ApiLink pkg="charts" type="TrendLineLayer" /> の表示方法は、<ApiLink pkg="charts" type="TrendLineLayer" member="appearanceMode" label="AppearanceMode" /> プロパティと <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> プロパティを使用して変更することもできます。<ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> は、-1.0 から 1.0 の範囲の値を受け取り、「Shift」 で終わるオプションに適用する 「シフト」 の量を決定します。

<ApiLink pkg="charts" type="TrendLineLayer" member="appearanceMode" label="AppearanceMode" /> プロパティのオプションは次のとおりです。

- `Auto`: デフォルトでは DashPattern 列挙体になります。
- `BrightnessShift`: トレンドラインは <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> ブラシを取得し、指定された <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> に基づいて明るさを変更します。
- `DashPattern`: トレンドラインは破線として表示されます。ダッシュの頻度は、<ApiLink pkg="charts" type="TrendLineLayer" /> の <ApiLink pkg="charts" type="TrendLineLayer" member="dashArray" label="DashArray" /> プロパティを使用して変更できます。
- `OpacityShift`: トレンドラインは <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> ブラシを取得し、指定された <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> に基づいて不透明度を変更します。
- `SaturationShift`: トレンドラインは <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> ブラシを取得し、指定された <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> に基づいてその彩度を変更します。
