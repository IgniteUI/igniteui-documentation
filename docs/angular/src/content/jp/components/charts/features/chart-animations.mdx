# Angular チャート アニメーション

アニメーションを使用すると、新しいデータ ソースを読み込むときにシリーズをイーズインできます。利用可能なアニメーションは、シリーズのタイプに基づきます。たとえば、縦棒シリーズは x 軸から上昇する描画アニメーションになります。折れ線シリーズは y 軸の原点から描画するアニメーションになります。

アニメーションは Ignite UI for Angular チャートで無効ですが、<ApiLink type="CategoryChart" member="isTransitionInEnabled" label="IsTransitionInEnabled" /> プロパティを true に設定することで有効にできます。そこから、<ApiLink type="CategoryChart" member="transitionInDuration" label="TransitionInDuration" /> プロパティを設定してアニメーションが完了するまでの時間を決定し、<ApiLink type="CategoryChart" member="transitionInMode" label="TransitionInMode" /> でアニメーションのタイプを決定できます。

## Angular チャート アニメーションの例

以下の例は、アニメーションをデフォルトの <ApiLink type="CategoryChart" member="transitionInMode" label="TransitionInMode" /> ("Auto") に設定した[折れ線チャート](../types/line-chart.mdx)を示しています。この例の一番上のドロップダウンとスライダーは、<ApiLink type="CategoryChart" member="transitionInMode" label="TransitionInMode" /> と <ApiLink type="CategoryChart" member="transitionInDuration" label="TransitionInDuration" /> をそれぞれ変更できるため、サポートされるさまざまなアニメーションが異なる速度でどのように見えるかを確認できます。
