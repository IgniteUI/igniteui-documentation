# Angular ポイント チャート

Ignite UI for Angular ポイント チャートは、ポイントのコレクションを描画します。値は Y 軸 (左側のラベル) に表示され、カテゴリは X 軸 (下部のラベル) に表示されます。これらのチャートは、プロットされた値の合計を表示することにより、一定期間の変化量を強調したり、複数の項目や全体の一部の関係を比較したりします。

## Angular ポイント チャートの例

<ApiLink pkg="charts" type="CategoryChart" /> コントロールで Angular  ポイント チャートを作成するには、以下の例のように、データを <ApiLink pkg="charts" type="CategoryChart" member="itemsSource" label="ItemsSource" /> プロパティにバインドし、<ApiLink pkg="charts" type="CategoryChart" member="chartType" label="CategoryChart.ChartType" /> プロパティを **Point** 列挙型に設定します。

## 複数シリーズの Angular ポイント チャート

Angular ポイント チャートを使用すると、複数のシリーズを組み合わせて時間の経過に伴う変化を比較または確認できます。中国と米国のデータを含むデータ ソースにバインドするだけで、ポイント チャートは追加データに合わせて自動的に更新されます。
