## **20.2.0 (November 2025)**

### igniteui-angular-charts (チャート)

#### <Badge variant="preview" /> ユーザー注釈

Ignite UI for Angular では、ユーザー注釈機能により、実行時に <ApiLink type="DataChart" /> にスライス注釈、ストリップ注釈、ポイント注釈を追加できるようになりました。これにより、エンドユーザーは、スライス注釈を使用して会社の四半期レポートなどの単一の重要イベントを強調したり、ストリップ注釈を使用して期間を持つイベントを示したりすることで、プロットに詳細を追加できます。ポイント注釈またはこれら 3 つの任意の組み合わせを使用して、プロットされたシリーズ上の個々のポイントを呼び出すこともできます。

これは、<ApiLink type="Toolbar" /> のデフォルトのツールと統合されています。

<Image src={dataChartUserAnnotationCreate} alt="Angular user-annotation-create" class="responsive-img" />

#### <Badge variant="preview" /> 軸注釈の衝突検出

軸注釈が自動で衝突を検出し、適切に収まるよう切り詰めます。この機能を有効にするには、次のプロパティを設定します:

- <ApiLink type="TimeXAxis" member="shouldAvoidAnnotationCollisions" label="ShouldAvoidAnnotationCollisions" />
- <ApiLink type="TimeXAxis" member="shouldAutoTruncateAnnotations" label="ShouldAutoTruncateAnnotations" />

## **20.1.0 (September 2025)**

### igniteui-angular-maps (地理マップ)

#### <Badge variant="preview" /> Azure マップ画像のサポート

<ApiLink type="IgxGeographicMap" /> は、 Azure ベースのマップ画像をサポートし、開発者は複数のアプリケーション タイプにわたって詳細かつ動的なマップを表示できるようになりました。複数のマップ レイヤーを組み合わせて地理データを視覚化し、インタラクティブなマッピング エクスペリエンスを簡単に作成できます。

注: Bing マップ画像のサポートは段階的に廃止されます。既存のエンタープライズ キーは引き続き Bing Maps にアクセスするために利用できるため、Azure Maps へ移行する間も現在のアプリケーションをそのまま利用可能です。

公開されている Azure Maps の一部は[こちら](https://azure.microsoft.com/ja-jp/products/azure-maps)をご覧ください。

### igniteui-angular-charts (チャート)

#### <Badge variant="preview" /> 新しい軸ラベル イベント

軸ラベルに対するさまざまな操作を検出できるように、次のイベントが <ApiLink type="IgxDataChart" /> に追加されました。

- `LabelMouseDown`
- `LabelMouseUp`
- `LabelMouseEnter`
- `LabelMouseLeave`
- `LabelMouseMove`
- `LabelMouseClick`

#### <Badge variant="preview" /> 対応軸

X 軸と Y 軸に `CompanionAxis` プロパティが追加され、既存の軸を簡単に複製できるようになりました。<ApiLink type="TimeXAxis" member="companionAxisEnabled" label="CompanionAxisEnabled" /> プロパティを有効にすると、複製された軸はチャートの反対側に配置され、そこから各軸プロパティを設定できます。

#### <Badge variant="preview" /> RadialPieSeries インセット アウトライン

<ApiLink type="RadialPieSeries" /> のアウトライン レンダリング方法を制御するために <ApiLink type="DataPieChart" member="useInsetOutlines" label="UseInsetOutlines" /> プロパティが追加されました。**true** に設定すると、アウトラインがスライス形状の内側に描画され、**false** (既定値) に設定すると、アウトラインはスライス形状の端に半分内側・半分外側で描画されます。

**重大な変更**

- <ApiLink type="ChartMouseEventArgs" /> クラスの <ApiLink type="ChartMouseEventArgs" member="plotAreaPosition" label="PlotAreaPosition" /> プロパティと <ApiLink type="ChartMouseEventArgs" member="chartPosition" label="ChartPosition" /> プロパティが逆になっている問題が修正されました。これにより、<ApiLink type="ChartMouseEventArgs" member="plotAreaPosition" label="PlotAreaPosition" /> と <ApiLink type="ChartMouseEventArgs" member="chartPosition" label="ChartPosition" /> が返す値が変更されます。

        ターゲット チャートは、行われた変更にリアルタイムで反応します。チャートに ItemsSource が設定されていない場合、IconMenu は非表示になります。
