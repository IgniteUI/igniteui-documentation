# Умения на агента за Ignite UI for Angular

Ignite UI for Angular се доставя с **[Agent Skills](https://agentskills.io/)** - структурирани файлове със знания, които учат AI помощници за кодиране (GitHub Copilot, Cursor, Windsurf, Claude, Gemini CLI, JetBrains Junie и т.н.) как да работят с Ignite UI for Angular. Тези файлове със умения предоставят контекстно-съответстващо ръководство за компоненти, grid, операции с данни и теми, което позволява на вашия AI помощник да генерира точен, идиоматичен код, който следва най-добрите практики.

<DocsAside type="info">
Пейзажът на AI инструментите се развива бързо. Местоположенията за откриване на умения и опциите за разпространение могат да се променят, когато инструментите и IDE-та се актуализират. Винаги консултирайте официалната документация на вашия конкретен инструмент или агент за най-новата информация.
</DocsAside>

## Наличните умения

Файлите със умения се намират в директорията [`skills/`](https://github.com/IgniteUI/igniteui-angular/tree/master/skills) на хранилището на Ignite UI for Angular:

| Умение                     | Пътека                     | Описание                                                                                       |
| :------------------------- | :------------------------- | :------------------------------------------------------------------------------------------------ |
| Компоненти и оформление    | [`skills/igniteui-angular-components/SKILL.md`](https://github.com/IgniteUI/igniteui-angular/blob/master/skills/igniteui-angular-components/SKILL.md) | Самостойни компоненти, управление на форми, overlay, оформление                                            |
| Grid за данни              | [`skills/igniteui-angular-grids/SKILL.md`](https://github.com/IgniteUI/igniteui-angular/blob/master/skills/igniteui-angular-grids/SKILL.md) | Grid, Tree Grid, Hierarchical Grid, Pivot Grid, сортиране, филтриране, групиране, разбиване, дистанционни данни |
| Теми и стилизиране         | [`skills/igniteui-angular-theming/SKILL.md`](https://github.com/IgniteUI/igniteui-angular/blob/master/skills/igniteui-angular-theming/SKILL.md) | Палитри, типография, възвишения, теми на компоненти, MCP сървър                                    |
| Генериране от дизайн на изображение | [`skills/igniteui-angular-generate-from-image-design/SKILL.md`](https://github.com/IgniteUI/igniteui-angular/blob/master/skills/igniteui-angular-generate-from-image-design/SKILL.md) | Конструирайте Angular приложения от снимки на екрана, макети и скици с помощта на компоненти на Ignite UI           |

<DocsAside type="info">
Начинавайки с Ignite UI for Angular **21.1.0**, тези умения се откриват автоматично, когато бъдат поставени в пътя на уменията на вашия агент (например `.claude/skills`, `.agents/skills`, `.cursor/rules/`). Този пуск включва опционална миграция за автоматично добавяне на тези умения към вашия проект.
</DocsAside>

## Местоположения на умения

Всеки инструмент за AI кодиране открива умения от специфични директории. Поставете файловете със умения в подходящото местоположение, така че вашият AI помощник да може да ги открие и използва автоматично. Общата конвенция `.agents/skills/` се поддържа в множество инструменти, докато всеки инструмент има и своите собствени специфични директории.

### Общо (`.agents/skills/`)

Директорията `.agents/skills/` е кроссагентна конвенция, поддържана от множество инструменти, включително [VS Code с Copilot](https://code.visualstudio.com/docs/copilot/customization/agent-skills), [OpenAI Codex](https://developers.openai.com/codex/skills), [Cursor](https://cursor.com/docs/skills), [Gemini CLI](https://geminicli.com/docs/cli/skills/#skill-discovery-tiers), [Antigravity](https://antigravity.google/docs/skills) и [Windsurf](https://docs.windsurf.com/windsurf/cascade/skills#skill-scopes). Копирайте директориите със умения в `.agents/skills/` в коренът на вашия проект:

```
.agents/
  skills/
    igniteui-angular-components/
      SKILL.md
    igniteui-angular-grids/
      SKILL.md
    igniteui-angular-theming/
      SKILL.md
    igniteui-angular-generate-from-image-design/
      SKILL.md
```

За умения на ниво потребител (глобални), налични на всички проекти, вместо това използвайте `~/.agents/skills/`.

### GitHub Copilot

[GitHub Copilot](https://docs.github.com/en/copilot/concepts/agents/about-agent-skills) открива умения от:

| Обхват   | Местоположение                                                                          |
| :------- | :-------------------------------------------------------------------------------- |
| Проект   | `.github/skills/`, `.claude/skills/`                                                    |
| Лично    | `~/.copilot/skills/`, `~/.claude/skills/` (само Copilot кодиращ агент и GitHub Copilot CLI) |

> **Съвет:** В [VS Code](https://code.visualstudio.com/docs/copilot/customization/agent-skills), тези местоположения също включват общото `.agents/skills/` и `~/.agents/skills/` и можете да конфигурирате допълнителни местоположения за умения, използвайки настройката `chat.agentSkillsLocations`.

### Claude

[Claude](https://code.claude.com/docs/en/skills#where-skills-live) открива умения от:

| Обхват   | Местоположение |
| :------- | :------------- |
| Проект   | `.claude/skills/` |
| Лично    | `~/.claude/skills/` |

### Cursor

[Cursor](https://cursor.com/docs/skills#skill-directories) открива умения от:

| Обхват         | Местоположение                       |
| :------------ | :----------------------------- |
| Проект         | `.agents/skills/`, `.cursor/skills/` |
| Потребител (глобално) | `~/.cursor/skills/`                 |

### Gemini CLI и Antigravity

[Gemini CLI](https://geminicli.com/docs/cli/skills/#skill-discovery-tiers) и [Antigravity](https://antigravity.google/docs/skills) откриват умения от:

| Обхват     | Местоположение                       |
| :-------- | :----------------------------- |
| Работно пространство | `.gemini/skills/`, `.agents/skills/` |
| Потребител | `~/.gemini/skills/`, `~/.agents/skills/` |

## Инсталиране на умения

Използвайте една от опциите по-долу, за да изтеглите и поставите файловете със умения в подходящото [местоположение на умения](#skill-locations) за вашия AI помощник.

### **Опция A - Използвайте Ignite UI CLI**

Командата `ai-config` конфигурира MCP сървъри, копира файлове със умения, специфични за рамката, в директорията на умения на всеки агент и настройва инструкционни файлове - всичко в един етап. Използвайте `--assistants`, за да изберете кои помощници за кодиране получават MCP конфигурация и `--agents`, за да изберете кои агенти получават файлове със умения. Съществуващите файлове се актуализират само, ако съдържанието им се е променило. Ако не са предоставени параметри, командата влиза в интерактивен режим, подсказвайки ви да изберете помощници и агенти. За налични опции вижте таблицата по-долу.

```bash
ig ai-config --assistants generic --agents claude
```

Използвайте `--agents` с множество стойности, за да целите няколко агенти наведнъж:

```bash
ig ai-config --assistants generic vscode --agents claude copilot cursor
```

| Флаг           | Стойности                                                                                                                                         | Подразбиране                |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| `--assistants` | `generic`, `vscode`, `cursor`, `gemini`, `junie`, `none`                                                 | Подсказване интерактивно |
| `--agents` | `generic`, `claude`, `copilot`, `cursor`, `codex`, `windsurf`, `gemini`, `junie`, `none` | Подсказване интерактивно |

**Използване на Angular Schematics:**

```bash
ng generate @igniteui/angular-schematics:ai-config --assistants cursor --agents claude copilot
```

Това също регистрира `@angular/cli` MCP сървър заедно със сървърите на Ignite UI.

<DocsAside type="info">
Ако сте инсталирали Ignite UI for Angular ръчно и желаете да копирате умения без да стартирате `ai-config`, файловете на уменията са налични и под `node_modules`. За да ги копирате в проекта си (например в `.agents/skills/`), изпълнете:
</DocsAside>

**macOS / Linux / Windows (PowerShell)**

```bash
cp -r node_modules/igniteui-angular/skills/. .agents/skills/
```

**Windows (Command Prompt)**

```cmd
robocopy node_modules\igniteui-angular\skills .agents\skills /E
```

Или копирайте отделни директории със умения според необходимост:

**macOS / Linux / Windows (PowerShell)**

```bash
cp -r node_modules/igniteui-angular/skills/igniteui-angular-components .agents/skills/
cp -r node_modules/igniteui-angular/skills/igniteui-angular-grids .agents/skills/
cp -r node_modules/igniteui-angular/skills/igniteui-angular-theming .agents/skills/
cp -r node_modules/igniteui-angular/skills/igniteui-angular-generate-from-image-design .agents/skills/
```

**Windows (Command Prompt)**

```cmd
robocopy node_modules\igniteui-angular\skills\igniteui-angular-components .agents\skills\igniteui-angular-components /E
robocopy node_modules\igniteui-angular\skills\igniteui-angular-grids .agents\skills\igniteui-angular-grids /E
robocopy node_modules\igniteui-angular\skills\igniteui-angular-theming .agents\skills\igniteui-angular-theming /E
robocopy node_modules\igniteui-angular\skills\igniteui-angular-generate-from-image-design .agents\skills\igniteui-angular-generate-from-image-design /E
```

### **Опция B - Използвайте `GitHub CLI`**

GitHub CLI може да се използва за изтегляне на файлове със умения директно от хранилището на Ignite UI for Angular. Изпълнете следните команди в коренът на вашия проект, за да стартирате процеса на инсталиране:

```bash
gh skill install IgniteUI/igniteui-angular
```

Ще бъдете попитани да изберете кои умения да инсталирате и целевите агенти за файловете със умения в вашия проект. CLI ще изтегли и поставит избраното умение според избраните агенти.

За актуализиране на умения по-късно, изпълнете следната команда:

```bash
gh skill update IgniteUI/igniteui-angular
```

### **Опция C - Използвайте `gemini skills` CLI**

Командата `gemini skills install` инсталира умения директно от Git хранилище. Поддържа два обхвата:

- **Обхват на потребител** (подразбиране) - инсталира умения глобално за вашия потребителски профил, което ги прави налични на всички проекти. Умениятa се поставят в `~/.gemini/skills/` или `~/.agents/skills/`.
- **Обхват на работното пространство** - инсталира умения локално в текущата директория на проекта под `.agents/skills/`, ограничавайки ги само до този проект.

**Инсталация в обхват на потребител (подразбиране):**

```bash
gemini skills install https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-components
gemini skills install https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-grids
gemini skills install https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-theming
gemini skills install https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-generate-from-image-design
```

**Инсталация в обхват на работното пространство:**

```bash
gemini skills install --scope workspace https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-components
gemini skills install --scope workspace https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-grids
gemini skills install --scope workspace https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-theming
gemini skills install --scope workspace https://github.com/IgniteUI/igniteui-angular.git --path skills/igniteui-angular-generate-from-image-design
```

След инсталацията файловете със умения са налични в съответното местоположение и автоматично ще бъдат открити от съвместими AI помощници.

### **Опция D - Използвайте `npx skills` CLI**

CLI-то `skills` е интерактивен инструмент, който изтегля и инсталира умения директно в вашия проект. Изпълнете следната команда в коренът на вашия проект:

```bash
npx skills add IgniteUI/igniteui-angular
```

CLI ще ви насочи през серия от подсказвания, за да:

1. Изберете кои умения да инсталирате (компоненти, теми и т.н.).
2. Изберете целевото местоположение за файловете със умения в вашия проект (например `.agents/skills/`, `.github/skills/`).
3. Автоматично изтегляне и запис на избраните файлове със умения.

След завършване умениятa са готови за использ - не е необходимо ръчно копиране на файлове.

> **Забележка:** Изисква Node.js и интернет връзка. Командата извлича най-новите файлове със умения от хранилището [IgniteUI/igniteui-angular](https://github.com/IgniteUI/igniteui-angular).

## Theming MCP сървър

**Умението Theming** включва инструкции за настройка на `igniteui-theming` MCP сървърът, което дава на AI помощници достъп до живи инструменти за теми, като например генериране на палитра и скелет на теми на компоненти. Вижте [файлът със умението Theming](https://github.com/IgniteUI/igniteui-angular/blob/master/skills/igniteui-angular-theming/SKILL.md) за стъпки на конфигурация за VS Code, Cursor, Claude Desktop и JetBrains IDE.

За повече информация относно Theming MCP, вижте документацията на [Ignite UI Theming MCP](/ai/theming-mcp).
