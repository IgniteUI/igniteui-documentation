## Първи стъпки с Ignite UI for Angular Avatar

За да започнете с компонента Ignite UI for Angular Avatar, първо трябва да инсталирате Ignite UI for Angular. В съществуващо Angular приложение въведете следната команда:

```cmd
ng add igniteui-angular
```

За пълно запознаване с Ignite UI for Angular прочетете темата [*първи стъпки*](/general/getting-started).

Следващата стъпка е да импортирате `IgxAvatarModule` във вашия файл **app.module.ts**.

```typescript
// app.module.ts

...
import { IgxAvatarModule } from 'igniteui-angular/avatar';
// import { IgxAvatarModule } from '@infragistics/igniteui-angular'; for licensed package

@NgModule({
    ...
    imports: [..., IgxAvatarModule],
    ...
})
export class AppModule {}
```

Алтернативно, от `16.0.0` можете да импортирате `IgxAvatarComponent` като самостоятелна зависимост.

```typescript
// home.component.ts

...
import { IgxAvatarComponent } from 'igniteui-angular/avatar';
// import { IgxAvatarComponent } from '@infragistics/igniteui-angular'; for licensed package

@Component({
    selector: 'app-home',
    template: '<igx-avatar shape="circle"></igx-avatar>',
    styleUrls: ['home.component.scss'],
    standalone: true,
    imports: [IgxAvatarComponent]
})
export class HomeComponent {}
```

Сега, когато имате импортиран модулът или компонента Ignite UI for Angular Avatar, можете да започнете със базова конфигурация на компонента `igx-avatar`.

## Използване на Angular Avatar компонент

Компонентът Ignite UI for Angular Avatar идва с три форми (квадрат, закръглено и кръг) и три опции за размер (малко, средно и голямо). Може да се използва за показване на инициали, изображения или икони.

### Форма на Avatar

Можем да променим формата на avatar чрез атрибута `shape`, задавайки неговата стойност на `square`, `rounded` или `circle`. По подразбиране формата на avatar е `square`.

```html
<igx-avatar shape="circle"></igx-avatar>
```

### Avatar показване на инициали

За да получите прост avatar с <ApiLink type="Avatar" member="initials" label="initials" /> (например JS за 'Jack Sock'), добавете следния код в шаблона на компонента:

```html
<igx-avatar initials="JS" shape="circle"></igx-avatar>
```

Нека подобрим нашия avatar, като го направим кръгов и по-голям по размер.

```html
<igx-avatar size="medium" initials="JS" shape="circle"></igx-avatar>
```

Можем също да променим фона, използвайки CSS променливата `--ig-avatar-background` или да зададем цвят на инициалите чрез свойството `color`.

```scss
// avatar.component.scss

igx-avatar {
  --ig-avatar-background: #e41c77;
  color: #000000;
}

```

<DocsAside type="warning">
Свойството `roundShape` на компонента `igx-avatar` е остаряло. Вместо него трябва да се използва атрибутът `shape`.
</DocsAside>

Ако всичко е минало добре, трябва да видите следното в браузъра:

<Sample src="/layouts/avatar-sample-1" height={100} iframeOnly />

### Avatar показване на изображение

За да получите avatar, който показва изображение, всичко, което трябва да направите, е да зададете източника на изображението чрез свойството `src`.

```html
<igx-avatar src="https://randomuser.me/api/portraits/men/1.jpg"
            shape="rounded"
            size="large">
</igx-avatar>
```

Ако всичко е минало добре, трябва да видите следното в браузъра:

<Sample src="/layouts/avatar-sample-2" height={100} iframeOnly />

### Avatar показване на икона

По подобен начин avatar може да показва икона чрез свойството <ApiLink type="Avatar" member="icon" label="icon" />. В момента се поддържат всички икони от набора от материални икони.

```html
<igx-avatar icon="person"
            shape="rounded"
            size="small">
</igx-avatar>
```

<DocsAside type="info">
Този компонент използва Material Icons. Добавете следния линк към вашия `index.html`: `<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">`
</DocsAside>

## Оформяне

### Avatar тема карта на свойствата

Промяната на свойството `$background` автоматично актуализира следните зависими свойства:

| Основно свойство | Зависимо свойство | Описание                                      |
| ---------------- | ----------------- | --------------------------------------------- |
| **$background**  | $color            | Цветът на текста, използван за avatar.       |
|                  | $icon-color       | Цветът на иконата, използван за avatar.      |

За да начнете с оформяне на avatar, трябва да импортирате файла `index`, където живеят всички функции на темата и компонентни миксини:

```scss
@use "igniteui-angular/theming" as *;

// IMPORTANT: Prior to Ignite UI for Angular version 13 use:
// @import '~igniteui-angular/lib/core/styles/themes/index';
```

Следвайки най-простия подход, създаваме нова тема, която разширява <ApiLink kind="sass" module="themes" type="function-avatar-theme" label="avatar-theme" code={true} /> и предоставя стойности за параметрите `$background` и `$border-radius`. `$color` (или `$icon-color`) се задава автоматично на черно или бяло, в зависимост от това кое осигурява по-добър контраст със зададения фон. Имайте предвид, че свойството `$border-radius` поражда ефект само когато `shape` на avatar е зададено на `rounded`.

При следния маркаж:

```html
<div class="avatar-sample initials">

  <igx-avatar initials="JS" shape="rounded" size="medium"></igx-avatar>
</div>
```

Създаваме следната тема на avatar:

```scss
$custom-avatar-theme: avatar-theme(
  $background: #72da67,
  $border-radius: 16px
);
```

Последната стъпка е да предадете потребителската тема на avatar:

```scss
.initials {
  @include tokens($custom-avatar-theme);
}
```

Ако всичко е минало добре, трябва да видите следното в браузъра:

### Оформяне с Tailwind

Можете да оформите `avatar` използвайки нашите потребителски Tailwind служебни класове. Предварително [конфигурирайте Tailwind](/themes/misc/tailwind-classes).

Заедно с импорта на tailwind във вашия глобален стилен лист, можете да приложите желаните служебни класове на тема както следва:

```scss
@import "tailwindcss";
...
@use 'igniteui-theming/tailwind/utilities/material.css';
```

Файлът на служебните класове включва както `light` така и `dark` варианти на тема.

- Използвайте `light-*` класове за светлата тема.
- Използвайте `dark-*` класове за тъмната тема.
- Допълнете името на компонента след префикса, например `light-avatar`, `dark-avatar`.

След прилагането тези класове активират динамични изчисления на тема. Оттам можете да замените генерираните CSS променливи, използвайки `arbitrary properties`. След двоеточието предоставете всеки валиден CSS цветен формат (HEX, CSS променлива, RGB и т.н.).

Можете да намерите пълния списък на свойствата в <ApiLink kind="sass" module="themes" type="function-avatar-theme" label="avatar-theme" />. Синтаксисът е както следва:

```html
<igx-avatar
class="!light-avatar ![--background:#FF4E00]"
initials="DY"
shape="rounded">
</igx-avatar>
```

<DocsAside type="info">
Знакът за възклицание(`!`) е необходим, за да се гарантира, че класът на помощната програма има предимство. Tailwind прилага стилове в слоеве, и без да маркирате тези стилове като важни, те ще бъдат заменени от темата по подразбиране на компонента.
</DocsAside>

В крайна сметка вашият avatar трябва да изглежда така:

### Потребителско оразмеряване

Можете да използвате променливата `--size`, насочена към `igx-avatar` директно:

```scss
igx-avatar {
  --size: 200px;
}
```

Или можете да използвате универсалната променлива `--ig-avatar-size` за насочване на всички екземпляри:

```html
<div class="my-app">

  <igx-avatar></igx-avatar>
</div>
```

```scss
.my-app {
  --ig-avatar-size: 200px;
}
```

Можете също да използвате един от предварително дефинираните размери, назначавайки го на променливата `--ig-size`, ако няма приложен размерен атрибут. Наличните стойности за `--ig-size` са `--ig-size-small`, `--ig-size-medium` и `--ig-size-large`:

```scss
igx-avatar {
  --ig-size: var(--ig-size-small);
}
```

Научете повече за това в статията [Size](/display-density).
