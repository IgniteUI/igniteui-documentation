# Angular Layout Manager Directives Overview

<div class="highlight">
The Ignite UI for Angular Layout Directives allow developers to specify a layout direction for any children of the container it is applied to. Layout can flow vertically or horizontally, with controls for wrapping, justification, and alignment.
</div>
<div class="divider"></div>

## Usage

Use the <ApiLink type="LayoutDirective" suffix={false} label="**igxLayout**" /> directive on a container element to specify the layout direction for its children:

<hr/>

### Horizontally with Row direction

Use <ApiLink type="LayoutDirective" suffix={false} member="dir" label="igxLayoutDir" />`="row"`.

<Sample src="/layouts/layout-direction-row" height={180} alt="" />

<hr/>

### Vertically with Column direction

Use <ApiLink type="LayoutDirective" suffix={false} member="dir" label="igxLayoutDir" />`="column"`.

<Sample src="/layouts/layout-direction-column" height={300} alt="" />

<hr/>

<DocsAside type="note">
Reverse order using <ApiLink type="LayoutDirective" suffix={false} member="reverse" label="igxLayoutReverse" />`="true"`.
</DocsAside>

<hr/>

### Customize the order of the elements

Customize the order of the element by using `igxFlexOrder`.

<Sample src="/layouts/layout-custom-order" height={110} alt="" />

<hr/>

### Change element spacing

Use <ApiLink type="LayoutDirective" suffix={false} member="justify" label="igxLayoutJustify" />`="space-between | space-around"`.

<Sample src="/layouts/layout-content-space" height={180} alt="" />

<hr/>

### Position elements along the main axis

Use <ApiLink type="LayoutDirective" suffix={false} member="justify" label="igxLayoutJustify" />`="flex-start | center | flex-end"` to specify the elements position along the main axis according to your preferences.

<Sample src="/layouts/layout-justify-content" height={180} alt="" />

<hr/>

### Position elements along the cross axis

Use <ApiLink type="LayoutDirective" suffix={false} member="itemalign" label="igxLayoutItemAlign" />`="flex-start | center | flex-end"` to specify the elements position along the cross axis according to your preferences.

<Sample src="/layouts/layout-align-items" height={400} alt="" />

<hr/>

### You can also wrap elements

Use <ApiLink type="LayoutDirective" suffix={false} member="wrap" label="igxLayoutWrap" />`="wrap"`.

<Sample src="/layouts/layout-wrap" height={180} alt="" />

<DocsAside type="note">
The <ApiLink type="LayoutDirective" suffix={false} label="igxLayout" /> directive affects the flow directions for that
</DocsAside>
container's **immediate** children.
<hr/>

## Nesting

Use the <ApiLink type="FlexDirective" suffix={false} label="igxFlex" /> directive for elements inside an <ApiLink type="LayoutDirective" suffix={false} label="igxLayout" /> parent to control specific flexbox properties.
