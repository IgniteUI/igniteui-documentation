# Angular Navigating Map Content

Navigation in the <ApiLink type="GeographicMap" /> control is enabled by default and it allows zooming and panning of the map content. However, this behavior can be changed using the <ApiLink type="GeographicMap" member="Zoomable" label="Zoomable" /> property. It is important to know that the map allows only synchronized zooming - scaling the map content with preserved aspect ratio. As result, it is not possible to scale the map content vertically without scaling it also horizontally and vice versa.

## Properties
The following table summarizes properties that can be used in navigation of the <ApiLink type="GeographicMap" /> control:

| Property Name  | Property Type   | Description   |
|----------------|-----------------|---------------|
|<ApiLink type="GeographicMap" member="WindowRect" label="WindowRect" />| Rect | Sets new position and size of the navigation window in viewable area of the map content. Rect with 0, 0, 1, 1 values will zoom out the entire map content in the navigation window. |
|<ApiLink type="GeographicMap" member="windowScale" label="WindowScale" />| number | Sets new size of the navigation window in of the map control. It is equivalent smallest value of Width or Height stored in the <ApiLink type="GeographicMap" member="WindowRect" label="WindowRect" /> property |
|<ApiLink type="SeriesViewer" member="windowPositionHorizontal" label="WindowPositionHorizontal" />| number | Sets new horizontal position of the navigation window’s anchor point from the left edge of the map control. It is equivalent to value stored in the Left of the <ApiLink type="GeographicMap" member="WindowRect" label="WindowRect" /> property. |
|<ApiLink type="SeriesViewer" member="windowPositionVertical" label="WindowPositionVertical" />| number | Sets new vertical position of the navigation window’s anchor point from the top edge of the map control. It is equivalent to value stored in the Top of the <ApiLink type="GeographicMap" member="WindowRect" label="WindowRect" /> property. |
|<ApiLink type="SeriesViewer" member="actualWindowRect" label="ActualWindowRect" />| Rect | Indicates current position and size of the navigation window in viewable area of the map content. Rect with 0, 0, 1, 1 values displays the entire map content in the navigation window.  |
|<ApiLink type="GeographicMap" member="actualWindowScale" label="ActualWindowScale" />| number | Indicates current size of the navigation window in of the map control. It is equivalent to smallest value of Width or Height stored in the <ApiLink type="SeriesViewer" member="actualWindowRect" label="ActualWindowRect" /> property |
|<ApiLink type="SeriesViewer" member="actualWindowPositionHorizontal" label="ActualWindowPositionHorizontal" />| number | Indicates current horizontal position of the navigation window’s anchor point from the left edge of the map control. It is equivalent to value stored in the Left of the <ApiLink type="SeriesViewer" member="actualWindowRect" label="ActualWindowRect" /> property. |
|<ApiLink type="SeriesViewer" member="actualWindowPositionVertical" label="ActualWindowPositionVertical" />| number | Indicates vertical position of the navigation window’s anchor point from the top edge of the map control. It is equivalent to value stored in the Top of the <ApiLink type="SeriesViewer" member="actualWindowRect" label="ActualWindowRect" /> property. |
