# Angular Spreadsheet Hyperlinks

The Angular Spreadsheet component allows display of pre-existing hyperlinks in your Excel workbook as well as insertion of new ones that can link to websites, file directories, and even other worksheets in the workbook.

## Hyperlinks Overview

Hyperlinks are added to the <ApiLink pkg="spreadsheet" type="Spreadsheet" /> control by accessing the `Hyperlinks` collection on the worksheet that you want to place the hyperlink on. This collection has an `Add` method that takes a <ApiLink pkg="excel" type="Worksheethyperlink" label="WorksheetHyperlink" /> object, where you can define the cell address, the hyperlink URL to be navigated to, the display text, and a tooltip to optionally be displayed on hover.

## Dependencies

When setting up your Angular spreadsheet control to use hyperlinks, you will need to import the <ApiLink pkg="excel" type="Worksheethyperlink" label="WorksheetHyperlink" /> class like so:


```ts
import { WorksheetHyperlink } from 'igniteui-angular-excel';
```
