# Angular Chart Highlighting Modes & Behaviors

All Angular Charts support a variety of highlighting options. <ApiLink pkg="charts" type="CategoryChart" member="highlightingMode" label="HighlightingMode" /> can be set to brighten or fade when the mouse is hovering over a series/data item rendered in the plot area. <ApiLink pkg="charts" type="CategoryChart" member="highlightingBehavior" label="HighlightingBehavior" /> can be set to directly over or the nearest data item to trigger the highlighting effect. Highlighting modes and behaviors is supported by the <ApiLink pkg="charts" type="CategoryChart" />, <ApiLink pkg="charts" type="FinancialChart" />, and <ApiLink pkg="charts" type="XamDataChart" /> controls and they have the same API for using the highlighting feature.

# Angular Chart Legend Highlighting

All Angular Charts support legend highlighting. <ApiLink pkg="charts" type="CategoryChart" member="legendHighlightingMode" label="LegendHighlightingMode" /> can enabled so that when mouse is hovering over a legend marker item then the rendered series will highlight in the plot area. Legend highlighting is supported by the <ApiLink pkg="charts" type="CategoryChart" />, <ApiLink pkg="charts" type="FinancialChart" />, and <ApiLink pkg="charts" type="XamDataChart" /> controls and they have the same API for using the highlighting feature.

## Highlight Layers

The Ignite UI for Angular <ApiLink pkg="charts" type="CategoryChart" /> can enable three types of highlighting when hovering over data items.

1. Series Highlighting will highlight the single data point represented by a marker or column when the pointer is positioned over it. This is enabled by setting the <ApiLink pkg="charts" type="CategoryChart" member="isSeriesHighlightingEnabled" label="IsSeriesHighlightingEnabled" /> property to true.

2. Item Highlighting highlights items in a series either by drawing a banded shape at their position or by rendering a marker at their position. This is enabled by setting the <ApiLink pkg="charts" type="CategoryChart" member="isItemHighlightingEnabled" label="IsItemHighlightingEnabled" /> property to true.

3. Category Highlighting targets all category axes in the chart. They draw a shape that illuminates the area of the axis closest to the pointer position. This is enabled by setting the <ApiLink pkg="charts" type="CategoryChart" member="isCategoryHighlightingEnabled" label="IsCategoryHighlightingEnabled" /> property to true.
