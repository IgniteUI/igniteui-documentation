# Angular Axis Types

The Ignite UI for Angular Category Chart uses only one <ApiLink pkg="charts" type="CategoryXAxis" /> and one <ApiLink pkg="charts" type="NumericYAxis" /> type. Similarly, Ignite UI for Angular Financial Chart uses only one <ApiLink pkg="charts" type="TimeXAxis" /> and one <ApiLink pkg="charts" type="NumericYAxis" /> types. However, the Ignite UI for Angular Data Chart provides support for multiple axis types that you can position on any side of the chart by setting [axis location](chart-axis-layouts.md#axis-locations-example) or even inside of the chart by using [axis crossing](chart-axis-layouts.md#axis-crossing-example) properties. This topic goes over each one, which axes and series are compatible with each other, and some specific properties to the unique axes.

## Cartesian Axes

The <ApiLink pkg="charts" type="DataChart" /> with Cartesian Axes, allows you to plot data in horizontal (X-axis) and vertical (X-axis) direction with 3 types of X-Axis
(<ApiLink pkg="charts" type="CategoryXAxis" />, <ApiLink pkg="charts" type="NumericXAxis" />, and <ApiLink pkg="charts" type="TimeXAxis" />) and 2 types of Y-Axis (<ApiLink pkg="charts" type="CategoryYAxis" /> and <ApiLink pkg="charts" type="NumericYAxis" />).

### Category X-Axis

The <ApiLink pkg="charts" type="CategoryXAxis" /> treats its data as a sequence of categorical data items. It can display almost any type of data including strings and numbers. If you are plotting numbers on this axis, it is important to keep in mind that this axis is a discrete axis and not continuous. This means that each categorical data item will be placed equidistant from the one before it. The items will also be plotted in the order that they appear in the axis' data source.

The <ApiLink pkg="charts" type="CategoryXAxis" /> requires you to provide a `DataSource` and a <ApiLink pkg="charts" type="NumericYAxis" member="label" label="Label" /> in order to plot data with it. It is generally used with the <ApiLink pkg="charts" type="NumericYAxis" /> to plot the following type of series:

### Category Y-Axis

The <ApiLink pkg="charts" type="CategoryYAxis" /> works very similarly to the <ApiLink pkg="charts" type="CategoryXAxis" /> described above, but it is placed vertically rather than horizontally. Also, this axis requires you to provide a `DataSource` and a <ApiLink pkg="charts" type="NumericYAxis" member="label" label="Label" /> in order to plot data with it. The <ApiLink pkg="charts" type="CategoryYAxis" /> is generally used with the <ApiLink pkg="charts" type="NumericXAxis" /> to plot the following type of series:

- <ApiLink pkg="charts" type="BarSeries" />
- `RangeBarSeries`
- <ApiLink pkg="charts" type="StackedBarSeries" />
- <ApiLink pkg="charts" type="Stacked100BarSeries" />

### Numeric X-Axis

The <ApiLink pkg="charts" type="NumericXAxis" /> treats its data as continuously varying numerical data items. Labels on this axis are placed horizontally along the X-Axis. The location of the <ApiLink pkg="charts" type="NumericXAxis" /> labels depends on the <ApiLink pkg="charts" type="BubbleSeries" member="xMemberPath" label="XMemberPath" /> property of the various [Scatter Series](../types/scatter-chart.md) that it supports if combined with a <ApiLink pkg="charts" type="NumericYAxis" />. Alternatively, if combined with the <ApiLink pkg="charts" type="CategoryXAxis" />, these labels will be placed corresponding to the <ApiLink pkg="charts" type="AreaSeries" member="valueMemberPath" label="ValueMemberPath" /> of the <ApiLink pkg="charts" type="BarSeries" />, `RangeBarSeries`, <ApiLink pkg="charts" type="StackedBarSeries" />, and <ApiLink pkg="charts" type="Stacked100BarSeries" />.

The <ApiLink pkg="charts" type="NumericXAxis" /> is compatible with the following type of series:

- <ApiLink pkg="charts" type="BarSeries" />

### Time X Axis

The <ApiLink pkg="charts" type="TimeXAxis" /> treats its data as a sequence of data items, sorted by date. Labels on this axis type are dates and can be formatted and arranged according to date intervals. The date range of this axis is determined by the date values in a data column that is mapped using its <ApiLink pkg="charts" type="TimeXAxis" member="dateTimeMemberPath" label="DateTimeMemberPath" />. This, along with a `DataSource` is required to plot data with this axis type.

The <ApiLink pkg="charts" type="TimeXAxis" /> is the X-Axis type in the <ApiLink pkg="charts" type="FinancialChart" /> component.

#### Breaks in Time X Axis

The <ApiLink pkg="charts" type="TimeXAxis" /> has the option to exclude intervals of data by using <ApiLink pkg="charts" type="TimeXAxis" member="breaks" label="Breaks" />. As a result, the labels and plotted data will not appear at the excluded interval. For example, working/non-working days, holidays, and/or weekends. An instance of <ApiLink pkg="charts" type="TimeAxisBreak" /> can be added to the <ApiLink pkg="charts" type="TimeXAxis" member="breaks" label="Breaks" /> collection of the axis and configured by using a unique <ApiLink pkg="charts" type="TimeAxisBreak" member="start" label="Start" />, <ApiLink pkg="charts" type="TimeAxisBreak" member="end" label="End" /> and <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" />.

#### Formatting in Time X Axis

The <ApiLink pkg="charts" type="TimeXAxis" /> has the <ApiLink pkg="charts" type="TimeXAxis" member="labelFormats" label="LabelFormats" /> property, which represents a collection of <ApiLink pkg="charts" type="TimeAxisLabelFormat" /> objects. Each <ApiLink pkg="charts" type="TimeAxisLabelFormat" /> added to the collection is responsible for assigning a unique <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="format" label="Format" /> and <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="range" label="Range" />. This can be especially useful for drilling down data from years to milliseconds and adjusting the labels depending on the range of time shown by the chart.

The <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="format" label="Format" /> property of the <ApiLink pkg="charts" type="TimeAxisLabelFormat" /> specifies what format to use for a particular visible range. The <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="range" label="Range" /> property of the <ApiLink pkg="charts" type="TimeAxisLabelFormat" /> specifies the visible range at which the axis label formats will switch to a different format. For example, if you have two <ApiLink pkg="charts" type="TimeAxisLabelFormat" /> elements with a range set to 10 days and another set to 5 hours, then as soon as the visible range of the axis becomes less than 10 days, it will switch to 5-hour format.

#### Intervals in Time X Axis

The <ApiLink pkg="charts" type="TimeXAxis" /> replaces the conventional <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" /> property of the category and numeric axes with an <ApiLink pkg="charts" type="TimeXAxis" member="intervals" label="Intervals" /> collection of type <ApiLink pkg="charts" type="TimeAxisInterval" />. Each <ApiLink pkg="charts" type="TimeAxisInterval" /> added to the collection is responsible for assigning a unique <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" />, <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="range" label="Range" /> and <ApiLink pkg="charts" type="TimeAxisInterval" member="intervalType" label="IntervalType" />. This can be especially useful for drilling down data from years to milliseconds to provide unique spacing between labels depending on the range of time shown by the chart. A description of these properties is below:

- <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" />: This specifies the interval to use. This is tied to the <ApiLink pkg="charts" type="TimeAxisInterval" member="intervalType" label="IntervalType" /> property. For example, if the <ApiLink pkg="charts" type="TimeAxisInterval" member="intervalType" label="IntervalType" /> is set to `Days`, then the numeric value specified in <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" /> will be in days.
- <ApiLink pkg="charts" type="TimeAxisLabelFormat" member="range" label="Range" />: This specifies the visible range at which the axis interval will switch to a different interval. For example, if you have two TimeAxisInterval with a range set to 10 days and another set to 5 hours, as soon as the visible range in the axis becomes less than 10 days it will switch to the interval whose range is 5 hours.
- <ApiLink pkg="charts" type="TimeAxisInterval" member="intervalType" label="IntervalType" />: This specifies the unit of time for the <ApiLink pkg="charts" type="NumericYAxis" member="interval" label="Interval" /> property.

## Polar Axes

The <ApiLink pkg="charts" type="DataChart" /> with Polar Axes, allows you to plot data outwards (radius axis) from center of the chart and around (angle axis) of center of the chart.

### Category Angle Axis

The <ApiLink pkg="charts" type="CategoryAngleAxis" /> treats its data as a sequence of category data items. The labels on this axis are placed along the edge of a circle according to their position in that sequence. This type of axis can display almost any type of data including strings and numbers.

The <ApiLink pkg="charts" type="CategoryAngleAxis" /> is generally used with the <ApiLink pkg="charts" type="NumericRadiusAxis" /> to plot [Radial Series](../types/radial-chart.md).

### Proportional Category Angle Axis

The <ApiLink pkg="charts" type="ProportionalCategoryAngleAxis" /> treats its data as a sequence of category data items. The labels on this axis are placed along the edge of a circle according to their position in that sequence. This type of axis can display almost any type of data including strings and numbers.

The <ApiLink pkg="charts" type="ProportionalCategoryAngleAxis" /> is generally used with the <ApiLink pkg="charts" type="NumericRadiusAxis" /> to plot a pie chart eg. [Radial Series](../types/radial-chart.md).

### Numeric Angle Axis

The <ApiLink pkg="charts" type="NumericAngleAxis" /> treats its data as continuously varying numerical data items. The labels on this axis area placed along a radius line starting from the center of the circular plot. The location of the labels on the <ApiLink pkg="charts" type="NumericAngleAxis" /> varies according to the value in the data column mapped using the <ApiLink pkg="charts" type="BubbleSeries" member="radiusMemberPath" label="RadiusMemberPath" /> property of the [Polar Series](../types/polar-chart.md) object or the <ApiLink pkg="charts" type="AreaSeries" member="valueMemberPath" label="ValueMemberPath" /> property of the [Radial Series](../types/radial-chart.md) object.

The The <ApiLink pkg="charts" type="NumericAngleAxis" /> can be used with either the <ApiLink pkg="charts" type="CategoryAngleAxis" /> to plot [Radial Series](../types/radial-chart.md) or with the <ApiLink pkg="charts" type="NumericRadiusAxis" /> to plot [Polar Series](../types/polar-chart.md) respectively.

### Numeric Radius Axis

The <ApiLink pkg="charts" type="NumericRadiusAxis" /> treats the data as continuously varying numerical data items. The labels on this axis are placed around the circular plot. The location of the labels varies according to the value in a data column mapped using the `AngleMemberPath` property of the corresponding polar series.

The <ApiLink pkg="charts" type="NumericRadiusAxis" /> can be used with the <ApiLink pkg="charts" type="NumericRadiusAxis" /> to plot [Polar Series](../types/polar-chart.md).
