# Angular Data Aggregations

In the Ignite UI for Angular <ApiLink type="CategoryChart" /> control Data Aggregations feature allows you to group data in the chart by unique values on the <ApiLink type="CategoryChart" member="xAxisMemberPath" label="XAxis" /> and then sort those groups. You may then apply summaries which will be reflected by the range of the <ApiLink type="CategoryChart" member="yAxisMemberPath" label="YAxis" /> and will be displayed in the tooltip when hovering the series.

## Angular Data Aggregations Example

The following example depicts a [Column Chart](../types/column-chart.md) that groups by the Country member of the <ApiLink type="CategoryChart" member="xAxisMemberPath" label="XAxis" /> and can be changed to other properties within each data item such as Product, MonthName, and Year to aggregate the sales data. Also a summary and sort option is available to get a desirable order for the grouped property.

Note, the abbreviated functions found within the dropdowns for <ApiLink type="CategoryChart" member="initialSummaries" label="InitialSummaries" /> and <ApiLink type="CategoryChart" member="groupSorts" label="GroupSorts" /> have be applied as shown to get a correct result based on the property you assign. eg. Sum(sales) as Sales | Sales Desc

<Sample src="/charts/category-chart/data-aggregations" height={500} alt="Angular Data Aggregations Example" />

```html
<igx-category-chart
    [dataSource]="salesData"
    initialGroups="country"
    initialSummaries="Sum(sales) as Sales"
    groupSorts="Sales Desc">
</igx-category-chart>
```
