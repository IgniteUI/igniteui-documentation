# Angular Axis Gridlines

All Ignite UI for Angular charts include built-in capability to modify appearance of axis lines as well as frequency of major/minor gridlines and tickmarks that are rendered on the X-Axis and Y-Axis.

<DocsAside type="note">
the following examples can be applied to <ApiLink pkg="charts" type="CategoryChart" /> as well as <ApiLink pkg="charts" type="FinancialChart" /> controls.
</DocsAside>

Axis major gridlines are long lines that extend horizontally along the Y-Axis or vertically along the X-Axis from locations of axis labels, and they render through the plot area of the chart. Axis minor gridlines are lines that render between axis major gridlines.

Axis tickmarks are displayed along all horizontal and vertical axes at each label at all major line positions of the Angular chart.

## Angular Axis Gridlines Properties

Setting the axis interval property specifies how often major gridlines and axis labels are rendered on an axis. Similarly, the axis minor interval property specifies how frequent minor gridlines are rendered on an axis.

In order to display minor gridlines that correspond to minor interval, you need to set <ApiLink pkg="charts" type="CategoryChart" member="xAxisMinorStroke" label="XAxisMinorStroke" /> and <ApiLink pkg="charts" type="CategoryChart" member="xAxisMinorStrokeThickness" label="XAxisMinorStrokeThickness" /> properties on the axis. This is because minor gridlines do not have a default color or thickness and they will not be displayed without first assigning them.

You can customize how the gridlines are displayed in your Angular chart by setting the following properties:

| Axis Visuals           | Type    | Property Names                                               | Description |
| -----------------------|---------|--------------------------------------------------------------|---------------- |
| Major Stroke Color     | string  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisMajorStroke" label="XAxisMajorStroke" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisMajorStroke" label="YAxisMajorStroke" />                   | These properties set the color of axis major gridlines. |
| Minor Stroke Color     | string  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisMinorStroke" label="XAxisMinorStroke" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisMinorStroke" label="YAxisMinorStroke" />                   | These properties set the color of axis minor gridlines. |
| Major Stroke Thickness | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisMajorStrokeThickness" label="XAxisMajorStrokeThickness" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisMajorStrokeThickness" label="YAxisMajorStrokeThickness" /> | These properties set the thickness in pixels of the axis major gridlines. |
| Minor Stroke Thickness | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisMinorStrokeThickness" label="XAxisMinorStrokeThickness" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisMinorStrokeThickness" label="YAxisMinorStrokeThickness" /> | These properties set the thickness in pixels of the axis minor gridlines. |
| Major Interval         | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisInterval" label="XAxisInterval" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisInterval" label="YAxisInterval" />                         | These properties set interval between axis major gridlines and labels. |
| Minor Interval         | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisMinorInterval" label="XAxisMinorInterval" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisMinorInterval" label="YAxisMinorInterval" />               | These properties set interval between axis minor gridlines, if used. |
| Axis Line Stroke Color | string  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisStroke" label="XAxisStroke" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisStroke" label="YAxisStroke" />                   | These properties set the color of an axis line. |
| Axis Stroke Thickness  | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisStrokeThickness" label="XAxisStrokeThickness" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisStrokeThickness" label="YAxisStrokeThickness" /> | These properties set the thickness in pixels of an axis line. |

Regarding the Major and Minor Interval in the table above, it is important to note that the major interval for axis labels will also be set by this value, displaying one label at the point on the axis associated with the interval. The minor interval gridlines are always rendered between the major gridlines, and as such, the minor interval properties should always be set to something much smaller (usually 2-5 times smaller) than the value of the major Interval properties.

On category axes, the intervals are represented as an index between first item and last category item. Generally, this value should equal to 10-20% of total numbers of category items for the major Interval so that all axis labels fit on axis so that they are not clipped by other axis labels. For minor intervals, this is represented as a fraction of the major interval properties. This value generally should equal between 0.25 and 0.5.

On numeric axes, the interval values are represented as a double between axis minimum value and axis maximum value. By default, numeric axes will automatically calculate and find a nice and round interval based on axis minimum values and maximum value.

On date time axes, this value is represented as time span between axis minimum value and axis maximum value.

## Angular Axis Tickmarks Example

Axis tick marks are enabled by setting the <ApiLink pkg="charts" type="CategoryChart" member="xAxisTickLength" label="XAxisTickLength" /> and <ApiLink pkg="charts" type="CategoryChart" member="yAxisTickLength" label="YAxisTickLength" /> properties to a value greater than 0. These properties specifies the length of the line segments forming the tick marks.

Tick marks are always extend from the axis line and point to the direction of the labels. Labels are offset by the value of the length of tickmarks to avoid overlapping. For example, with the <ApiLink pkg="charts" type="CategoryChart" member="yAxisTickLength" label="YAxisTickLength" /> property is set to 5, axis labels will be shifted left by that amount.

| Axis Visuals           | Type    | Property Names                                             | Description |
| -----------------------|---------|------------------------------------------------------------|------------------------- |

| Tick Stroke Thickness  | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisTickStrokeThickness" label="XAxisTickStrokeThickness" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisTickStrokeThickness" label="YAxisTickStrokeThickness" /> | These properties set the thickness of the axis tick marks. |
| Tick Stroke Length     | number  | <ApiLink pkg="charts" type="CategoryChart" member="xAxisTickLength" label="XAxisTickLength" /> <br /> <ApiLink pkg="charts" type="CategoryChart" member="yAxisTickLength" label="YAxisTickLength" />                   | These properties set the length of the axis tick marks. |
