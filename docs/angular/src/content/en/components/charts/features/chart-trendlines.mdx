# Angular Chart Trendlines

In Ignite UI for Angular charts, trendlines help in identifying a trend or finding patterns in data. Trendlines are always rendered in front of data points bound to the chart and are supported by the <ApiLink pkg="charts" type="CategoryChart" />, <ApiLink pkg="charts" type="FinancialChart" />, and <ApiLink pkg="charts" type="DataChart" /> (except for stacked series, shape series, and range series).

Trendlines are off by default, but you can enable them by setting the <ApiLink pkg="charts" type="CategoryChart" member="trendLineType" label="TrendLineType" /> property. Also, you can modify multiple appearance properties of trendlines such as its brush, period, and thickness.

The trendlines also have the ability to have a dash array applied to them once enabled. This is done by setting the <ApiLink pkg="charts" type="CategoryChart" member="trendLineDashArray" label="TrendLineDashArray" /> property to an array of numbers. The numeric array describes the length of the dashes of the trendline.

## Angular Chart Trendline Layer

The <ApiLink pkg="charts" type="TrendLineLayer" /> is a series type that is designed to display a single trendline type for a target series. The difference between this and the existing trendline features on the existing series types is that since the <ApiLink pkg="charts" type="TrendLineLayer" /> is a series type, you can add more than one of them to the `Series` collection of the chart to have multiple trendlines attached to the same series. You can also have the trendline appear in the legend, which was not possible previously.

## Trendline Layer Usage

The <ApiLink pkg="charts" type="TrendLineLayer" /> must be provided with a <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> and a <ApiLink pkg="charts" type="TrendLineLayer" member="trendLineType" label="TrendLineType" /> in order to work properly. The different trendline types that are available are the same as the trendlines that are available on the series.

If you would like to show the <ApiLink pkg="charts" type="TrendLineLayer" /> in the Legend, you can do so by setting the <ApiLink pkg="charts" type="TrendLineLayer" member="useLegend" label="UseLegend" /> property to `true`.

## Styling the Trendline Layer

By default, the <ApiLink pkg="charts" type="TrendLineLayer" /> renders with the same color as its <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> in a dashed line. This can be configured by using the various styling properties on the <ApiLink pkg="charts" type="TrendLineLayer" />.

To change the color of the trendline that is drawn, you can set its <ApiLink pkg="charts" type="TrendLineLayer" member="brush" label="Brush" /> property. Alternatively, you can also set the <ApiLink pkg="charts" type="TrendLineLayer" member="useIndex" label="UseIndex" /> property to `true`, which will pull from the chart's `Brushes` palette based on the index in which the <ApiLink pkg="charts" type="TrendLineLayer" /> is placed in the chart's `Series` collection.

You can also modify the way that the <ApiLink pkg="charts" type="TrendLineLayer" /> appears by using its <ApiLink pkg="charts" type="TrendLineLayer" member="appearanceMode" label="AppearanceMode" /> and <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> properties. The <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" /> takes a value between -1.0 and 1.0 to determine how much of a "shift" to apply to the options that end in "Shift".

The following are the options for the <ApiLink pkg="charts" type="TrendLineLayer" member="appearanceMode" label="AppearanceMode" /> property:

- `Auto`: This will default to the DashPattern enumeration.
- `BrightnessShift`: The trendline will take the <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> brush and modify its brightness based on the provided <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" />.
- `DashPattern`: The trendline will appear as a dashed line. The frequency of the dashes can be modified by using the <ApiLink pkg="charts" type="TrendLineLayer" member="dashArray" label="DashArray" /> property on the <ApiLink pkg="charts" type="TrendLineLayer" />.
- `OpacityShift`: The trendline will take the <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> brush and modify its opacity based on the provided <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" />.
- `SaturationShift`: The trendline will take the <ApiLink pkg="charts" type="TrendLineLayer" member="targetSeries" label="TargetSeries" /> brush and modify its saturation based on the provided <ApiLink pkg="charts" type="TrendLineLayer" member="shiftAmount" label="ShiftAmount" />.
