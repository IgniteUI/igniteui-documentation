# Angular Chart Annotations

The Angular chart's hover interactions and annotations are implemented through hover interaction layers, which are series that are added to the series collection. These layers are dependent on the cursor position. Each of these annotation layers provides a different hover interaction that may be used individually or combined with others to provide powerful hover interactions.

## Angular Crosshair Layer

The <ApiLink type="CrosshairLayer" /> renders as crossing lines intersecting at the actual value of every series that they are configured to target with each series rendering a separate set of lines.

Crosshair types include:
- Horizontal
- Vertical
- Both

The chart's crosshairs can also be configured to snap to data points by setting the <ApiLink type="CategoryChart" member="crosshairsSnapToData" label="CrosshairsSnapToData" /> property to true, otherwise the crosshairs will be interpolated between data points. Annotations can also be enabled to display the crosshair's value along the axis.

You can configure the crosshair layer so that the layer will only display on one specific series, as by default they will target all series in the chart control. To achieve this, set the <ApiLink type="CrosshairLayer" member="targetSeries" label="TargetSeries" /> property.

By default, the color of the crosshair lines is a lighter color than the series that it is interacting with. However, this default setting can be overridden so that you can select a color that will be used for the crosshair lines. This is done by setting the <ApiLink type="CrosshairLayer" member="brush" label="Brush" /> property of the Crosshair Layer.

## Angular Final Value Layer

The <ApiLink type="FinalValueLayer" /> of the <ApiLink type="DataChart" /> control provides a quick view along the axis of the ending value displayed in a series.

You can configure this annotation to target a specific series if you want to have multiple final value layers present with different configurations. This can be done be setting the <ApiLink type="FinalValueLayer" member="targetSeries" label="TargetSeries" /> property.

You can also customize this annotation by setting the following properties:

- <ApiLink type="FinalValueLayer" member="axisAnnotationBackground" label="AxisAnnotationBackground" />: This property is used to choose the brush for the annotation's background color. The default is to use the series brush.
- <ApiLink type="FinalValueLayer" member="axisAnnotationTextColor" label="AxisAnnotationTextColor" />: This property is used to choose the brush for the annotation's text color.
- <ApiLink type="FinalValueLayer" member="axisAnnotationOutline" label="AxisAnnotationOutline" />: This property is used to choose the brush for the annotation's outline color.

The following example demonstrates how to style the final value layer annotation by setting the properties listed above.

<Sample src="/charts/data-chart/final-value-layer-styling" height={500} alt="Angular Final Value Layer Styling" />

```html
<igx-category-chart
    [dataSource]="data"
    finalValueAnnotationsVisible="true">
</igx-category-chart>
```

## Angular Callout Layer

The <ApiLink type="CalloutLayer" /> displays annotations from existing or new data on the chart control. The annotations appear next to the given data values in the data source.

Use the callout annotations to display additional information, such as notes or specific details about data points, that you would like to point out to your users.

You can configure the callouts to target a specific series if you want to have multiple callout layers present with different configurations. This can be done by setting the <ApiLink type="CalloutLayer" member="targetSeries" label="TargetSeries" /> property.

You can also customize this annotation by setting the following properties:

- <ApiLink type="CalloutLayer" member="calloutLeaderBrush" label="CalloutLeaderBrush" />: This property is used to choose the brush for the leader lines for the callouts for the layer.
- <ApiLink type="CalloutLayer" member="calloutOutline" label="CalloutOutline" />: This property is used to choose the brush for the annotation's outline color.
- <ApiLink type="CalloutLayer" member="calloutBackground" label="CalloutBackground" />: This property is used to choose the brush for the annotation's background color. The default is to use the series brush.
- <ApiLink type="CalloutLayer" member="calloutTextColor" label="CalloutTextColor" />: This property is used to choose the brush for the annotation's text color.
- <ApiLink type="CalloutLayer" member="calloutStrokeThickness" label="CalloutStrokeThickness" />: This property is used to choose the thickness for the callout backing.
- <ApiLink type="CalloutLayer" member="calloutCornerRadius" label="CalloutCornerRadius" />: This property is used to curve the corners of the callouts.
- <ApiLink type="CalloutLayer" member="allowedPositions" label="AllowedPositions" />: This property is used to choose which positions that the callout layer is allowed to use. eg. top, bottom

The following example demonstrates how to style the callout layer annotations by setting the properties listed above:

<Sample src="/charts/data-chart/callout-layer-styling" height={500} alt="Angular Callout Layer Styling" />

```html
<igx-category-chart
    [dataSource]="data"
    calloutsVisible="true"
    [calloutsDataSource]="calloutData"
    calloutsXMemberPath="index"
    calloutsYMemberPath="value"
    calloutsLabelMemberPath="info">
</igx-category-chart>
```
