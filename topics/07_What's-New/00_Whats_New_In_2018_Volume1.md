## 概要

### <a id="excelFunctions"></a> サポートされるその他の Excel 関数

Excel ライブラリでサポートされる Excel 関数が追加されました。サポートされる関数およびサポートされていない関数のリストについては、以下のリンクを参照してください。

- [Infragistics Excel Engine でサポートされる関数](ExcelEngine-List-of-Supported-Built-in-Functions.html)
- [JavaScript Excel Library でサポートされる関数](JavaScript-Excel-Library-List-of-Supported-Built-in-Functions.html)

### <a id="SpreadSheetSorting"></a> スプレッドシートの WorkSheet レベルの並べ替え

このリリースでは、igSpreadsheet コントロールで Worksheet オブジェクトに並べ替え機能を追加しました。列または行を昇順または降順に並べ替えることができます。

* igSpreadsheet の詳細については、「[igSpreadsheet 並べ替え](igSpreadSheetSorting.html)」を参照してください。

### <a id="ExcelEngineSorting"></a> ExcelEngine の WorkSheet レベルの並べ替え

このリリースでは、Excel エンジンで Worksheet オブジェクトに並べ替え機能を追加しました。列または行を昇順または降順に並べ替えることができます。

* Infragistics Excel Engine の詳細については、「[Infragistics Excel Engine 並べ替え](igExcelEngineSorting.html)」を参照してください。
