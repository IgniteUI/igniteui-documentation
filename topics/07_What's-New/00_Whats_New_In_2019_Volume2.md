# 2019 Volume 2 の新機能

このトピックは、Ignite UI™ 2019 Volume 2 リリースの新機能について説明します。

### Infragistics MVC ヘルパー
Infragistics ASP.NET MVC ヘルパーで ASP.NET Core 3 プロジェクトがサポートされるようになりました。詳細情報は、[Ignite UI コントロールを ASP.NET Core で使用](Using-IgniteUI-Controls-in-ASP.NET-Core-project.html)トピックを参照してください。


### Infragistics タグ ヘルパー
Infragistics ASP.NET タグ ヘルパーで ASP.NET Core 3 プロジェクトがサポートされます。詳細情報は、[Ignite UI タグ ヘルパーの使用](using-ignite-ui-tag-helpers.html)トピックを参照してください。
