# igHierarchicalGrid 機能

### 概要

このトピックは、igHierarchicalGrid™ コントロールの機能について説明します。

### トピック
igHierarchicalGrid 機能についての詳細情報は、以下のトピックとセクションで解説します。

- [列とレイアウト](igHierarchicalGrid-Columns-and-Layouts.html): このトピックでは、自動生成などで *igHierarchicalGrid* の列およびレイアウトを定義する方法を紹介します。
- [列テンプレート](igHierarchicalGrid-Column-Template.html): このトピックでは、igHierarchicalGrid の列テンプレート機能の概要について説明します。
- [機能の継承](igHierarchicalGrid-Feature-Inheritance.html): このトピックは、*igHierarchicalGrid* の子レイアウトの機能を継承する方法を示します。
- [グループ化](igHierarchicalGrid-Grouping-Overview.html): このトピックは、igHierarchicalGrid™ のグループ化 (GroupBy) 機能に関する内容にリンクしています。
- [複数列ヘッダー](igHierarchicalGrid-MultiColumnHeaders-Configuring.html): このトピックでは、igHierarchicalGrid 複数列ヘッダー機能の構成方法について説明します。
- [ロード オン デマンド](igHierarchicalGrid-Load-on-Demand.html): このトピックでは、データを一度にオン デマンドで *igHierarchicalGrid* に読み込む 2 とおりの方法を示します。
- [行セレクター](igHierarchicalGrid-Row-Selectors-LandingPage.html): このトピックは、igHierarchicalGrid の行セレクター機能に関する内容にリンクしています。
- [選択](jQuery-igHierarchical-Grid-Selection-Landing-Page.html): このトピックは、igHierarchicalGrid™ のセレクター機能に関する内容にリンクしています。
- [仮想化](igHierarchicalGrid-Virtualization.html): 仮想化は、アクティブなイン メモリー DOM オブジェクトの数を減らす igHierarchicalGrid コントロールの機能です。仮想化は、仮想化されていないグリッドと比較してユーザーからは何も変わりません。
