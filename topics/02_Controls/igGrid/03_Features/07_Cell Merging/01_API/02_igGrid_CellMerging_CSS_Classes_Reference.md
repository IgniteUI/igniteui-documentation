# CSS クラス リファレンス (セル結合、igGrid)


## トピックの概要

### 目的

このトピックでは、`igGrid`™ コントロールのセル結合機能の CSS クラスに関する参照情報を提供します。

### 前提条件

このトピックを理解するためには、以下のトピックを理解しておく必要があります:

- [セル結合の概要 (igGrid)](igGrid-CellMerging-Overview.html): このトピックは、`igGrid` コントロールのセル結合機能とその機能性について説明します。`igGrid` においてセル結合を有効にし構成する方法のコード例が含まれます。



## セル結合スタイリングの参照

このセクションでは、`igGrid` コントロールの機能を使用するときの セル結合関連の各種 CSS クラスについて説明します。


以下の表は、セル結合機能を有効にしたときに発生するイベントをまとめたものです。

CSS クラス|説明
---|---
[ui-iggrid-mergedcell](%%jQueryApiUrl%%/ui.iggridcellmerging#theming) |最初の 1 つを除いて結合されたグループの各セルに適用されるクラス
[ui-iggrid-mergedcellsbottom](%%jQueryApiUrl%%/ui.iggridcellmerging#theming) |結合されたグループの一番下のセルに適用されるクラス
[ui-iggrid-mergedcellstop](%%jQueryApiUrl%%/ui.iggridcellmerging#theming) |結合されたグループの一番上のセルに適用されるクラス
