# igFunnelChart

## このグループのトピックについて

### 概要

このセクションには、`igFunnelChart`™ コントロールに関するトピックが含まれています。

`igFunnelChart` コントロールは、Canvas 要素とその関連 API に基づいて HTML5 Web アプリケーションとサイトにファンネル チャートを表示するための機能を提供します。

![](images/igFunnelChart_%28Landing_Page%29_1.png)
