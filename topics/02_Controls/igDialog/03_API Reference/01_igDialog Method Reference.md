## <a id="methods"></a> メソッド リファレンス

igDialog コントロールの各種メソッドの目的と機能を表にまとめると、次のようになります。

メソッド|説明
--- | ---
[state](%%jQueryApiUrl%%/ui.igDialog#methods:state)|エディターの状態を取得します。
[mainElement](%%jQueryApiUrl%%/ui.igDialog#methods:mainElement) |動的に作成される div 要素 (ダイアログを表す要素) への参照を取得します。
[content](%%jQueryApiUrl%%/ui.igDialog#methods:content) |`igDialog` コンテンツのコンテナーを取得するか、提供された新しいコンテンツを設定します。ヘッダー、サイズ変更のハンドルなど以外のダイアログ ウィンドウの内部コンテナーです。
[close](%%jQueryApiUrl%%/ui.igDialog#methods:close) |開いている `igDialog` を閉じます。
[open](%%jQueryApiUrl%%/ui.igDialog#methods:open) |閉じている `igDialog` を開きます。
[minimize](%%jQueryApiUrl%%/ui.igDialog#methods:minimize) |開いている `igDialog` を最小化します。
[maximize](%%jQueryApiUrl%%/ui.igDialog#methods:maximize) |開いている `igDialog` を最大化します。
[pin](%%jQueryApiUrl%%/ui.igDialog#methods:pin) |ピン固定されていない `igDialog` をピン固定します。
[unpin](%%jQueryApiUrl%%/ui.igDialog#methods:unpin) |ピン固定されている `igDialog` の固定を解除します。
[getTopModal](%%jQueryApiUrl%%/ui.igDialog#methods:getTopModal) |一番上のモーダル ダイアログへの参照を取得します。
[isTopModal](%%jQueryApiUrl%%/ui.igDialog#methods:isTopModal) |ダイアログがモーダルで現在アクティブであることを示します。
[moveToTop](%%jQueryApiUrl%%/ui.igDialog#methods:moveToTop) |ダイアログを一番上に移動させます。
[destroy](%%jQueryApiUrl%%/ui.igDialog#methods:destroy)|`igDialog` コントロールを破棄します。
