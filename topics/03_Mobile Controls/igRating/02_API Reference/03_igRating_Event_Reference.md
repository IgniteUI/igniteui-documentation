# イベント リファレンス

## トピックの概要

### 目的

このトピックでは、`igRating`™ コントロールのイベントを紹介します。


## イベント リファレンス

以下の表に、`igRating` Mobile コントロールの特長あるイベントの目的と機能の概要を示します。

イベント|説明|キャンセル可能
---|---|---
[hoverChange](%%jQueryApiUrl%%/mobile.igRating#events:hoverChange) |イベントは、選択された値がホバーされる前に発生します。|True
[valueChange](%%jQueryApiUrl%%/mobile.igRating#events:valueChange) |イベントは、選択された値が変更される前に発生します。false が返されると、操作はキャンセルされ、値は変更されません。|True
