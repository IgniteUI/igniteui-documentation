# クラス リファレンス

## トピックの概要

### 目的

このトピックでは、`igRating`™ コントロールの CSS クラスを紹介します。

### このトピックの内容

このトピックは、以下のセクションで構成されます。

-   [概要](#introduction)
-   [CSS クラス リファレンス](#css-reference)
-   [関連コンテンツ](#related-content)



## <a id="introduction"></a>概要

このセクションでは、`igRating` コントロールの CSS クラスを説明します。すべてのスタイルを上書きし、カスタムの `igRating` コントロール レイアウトを作成できます。項目にユーザー設定の画像を設定する場合、項目の幅と高さは画像のサイズと同じにならなければなりません。カスタム レイアウトを描画する方法については、`igRating` [カスタム項目](%%SamplesUrl%%/mobile-rating/custom-items)のサンプルを確認してください。



## <a id="css-reference"></a>CSS クラス リファレンス

以下の表に、 `igRating` コントロールの主な CSS クラスの目的と機能の概要を示します。この実装方法の詳細については、表の下にある関連サンプルをご覧ください。

CSS クラス|説明
---|---
[ui-igrating](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating)|コンテナー DIV 要素に適用されるクラス。
[ui-igrating-selected](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-selected)|ホバー状態の投票のコンテナー DIV 要素に適用されるクラス。
[ui-igrating-vote](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-vote)|投票の SPAN 要素に適用されるクラス。
[ui-igrating-voteselected](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-voteselected)|選択された状態の投票の SPAN 要素に適用されるクラス。
[ui-igrating-votereadonly](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-votereadonly)|読み取り専用モードの投票の SPAN 要素に適用されるクラス。
[ui-igrating-votereadonlyselected](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-votereadonlyselected)|`igRating` が読み取り専用に設定されている場合の、選択された状態の投票の SPAN 要素に適用されるクラス。
[ui-igrating-votehover](%%jQueryApiUrl%%/mobile.igRating#theming:ui-igrating-votehover)|ホバー状態の投票の SPAN 要素に適用されるクラス。
