# プロパティ リファレンス

## トピックの概要

### 目的

このトピックでは、`igRating`™ コントロールのプロパティを紹介します。

### このトピックの内容

このトピックは、以下のセクションで構成されます。

-   [プロパティ リファレンス](#property-reference)
-   [関連コンテンツ](#related-content)


## <a id="property-reference"></a>プロパティ リファレンス 

以下の表に、`igRating` モバイル コントロールの主なプロパティの目的と機能の概要を示します。`inputName` 値を除き、すべてのプロパティは動的に変更できます。

JavaScript のプロパティ|HTML のプロパティ|MVC ラッパーのプロパティ|説明
---|---|---|---
[value](%%jQueryApiUrl%%/mobile.igRating#options:value) |data-value |[Value](Infragistics.Web.Mvc.Mobile~Infragistics.Web.Mvc.Mobile.RatingModel~Value.html) |選択された投票値を取得または設定します。
[voteCount](%%jQueryApiUrl%%/mobile.igRating#options:voteCount) |data-vote-count |[VoteCount](Infragistics.Web.Mvc.Mobile~Infragistics.Web.Mvc.Mobile.RatingModel~VoteCount.html) |投票の数を取得または設定します。
[voteWidth](%%jQueryApiUrl%%/mobile.igRating#options:voteWidth) |data-vote-width |[VoteWidth](Infragistics.Web.Mvc.Mobile~Infragistics.Web.Mvc.Mobile.RatingModel~VoteWidth.html) |投票のカスタム幅 (ピクセル単位) を取得または設定します。値 0 の場合、実行時スタイル値が使用されます。
[voteHeight](%%jQueryApiUrl%%/mobile.igRating#options:voteHeight) |data-vote-height |[VoteHeight](Infragistics.Web.Mvc.Mobile~Infragistics.Web.Mvc.Mobile.RatingModel~VoteHeight.html) |投票のカスタム高さ (ピクセル単位) を取得または設定します。値 0 の場合、実行時スタイル値が使用されます。
[readOnly](%%jQueryApiUrl%%/mobile.igRating#options:readOnly) |data-read-only |[ReadOnly](Infragistics.Web.Mvc.Mobile~Infragistics.Web.Mvc.Mobile.RatingModel~ReadOnly.html) |レーティングが編集可能かどうかを取得または設定します。
[inputName](%%jQueryApiUrl%%/mobile.igRating#options:inputName) |data-input-name |なし|MVC ラッパーで使用された非表示の入力の名前を取得または設定し、値をサーバーに送信します。このオプションは MVC ラッパーでのみ使用され、評価値をサーバーに戻します。
