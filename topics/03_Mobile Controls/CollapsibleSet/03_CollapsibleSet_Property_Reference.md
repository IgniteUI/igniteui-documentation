# CollapsibleSet プロパティ リファレンス



## *CollapsibleSet* プロパティ リファレンス
#### 概要

このトピックでは、`CollapsibleSet` ウィジェットのプロパティに関するリファレンス情報を紹介します。

`CollapsibleSet` に設定されたすべてのプロパティは、collapsible ブロック内のすべての collapsible コントロールに継承されます。

#### *CollapsibleSet* プロパティの参照チャート

以下の表は、`CollapsibleSet` MVC ラッパーのプロパティについて説明し、そのデフォルト値をリストしています。プロパティはアルファベット順に紹介します。

> **注:**
>
> デフォルトで null 値を持つプロパティでは、jQuery モバイル フレームワークのデフォルト設定でウィジェットを描画します。このため、この表に示されたすべてのデフォルトの状態は、記載時のものであり、予告なく変更される場合があります。最新のデフォルト設定を確認するには、jQuery Mobile マニュアルの [CollapsibleSet](http://jquerymobile.com/test/docs/buttons/buttons-icons.html) の記事を参照してください。
