# Collapsible のプロパティ参照


## *Collapsible* のプロパティ参照

このトピックでは、`Collapsible` ウィジェットのプロパティに関する参照情報を提供します。

以下の表は、`Collapsible` MVC ラッパーのプロパティについて説明し、そのデフォルト値をリストしています。プロパティはアルファベット順に紹介します。

> **注:**
>
> デフォルトで *null* 値を持つプロパティでは、jQuery モバイル フレームワークのデフォルト設定でウィジェットを描画します。このため、この表に示されたすべてのデフォルトの状態は、記載時のものであり、予告なく変更される場合があります。最新のデフォルト設定を確認するには、jQuery Mobile マニュアルの[Collapsible コンテンツ](http://jquerymobile.com/test/docs/content/content-collapsible-options.html) の記事を参照してください。
