# PageFooter の概要
Page Footer MVC ラッパーは、[jQuery Mobile](http://jquerymobile.com/demos/1.1.1/docs/pages/page-anatomy.html) のコンテキストでフッターを定義する場合に使用します。ページ フッター構造の詳細は、jQuery Mobile で提供される[チュートリアル](http://jquerymobile.com/demos/1.1.1/docs/toolbars/docs-footers.html)をご覧ください。

## Page Footer MVC ラッパーの機能
### 固定オプション
このオプション セットにより、ページ フッターを配置および構成することができます。全画面表示またはフォーカス中の Page Footer のビヘイビアーを決定できます。指定できるすべてのオプションについては、[PageFooter プロパティの参照](PageFooter-Property-Reference.html)を参照するか、クライアント側実装の場合、[jQuery Mobile Footer ウィジェットの API ドキュメント](http://api.jquerymobile.com/footer/)を参照してください。

### テーマ
Page は、標準の [jQuery Mobile テーマ](http://jquerymobile.com/demos/1.1.1/docs/api/themes.html)に対応しています。デフォルトで、MVC ラッパーはデフォルトの jQuery Mobile フォーム「a」を使用してコントロールを描画します。

### 属性
MVC Page ラッパーは、クライアント側で表示される一連の HTML 属性に対応したメソッドを備えています。
