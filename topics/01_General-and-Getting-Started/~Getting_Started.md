# Ignite UI を使用した作業の開始
デスクトップおよびモバイル アプリケーションを Ignite UI&trade; で作成する方法がいくつかあります。アプリケーションの作成を開始するには、以下のリンクを参照してください。

## ファイルのインストール、ダウンロード、または参照
Ignite UI ファイルへアクセスするためのさまざまなオプションがあります。

- **[プラットフォーム インストーラー](https://jp.infragistics.com/my-account/keys-and-downloads)**: ライセンスをお持ちの場合、ログインしてプラットフォーム インストーラーをダウンロードできます。

- **[ダウンロード (製品またはカスタム ビルド)](https://jp.igniteui.com/download)**: Ignite UI のフルバージョンをダウンロードするか、特定のコントロールに関連する CSS および JavaScript ファイルのみを含むカスタム ビルドを作成できます。

- **[CDN リンク](https://jp.igniteui.com/download)**: ダウンロード ページから Infragistics CDN で Ignite UI のバンドル バージョンの Ignite UI ファイルもアクセスできます。  
 
> **注**: ダウンロードおよび CDN オプションの場合、既存の顧客としてログインすると、ファイルのライセンス版を取得できます。それ以外の場合、Ignite UI のトライアル版を取得できます。

## コントロールを使用した作業の開始
コントロールの使用を開始するには、[ページ デザイナーを使用すると、コントロールを WYSIWYG 環境で使用できます](http://designer.igniteui.com/index-release-jp.html)。デザイナーは、セットアップ処理なしサンプル データを提供します。

## 最初のページの作成
最初のページを作成するときに、[Ignite UI ヘルプ リソース](http://jp.igniteui.com/getting-started)を参照できます。セットアップおよびページで参照するファイルを表示します。

## Ignite UI の配備
ページを作成したあとに[配備ガイド](deployment-guide)をご確認ください。Ignite UI のファイルについて、およびアプリケーション構成に基づいてファイルを配備する方法の説明があります。
